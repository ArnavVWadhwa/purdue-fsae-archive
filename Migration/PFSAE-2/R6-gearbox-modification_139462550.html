Purdue Formula SAE : R6 gearbox modification

Created by Unknown User (jpeerbol), last modified on Jan 25, 2020

Taken from dirtyuncleron69

We took the shift barrel out, and re-machined the slots so we could go from 1st into 6th, and blocked the shift from 5th into 6th (I think this is what we did, it was a long time ago).

We removed 6th gear (by machining all the teeth off, and using the remaining gear as a spacer), which allowed a true neutral below 1st, with a hard stop so the driver knew when they were there.

the whole thing was done manually with a rotary table and mill. Pretty free form, but it worked really well.

E: If you don't use first gear, it's a lot easier to remove first, and remove the neutral 'detent'. This would make neutral the size of first plus neutral by reshaping the 'star" gear selector

Making a custom shift drum seems like a lot of work for something that can be done much simpler, also it sounds like it could be difficult to get the drum to slide smoothly like an oem shift drum.

Making aluminum gear spacers might also be something worth looking into, but lathing down gears could be easier especially with spare gears around.
