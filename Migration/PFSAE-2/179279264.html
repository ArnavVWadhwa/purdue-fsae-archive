Sub-team Minimum Viable Product 

  • see Subteam front pages  update these for next meeting

Suspension:

  1. Lap sim correlation on steady state:
    1. Direct test of weight
    2. Direct test of downforce + drag (weight independent)
    3. Direct test of powertrain performance
  2. Implement full vehicle suspension and aero tuning plan going from steady state to transient tuning
    1. Validate steady state balance predictions from springs and ARBs
    2. Validate transient maneuver predictions from dampers and rear toe adjustment
  3. Validate kinematic + compliance predictions:
    1. Camber curve
    2. Toe curve
    3. Camber compliance
    4. Toe compliance
  4. Develop user manual for Adams model + lap time simulation

Chassis

  1. Educate new and current members on SES and rules compliance
  2. Build current team member skills in welding, manufacturing, & design
    1. Develop welding curriculum for current members
    2. Add documentation to onboarding project
  3. Reliable driver controls (see level 2)
  4. Generate compliance predictions (chassis side)
    1. Toe 
    2. Camber

Powertrain

  1. Build reliable dyno in order to validate new design decisions/ hand calculations and use it to improve our modeling 
    1. assign modeling assignment to go along with dyno build
    2. cooling sim validation
  2. Develop torque curve that increases driver confidence
    1. Complete the Acceleration Event with only a constant 100% throttle input from driver
  3. Increase number of reliable subteam members by improving their knowledge


Drivetrain

  1. Develop final drive ratio for best overall performance in dynamic events
  2. Tune differential for best lockup settings in order to improve performance in skid pad and corner events
  3. Educate new members on drivetrain design process and manufacturing

Electronics

  1. Improve support capabilities for other systems
    1. Motec integrated lap timing system
    2. Data storage and post processing to track and predict component issues
  2. Improve system reliability
    1. All components achieve IP rating
    2. Easier to maintain and manufacture harness
  3. Automated Dyno load and eventually electronically controlled throttle
  4. Increase sub team membership and reduce specialization of subteam members

Aerodynamics

  1. Achieve vehicle performance targets listed below
    1. Meet mass flow target as determined by powertrain to complete endurance without fan use. Currently 0.125kg/s mass flow per radiator (6.5"x10") for maximum 90C coolant temp at 99F ambient temp.
    2. Achieve balance target as determined by suspension, currently 50%. Provide sufficient adjustment to account for simulation errors and driver preference, and to tune according to vehicle dynamics goals (likely +/-10%).
    3. Achieve ClA > -4.00, CdA < 2.00, weight < 30lb. Chose these values to maintain 2020 weight and stiffness while improving aero performance. Should result in a xx point gain overall for aero package.
  2. Document and validate custom tools used
    1. CFD macros, slurm commands, sim environment justification (cell size, turbulence model, mesh refinement justification)
    2. Full vehicle WT test + on track testing to validate CFD environment
    3. Composite calculators (FEA if implemented, hand calcs, manufacturing calcs)
    4. Destructive testing of 2020 wings to validate stiffness and points of failure calculations
  3. Improve overall package finish
    1. Leading edge seam flange leftovers
    2. Trailing edge straightness
    3. Maintain positional accuracy to 1/8" for all components in all dimensions
    4. Keep all wings aligned to within 1 degree of designed angle (keep wings parallel to each other/perpendicular to endplates)
