designed around current intercomp 10x10x2 scale pads, looks to be their standard size for kart scales, we have no need for largers, 400lb capacity per corner


Why we have these:

corner balancing is worthless without level scales. Hoping we're going to find a flat enough surface will not cut it anymore. Previous scale stands were 4 individual leveling stands, these took about an hour to set up on average and if they got bumped they had to be re-leveled.

Tying the front and rears together allows easy setup for left to right balancing, this should limit adjustment to 3 axes. These do take up slightly more space, but the time savings in adjustment should make up for any other inconveniences.Â 


Scale holders will be bolted in so they can be replaced and adjusted slightly if track width changes.
