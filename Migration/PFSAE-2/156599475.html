Description: 

Figure: PF20 Steering Assembly, PF20 Steering Assembly in Context

The introductory project for chassis will be to design a steering assembly, front roll hoop, and corresponding mounting systems that meet or exceed the pre-defined project goals / design targets (see below). The goal of this project is to continue to progress with CAD skills learned in the initial CAD training videos by incorporating more independent and more complex design problems. In addition, this project should provide you with opportunities to learn simulation techniques in ANSYS (the FEA software used by the team) as well as get familiar with the documentation and hand calculation tools provided. 

This project will be done by all new members to the chassis team alongside any current members who wish to participate (this is not required but highly encouraged).

As a general rule for this intro project, we don't expect you to have any knowledge of car design, CAD software (other than concepts covered in the introduction), or ANSYS. The goal of this project is to let you get comfortable with the software and design process used by the team through a challenging design problem. The purpose of this project is NOT to be frustrating or extremely difficult or time-consuming, because of this make sure you are asking as many questions as you need to (I would rather you ask than progress in the wrong direction and get frustrated and stop working on the project). 

The goals of the project are outlined in detail below along with suggested resources and required deliverables.
