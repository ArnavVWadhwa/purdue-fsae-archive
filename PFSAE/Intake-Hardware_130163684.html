Purdue Formula SAE : Intake Hardware

Created by Unknown User (jpeerbol), last modified on Oct 25, 2019

This page covers all hardware used on the intake. This information is all based on using Accura 60 SLA from Realize.

Screw to expand inserts were first used on the pf19 intake. 10-32 size inserts were selected because most other hardware on the intake is 10-32 and it fits the current throttle position sensor and MAP sensor mounts well. 

Part number from McMaster Carr: 92395A515 
The suggested drill bit size for 92395A515 is 0.25" but after trying to install them into the SLA parts a 0.25" hole was found to be too tight. A 0.257" or F drill was used and was still on the tight size. A 0.26" print diameter with a depth of 0.377" will be used for pf20 intake parts, if the fit is too loose, plastics specific super glue will be used to retain the insert.
