F.7 MONOCOQUE

F.7.3 Front Bulkhead

  • For ‘Composite Panel Height’, we used the height of the monocoque. We would recommend submitting a rules clarification request.
  • The optional section of ‘Second Moment, Surpassing Flat Panel’ was used. We found that we could not pass SES without it.
    • To get the composite cross section, we found the perimeter of half the cross-section and multiplied it by the thickness of the carbon fiber.
    • To get the ‘composite second moment about car centerline (Izz)’, I split the monocoque into a series of triangles and rectangles and made a MATLAB script to do the math. In order for this script to work, various distances and lengths from the CAD model must be input. The image below shows which lengths and points correspond to which in the code. Please note that this code is set up for a configuration like the one depicted below. If a new design is different than this, the functions of the MATLAB code can easily be used, but the set up will need to vary.
