Purdue Formula SAE : FW-14-07

Created by Zhang, Chenkai, last modified on Jun 27, 2023

Simulation: CZ-036

Changes/Justifications

Description of the Change

Picture

Justification

Moved strake outboard and moved leading edge forward

Moved leading edge forward so that the strakes don't see as much inwash, since inwash increases as you move downstream. This should help with separation.

Moved strakes outboard so that the wake doesn't form as inboard, limiting its impact on undertray.

Added mounting cascades

Who says mass flow injection and downforce production are mutually exclusive? Having cascades is actually a fairly popular way of dealing with junction losses while not losing downforce.

The slot gaps are big so that it reduces blockage in increase high CpT flow injection (compared to normal slot gaps). Big slot gap also means each subsequent element does not see as much flow rotating, meaning angle of attack increase can be limited without messing up stagnation point or introducing reflex, so that downforce can be produced while not "stressing" the flow too much.

CFD Results Analysis

CFD Picture

Description

This is now separating across the slot gap. We didn't see that with PF23 front wing because this wing is inwashing on pressure side.

All the these need reduced angle of attack. Which is good since this configuration did not solve the issue of junction losses. Lowering angle of attack will get rid of the separation, allowing more high CpT flow through, it will also stress the flow less, reducing junction losses.

Interestingly both of these vortices get shed. I don't really know why or how or if this is real.
