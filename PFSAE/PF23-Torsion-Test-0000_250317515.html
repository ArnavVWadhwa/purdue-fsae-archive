Purdue Formula SAE : PF23 Torsion Test 0000

Created by Unknown User (hammerly), last modified by Dien Quang Nguyen on May 25, 2023

Equipment

  • 5 dial indicators (loaned from ME Shop)

  • Load bar (in Formula Shop)
  • Bucket to hold load masses (stolen/borrowed from PER)
  • Counterweight bucket (and bolts from fk-it-bucket)
  • Front Fixtures

  • Rear Fixtures (front and rear fixtures should be in G006)

Preparation

Mass of the bar is 122 lbs, with its CG measured by balancing the bar on a tube as shown below. This let me calculate that I'd need a 43.2 lb counterweight 48'' from the center. This negates the weight of the offset bar and makes the exclusion of standard gravity in the ANSYS model acceptable.

Set Up Procedure

  1. Set shock placeholders and pushrods at the same length as the model.
  2. Set up rear fixture and secure to bench using C-clamp
  3. Set up front fixture with front hubs
  4. Set up load bar with front fixtures
  5. Position load bar w/pivot on bench at center axis of vehicle
  6. Tack pivot block to bench to secure in place
  7. Lube load bar pivot
  8. Hang counterweight bucket at marked location and secure in place 
  9. Hang empy load bucket at marked location and secure in place
  10. Set up dial indicators at locations of rear floorpan tabs and (?) tabs
  11. Successively add weights to load bucket and measure deflections in dial indicators 
  12. Successively remove weights from load bucket and measure deflections in dial indicators 

Issues Encountered

Rigid Shock Placeholders

The rigid shocks we've been using are from the dark ages. Most of them have rod ends with only right handed threads - e.g. they can't extend like a pushrod with opposing threads. Most of these rod ends have a larger hole size too, 0.375'' ID instead of the correct size for the Ohlins shocks. Manufacturing for next year's test should include creating new steel rigid shocks with correctly sized and opposed rod ends.
