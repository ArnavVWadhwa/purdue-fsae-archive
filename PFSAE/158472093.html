Okay how to design a chassis

Step 1: Import Suspension Points

  • Edit NX Open script to have appropriate file mapping
  • Run NX Open script within PFXX_CHA_0001_SusPoints
    • Right-click on the main ribbon (where assemblies and curve are) and enable the Developer tab. Press Edit and select the SusPoints_Import.vb file to run. 
    • Evan created a confluence page about it here → https://wiki.itap.purdue.edu/x/SSDCBw
  • Boom! Now a lot of suspension points are in your NX Model as shown below.

Step 2: Group all chassis-related points under one Reference Set

We need to know which points we should use to develop the chassis, so distilling the suspoints into recognizable components is helpful.

Click Add New Reference Set while in the SusPoints file and select the following:

  • Outboard upper control arm (uca) and outer lower control arm (lca) points
  • Inboard uca/lca points
  • Inboard damper points
  • Inboard/outboard tierod points

Step 3: Create Skeleton Model (PFXX_CHA_0002_SkelModel)

  • Use the WAVE Geometry linker to bring the applicable suspension points into the skeleton model prt. might want to ask evan how to do this 
  • Create reference geometry for the main roll hoop, front roll hoop, bulkhead, and engine placement. Use expressions to define this stuff so it's less confusing.
