Purdue Formula SAE : Composites FEA - NX/Ansys/Star-CCM

Created by Unknown User (green338) on Apr 04, 2021

Changes to Ansys between the 2019 and 2020 revisions require that we switch our meshing process to be able to export files compatible with Star-CCM's pressure mapping feature. the high level process for our composites FEA is as follows:

  1. Extract surfaces in NX and clean up model for analysis
  2. Mesh model using NX pre/post and export .dat file
  3. Import .dat file to Ansys and set up analysis using ACP
  4. Import .dat file to Star-CCM and export .dat file with loads
  5. Edit Star-CCM export to have correct APDL pressure loading syntax for use in Ansys
  6. Import APDL commands into workbench and solve
