Purdue Formula SAE : General Onboarding

Created by Giachini, Gabriella, last modified by Aaltonen, Emil Johan on Oct 10, 2023

The new member onboarding project is to CAD a bike pedal mechanism where the goals are:

A. CAD BIKE PEDAL

  1. Learn how to CAD in NX following the FSAE format (robust designs that do not explode)
    1. Set up a blank top-level assembly
    2. Have a reference geometry file with datum planes, coordinate system
    3. Use WAVE geometry linker to connect parts to the reference geometry file
  2. Learn how to CAD with NX. The different topics to be covered are:
    1. WAVE Geometry Linker
    2. Sketches: Geometric constraints, Mirror curves
    3. Features: Holes, Extrusions, Revolve, Mirror Geometry, Trim
    4. Assembly: Assembly constraint for separate parts
  3. Download and use parts from McMaster-Carr

B. Make 2D drawings of all machinable parts

C. Create a Bill of Materials for all parts 

D. Learn how to create a DXF file for waterjetting tabs

**For any questions or to check your final work, contact Emil Aaltonen eaaltone@purdue.edu

Submit files in a zipped folder named yourname_FSAEonboarding

The folder should have

  • all NX parts
  • Excel sheet of your BOM
  • pdfs of your part drawings

Parts to Create within Assembly (+Create New)

a. Reference Geometry (PF_1000_ReferenceGeo)

You will create datum planes and sketches to use as reference for other parts. Follow instructions:

First, create three datum planes: fixed on the XZ plane (1), at distance of 1.3 inches to the left XZ plane (2), at distance of 1.3 inches to the right XZ (3)

Second, make the following sketch with the given dimensions. Make sure that the sketch is fully constrained by using Rapid Dimension and Geometric Constraints.

b. Part of bike frame (PF_2000_Frame)

First step is to use WAVE geometry linker for the concentric circles in the sketch and the two at distance datum planes.

c. Bottom Axle (PF_3000_Axle)

The bottom axle or bottom bracket goes inside the center tube of the bike frame. The crank arms of the pedals fit in them, allowing you to pedal. 

There are many types of bottom axles, but we will do a simple version of it to learn how to revolve. 


First step is to use WAVE geometry linker on the reference geometry part (add the two concentric circles and one of the offset datum planes). 

Second step is to create an offset datum plane at 1/16 from the linked one. 

Third step is to create the following sketch (check video for correct dimensions) and mirror it. And finally, revolve the whole thing:

d. Crank Arm (PF_4000_CrankArm)

Now we will work on the two crank arms that hold the pedal:

First, use WAVE geometry linker to add the sketch and both at distance datum planes. Extrude the sketch at the right distance (you can do this by creating new datum planes and extrude from one to the other). For the first crank arm: create a new datum plane at 1/16in from the linked one and then create a second datum plane at 0.4in from it. Extrude from the first to the second datum plane created to have a crank arm of 0.4in. Use Edge Blend to make the corners of the extrusion look nicer. To finish one crank arm, extrude a cylinder with diameter of 0.375 inches at the center of the concentric circles of the sketch.



For the second crank arm, rotate 180deg the original sketch by making a new one. Then, create a new datum plane at 1/16 + 0.168 in from the linked one and then create a second datum plane at 0.4in from it. Extrude from the first to the second datum plane created to have a crank arm of 0.4in. Repeat the same last steps as with the first crank arm.

a. Pedals (PF_5000_Pedal)

Now we will make a part outside the assembly and then add and constrain it to the other existing parts in the assembly. So first thing to do is to create a new part (the same way as you creating a new assembly, but click model instead of assembly). Name it and add it to the same file path as the assembly. 

First create two datum planes at distance from zy plane of 0.7874in and 3.9370in. The sketch a rectangle using the datum planes the set the width and a height of 3.9370in. Extrude the sketch 0.25 to each side. Make a new sketch with the following parameters and constraints, and revolve it.

Parts to Download from Mc Master - Carr (+Add to PF_OnboardingProject)

Download part numbers from Mc Master Carr: https://www.mcmaster.com/


First, download STEP file, then open file in NX 12 and save it as a part file. Finally, add to assembly and constrain it to existing parts. (No video → contact me if you have any questions)

a. Chain Wheel (PF_7000_ChainWheel)

Part number: 2299K318

Amount: 1


Constrain it to the between frame and 180 rotated crank arm (there's space for it). Use assembly constrains → touch align


b. Bearings (PF_6000_Bearing)

Part number: 6383K16

Amount: 4


Constrain them to the highlighted areas using assembly constrains → touch align. 


Test

To test how robuts your drawing is, change a few dimensions and see how the overall assembly changes. If nothing breaks, GOOD JOB! If something explodes and you start getting issues: check again your work and try to have everything connected (using the reference geometry part file).

Make sure you do this before submitting your work!



BILL OF MATERIALS (BOM)

Create a bill of materials for all parts showing part names and quantity, as well as differentiating between assembly, manufactured and purchased parts. 

Your BOM does not have to be this exhaustive, but model it after the image below (PF23 Chassis metal stock BOM)

2D DRAWINGS

Make 2D drawings in NX drafting of all machinable parts. Make sure to add all important views and dimensions. A dimension is important if it is necessary for the manufacturer!

Remember to remove datums, axis ... basically anything that would prevent someone to clearly see the part and eventually manufacture it. 


Drafting is found in NX in the Applications tab → Drafting
