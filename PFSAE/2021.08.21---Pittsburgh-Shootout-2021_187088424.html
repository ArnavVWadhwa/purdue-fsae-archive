Suspension:

In Shop:

  • Check Brake Fluid and leaks in circuit
  • Thorough Bolt check, with torque stripes
  • Compliance check including lifting vehicle to check for vertical compliance at LCA
  • Plan tests to perform

Bring to Testing

  • Track box
    • Organizers with spare hardware/components
  • Tire Wrench
  • Red Air Tank
  • Wood blocks to lift vehicle
  • Data Binder
  • Anglometer
  • Rain Tires
  • Spare Components
    • Brake Pads
    • Steering Rack
  • Alignment
    • Toe Bar
    • Calipers
    • Angleometer (camber)
    • Scales
    • Miller Stands
    • Additional Springs

Before Run

  • Look for brake leaks
  • Check for compliance (camber and toe wobble, linkage movement)
  • Set corner balance if springs or other recently changed
  • Turn car on with steering centered to provide offset to apply when postprocessing data
  • Turn car on flat surface to provide offset for shock pot data.
  • Set tire pressures
  • Record environment conditions in testing data (temp, humidity, state of car)

After Run / Before Next Stint

  • Check compliance
  • Visual check of bolts/torque stripes
  • Measure tire temps and pressures
  • Maintain desired tire pressures
  • Check Steering Column Bearings

After Testing

  • Bolt check
  • Brake pad check
  • Compliance check with vehicle on blocks
  • Uploading car data sheets to testing page

Powertrain:

In Shop

  • Pressure Test Cooling System to 15PSI and check for leaks, mounted securely
  • Verify Oil System is filled and no visible leaks, oil pump belt not loose, mounted securely, fittings tight
  • Intake / throttle mounted securely
  • Fuel system mounted securely, no leaks 
  • Exhaust / muffler mounted securely
  • Drivetrain mounted correctly, differential filled, CV joints "gooked" up 
  • Drivetrain chain appropriate slack (may require tensioning)
  • Verify powertrain and non-powertrain components are safe from exhaust heat 
  • Cycle Lock

Bring to Testing

  • Fuel
  • Distilled Water
  • Engine Oil
  • Diff Oil
  • Drivetrain Box
  • Track box 1/ 2
  • AN box

Before Run

  • Warm up engine oil to operating temp above 75C
  • Verify Oil Pressures (Hot) to be at an acceptable range varying with RPM
  • Verify Coolant Temps within acceptable range
  • Verify no leaks, or monitor slow leaks (if any)

After Run / Before next stint

  • Verify no leaks
  • Verify oil pressure / temp within acceptable range
  • Verify coolant temp within acceptable range
  • Verify Drivetrain components are torqued

After Testing

  • Temperature over night, if less than 35F drain cooling system.

Electronics

In Shop

  • Get readings from any new sensors
  • Turn on car and check safety systems run if any changes
  • Check for loose wiring in cockpit / near moving or hot parts

Bring to Testing

  • 2 or 3 batteries
  • Battery charger
  • Computer with i2Standard/M1 Tune/M1 Build etc and Ethernet port/connector 
  • Electronics Track box
  • Will to live

Before Run

  • Check break OT switch. 0 means not triggered

After Run/Driver Stops

  • Download Run data
  • Check Battery level
  • Check Run data for discrepancies

After Testing

  • Check data for issue
  • Add data to Google Drive

Aerodynamics:

In Shop

  • Check all bolts on endplates
  • Check swan and tierod bolts 
  • Re-tape any sharp edges
  • Check seat connection

Before Run

  • Check all bolts on endplates
  • Check swan and tierod bolts 

After Run

  • Check all bolts on endplates
  • Check swan and tierod bolts
  • Check seat connection

Bring to Testing

  • Aero Track Box
  • Gaff Tape
  • Top Cover/Nosecone
