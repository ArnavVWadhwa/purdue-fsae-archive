Problem:

  • The team requires a reliable system to convey electrical inputs and outputs for vehicle function, control, and analysis.


Requirements:

VehicleHarness
Maximize Competition Points
  • Minimize weight
  • Vehicle performance data collection and storage
  • Routing that does not interfere with cockpit template testing
  • Minimize CG height
  • Comply with desired weight distribution target
Driver Safety
  • Rules-compliant shut-down systems
    • BOTS
    • Cockpit Main Switch
    • Primary Master Switch
  • Crash safety features beyond the rules
  • Highly visible and reliable brake light
  • The firewall must separate the driver and battery
System Reliability
  • Fully heat-shrunk and booted
  • All components IP-65 rated or higher
  • Routing that considers interference sources
  • Routing that considers moving/hot vehicle components
    • Drivetrain
    • Driver
    • Suspension
    • Engine
    • Oil/Coolant
  • All components securely attached to the vehicle
System Serviceability
  • Quick and easy installation and removal
  • Routing that does not impede other system's serviceability
  • Extra length for each wire to allow for re-crimping if necessary
Driver Confidence
  • Push to talk system for all drivers
  • Reliable and easily accessible controls
  • Completely autonomous engine management
  • No critical system failures during the testing season
