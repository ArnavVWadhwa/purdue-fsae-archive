Notes:

  • The spec sheet that Milspecwiring.com (sales@milspecwiring.com) provided upon request is for a Race Grade sensor (RG_SPEC-0013). (see document attached) This is the same document that was in our files but was confusing because there is no Bosch part number on the document and the last sensor we received did not have RaceGrade packaging, which caused us to not trust the spec sheet.
  • The mechanical engineers on the team had difficulty understanding how you get a temperature from the ohms on the spec sheet because Motec reads a voltage. (it's not V = I x R) The temperature sensor requires a pull-up resistor (which is built into the Motec analog temperature sensor input pins).
  • Here's a helpful spreadsheet from some random forum, the original spreadsheet link can be found at the bottom of the page. https://www.syvecs.com/forum/viewtopic.php?t=495
  • Attached is the spreadsheet with values from the race grade spec sheet (RG_SPEC-0013).

         

  • The Bosch 0 261 230 217 and Bosch 0 261 230 099.0.1 sensors are interchangeable and have the same calibration. The Bosch 0 261 230 099.0.1 sensor is an already calibrated sensor with values found in Motec.
  • We have decided to run the temperature and pressure calibrations for the Bosch 0 261 230 099.0.1 sensor. The temperature sensor calibrations between the race grade spec sheet and Bosch 0 261 230 099.0.1 Motec calibration, however, the Bosch 0 261 230 099.0.1 Motec calibration has more points and the difference in voltage is very small. We are going with the Motec calibration because we trust that more than a manually calculated sensor (even though the calculations should be correct), there are more data points, and you cannot accidentally change a table value if you run a selected Motec calibration.

         

  • The pressure calibrations are exactly the same between the given race grade spec sheet and Bosch 0 261 230 099.0.1, we are selecting the Bosch 0 261 230 099.0.1 calibration for continuity. Make sure to set your reference voltage for the correct 5V sensor pin.
