Notes:

  • Not terminal blocks
  • SAE specification AS81714

CTD Bussing Modules for power distribution. (For larger pin sizes, not used on car)

CTJ1 Feedback Modules with various bussing arrangements. (Used for CAN and sensor voltage on car)

CTJ7 Grounding Modules, using either flange mount or stud mount, provide sealed multiwire grounding solutions. (Used for grounding on car)

Size 20 contacts match MoTeC pins, they use female pins.
