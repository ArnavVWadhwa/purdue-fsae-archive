Purdue Formula SAE : Installing Simcenter STAR-CCM+

Created by Unknown User (agarw162), last modified by Zhang, Chenkai on May 07, 2023

Installing STAR-CCM+ on personal laptops

  • Download STAR-CCM+ CAD Clients not Linux (Unless you have a linux OS). This is the one for personal windows computers

  • Requires a PoD Key which is a Power on Demand Key. The Power-On-Demand (PoD) session allows you to use a Simcenter STAR-CCM+ Power Session from a Siemens PLM Software public license server rather than a license server on your own network. We do have access to the license being hosted off the dock.ecn server at Purdue however a PoD is much easier to manage since the person with Steve Portal access gets to control each individual license

  • For PoD license, have team lead contact Chenkai


Installing STAR-CCM+ on the Scholar Cluster (only need to do this the first time)

  • Download the STAR-CCM+ installer for linux from the installers

  • Move the installer to your RCAC home drive, that you mapped in the previous step. 

  • If you’re using a cluster, install it to your desktop. Don’t use the default location in /opt/ if it tries to install it to /opt/. Don’t install it to scratch.

  • Navigate to desktop.scholar.rcac.purdue.edu in your web browser of choice. Open a terminal window after logging in with your Purdue credentials.

  • Follow the installation guide PDF uploaded in the same ‘installers for STAR’ folder. There’s a section on installation instructions for Linux. You’re going to want to use the express installation, with the POD licensing option. When it asks for installation directory don’t use the default path. Install it to your RCAC desktop. Not /opt/. 


Mapping your RCAC drive 

  • Map \\home.rcac.purdue.edu\username to your computer. It’s the same exact process that you presumably already went through for mapping your ITaP home drive and the mefsae drive. 

  • This rcac drive has 25 GB of storage, which should be enough for a STAR-CCM+ installation and whatever simulations you want to run, as long as you move simulations off of the RCAC drive once they’re completed.


Mapping your Scratch drive 

  • As of May 2023, this doesn't work

  • Map \\scratch.scholar.rcac.purdue.edu to your computer. It’s the same exact process that you presumably already went through for mapping your ITaP home drive and the mefsae drive.
