Purdue Formula SAE : Chassis Tab FEA Documentation

Created by Aaltonen, Emil Johan, last modified on Jan 11, 2023

To analyze the chassis tabs (The tabs connecting the pedal box rails to the chassis), we used a slightly elaborated version of the pedal box body and rails FEA. For more details on that, check out the "Pedal box rails and pedal box body FEA documentation" page.


The changes we made to the original FEA were:

Including the tabs in question.

To accommodate the new tabs, we added revolute joints which connected the tabs to the rails. On the throttle side the rail is split in two, so two joints per tab are used. All joints have 0 torsional stiffness and damping, as we are assuming the joint does not prevent the rail from rotating relative to the tab it is connected to.

The parts of the tabs welded to the chassis were given fixed support, with the assumption that the chassis will not deform significantly when the driver activates the pedals.

The mesh was done very differently. The rails, pedal box body, and throttle rail support insert were given a 0.1" mesh, and the tabs themselves had a .05" mesh. This was because the focus of the simulation was on the tabs, and precision elsewhere was of less importance.
