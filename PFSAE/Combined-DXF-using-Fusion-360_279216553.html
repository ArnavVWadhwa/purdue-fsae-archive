Purdue Formula SAE : Combined DXF using Fusion 360

Created by Schuyler James Cox on Nov 11, 2023

This is a guide to arranging tabs to in Fusion and then exporting to AutoCad, assuming you have imported the tabs in from NX.


Compiling and Arranging

In order to make the best layout, there are tools in Fusion to assist with making the optimized layout

  • Make sure all tabs have been exported as DXF from NX and imported to Fusion 360.
  • Open up a new document to make the layout in and save it under a new name
  • Drag the designs into the new document, and break the links once they have been added. (it is not required that you move them away from each other, they can simply be placed one on top of another).
  •  
    • Make sure to check units, and scale the parts accordingly, which can be done by navigating Surface→ Modify→ Scale. To enable the ability to select, click on a line before you select Scale, then deselect it once the Scale box has opened, and you will be free to highlight all of the sketches/bodies. Select any point to scale from, and choose your scale factor (DXFs are often in cm, so use a conversion factor of 2.54 to convert to inches)
    •  

  • Create a sketch of the stock that will be used for the operation.
  • Once the stock has been created, navigate Solid→ Modify→ Arrange. Then highlight the bodies to be arranged either from the workspace of the panel on the left (panel is easier to highlight, just select the top body then shift+select the last body)
  • Then navigate to the Envelopes tab and select the stock sketch. The bodies will automatically be arranged in the stock, in the most area- efficient way. If there are certain constraints about the distances between parts or the edge, this can be adjusted in the Envelopes tab under spacing.


Exporting to AutoCad

Once you have you layout, it will be a layout of bodies, so if exported as a DXF, it will crash AutoCad upon opening. Instead, we need to convert the layout to a sketch

  • Hide the stock sketch, so it is just the layout
  • Navigate to the Derive function by going Solid→ Create→ Derive. This will require you to save the project.
  • Once saved, a the derive function will pop up and will be ready to select geometries. Make sure that Destination is New File and Type is Derive Objects. Highlight the layout.
    • Select the Sketch dropdown to make sure the objects are sketches.
  • Once you hit Ok, Fusion will open a new document with the sketches in the preferred layout. Make sure to save this file under a suitable name.
  • After the file has been saved, go to File→ Export and select DXF from the dropdown.
  • Hit export
  • Open up AutoCad and select the Open button. Navigate to the save location of the DXF file
  • AutoCad often loads these files extremely small, so navigate View→ Unsaved View→ Top and the layout should come into view
