We currently have two M130 GPR ECUs.

MoTeC ecu's are available with different GP ranges, GP standards for General Purpose.

The GPR Package is perfect for our current needs, however if you ever want to run paddle shifters you will want to have a GPRP Package (I am not sure if you can just pay for an upgrade or if you need a whole new ECU). I think the GPRP will add features to M1 Build and Tune that will make running paddle shifting easier.
