Routing Lines

https://www.swooshtech.com/2021/06/18/linear-paths-in-nx-mechanical-routing/ - pretty good tutorial on how to start a line

     How I like to do it:

  1. Mechanical Routing
  2. Use Linear Path - but make sure to have dynamic axis turned on, and make sure your sketch is oriented so it matches the sketch origin, otherwise the lines go insane
    1. After I right the sketch I like to just drag the axis until it gets to a point I like (e.g. a certain distance away)
    2. an "overhead" view is helpful for this (whatever axes the pipe runs along)
  3. Use "Assign Corner" to insert a certain bend radius at point corners – DO THIS BEFORE INSERTING SPACE RESERVATION FOR TUBING
  4. Add tubing with the space reservation
