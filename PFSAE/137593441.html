Purdue Formula SAE : Anti-Roll-Bar (ARB) Combined Stiffness Calculator

Created by Unknown User (reyno105), last modified on Aug 28, 2020

This tool can be used to determine effective ARB rates in terms of additional spring rate equivalence from combined axial torsion in the arb, bending of the arb (if cantilevered), and bending of the arb arm if being used as adjustment.

ARB Rate Calculator

This combines the stiffnesses of the tube torsion, cantilevered bending, and arm bending to come up with total stiffness of the system. It calculates an equivalent wheel rate addition. This is means the role stiffness increases by the same amount that a wheel rate increase of the calculated value would. This makes it simple to think about ARB rate changes relative the spring rate changes.

It can be used to design ARB systems of two types:

Arm length adjustment

The length of the arb arm is adjusted by changing where the drop link attaches. This is typically the simpler and thus better design.

Arm stiffness adjustment (blade style)

A blade is rotated and locked in place to adjust the stiffness of the arm. This is harder to analyze than above, and should be avoided unless packaging is an issue for moving the drop link. In the image below, this is the case because of how close the arb had to be to the rocker.



The stiffness of the ARB as a whole comes from three effects:

Tube torsion

This is the primary effect being used. The tube is designed to twist to provide the correct stiffness that couples the wheel motions on the left and right side of the car. The length of the arms twisting the tube serves as the adjustment method, unless a blade style system is used where the arm stiffness can be adjusted. Even in this case, tube torsion still plays a role that needs to be accounted for.

Tube bending due to cantilever

If the mounting points for the arb tube are far from the arms, the tube bends enough to significantly reduce the system stiffness. In this case, the effect must be accounted for by beam bending calculations. Note: this calculator assumes that the bushing used to mount the arb tube fixes it (prevents rotation at that point). With a soft, small bushing, this may not be a good assumption. Should validate system stiffness in any case.

Arm bending

If a blade style arb is used as an adjustment, the arm bending is a significant modifier of system stiffness. If the length of the arm is being used, it is likely that the arm stiffness is high enough to have a negligible effect, so the arm stiffness may be set to an arbitrarily high number (say 1e10).

(Motion Ratio)

The position the drop link attaches to the rocker drives the motion ratio and can also be used as an adjustment if the stiffness range from the above tuning methods is not sufficient. This does not change the stiffness of the arb, but it changes how much the arb is moved by a given amount of wheel movement.


Blade Initial Sizing Sheet

The initial sizing sheet just does beam calculations for a square cross-section beam to provide a stiffness and stress results. This will give an initial estimate before going to FEA. It had less than 25% error for stiffness when used for making the design shown here. The stress results are dicier because the off axis loading is hard on the thing. So expect stresses up to 80% higher than predicted.

This is only needed if using blade style ARB.


Blade FEA Post Process Sheet

If using a blade arb, there is not going to be any way around doing some FEA. For this reason, it is best to avoid if possible. However, this tool provides the inputs and post processing to create a curve of blade stiffness vs. angle and then to interpolate on that curve to select the angular positions that will provide desired increments in stiffness.

This is only needed if using blade style ARB.

Theory:

This tool takes an angle of rotation, converts it to cartesian components of a unit load (1 lbf) and provides those to the user. Those can then be put into FEA as forces in a parametric study for a sweep of angle from 0 to 90 degrees. The tool then receives the displacements and takes the dot product with the unit vector of the direction of force. You then have a unit load (1 lbf) and a resulting displacement in the direction that the drop link is pulling, so a stiffness can be calculated. 


Instructions:

  1. Bring ARB design into ANSYS static structural:
    1. Fix where it held into bar
    2. Apply a remote force to the bolt hole where the drop link attaches. The force should be applied at in the middle of the rod end spherical bearing.
    4. Make the remote force a components based force with the y and z components parametric
    5. Create two directional deformation results
      1. Select the curve around the bolt hole for each
      2. Set one to y and the other to z deformation
      3. Make the maximum deformation parametric
    6. Create an equivalent stress result
      1. Make the maximum value parametric
      2. Most likely this will be on the blade, providing the result you want. But you may need to select particular geometry if not
    7. Make sure to have at least 4 element through the thickness of the blade to avoid inaccurate stiffness results. Set the element order to be quadratic
      1. The below mesh ran in 1/2 min with 16 gb of ram
  2. Set up the parametric study
    1. Copy the input force vector from excel in the BLADE FEA POST PROCESS tab
    2. Paste this into the y and z force component parametric study rows
    3. Run the study
    4. Copy the directional deformation and stress results from the parametric results
    5. Paste them into the corresponding cells
    6. Now desired stiffnesses and required displacements can be entered, and the sheet will interpolate from the FEA results to provide required rotation angles and resultant stresses
      2. Note: these inputs are calculated in the following sheet
