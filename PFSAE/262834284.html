Purdue Formula SAE : Harness: RapidHarness

Created by Paradise, Hunter, last modified on Dec 06, 2023

Background:

Harness CAD is an immensely useful tool for designing a harness. Not only does it offer wonderful tools for keeping track of every component to be purchased or used within the harness but it is also invaluable when it comes to visualizing the technical aspects of the harness. Normal CAD simply does not capture the complexity of individual wires, splices, loop lengths, etc. and things will be missed if that is the only tool used.

Tutorial:

This video walks through the basics of RapidHarness use, but does not cover team practices.

Tips:

  • Version everything so that it can be used and referenced by anyone on the team
  • Leave adding wire gauge and color until wire lengths are finalized, it is difficult to keep track of them once they are declared
  • Use previous harness' layout and re-bundle it based on the new routing instead of creating a new one
  • Connector designs can be changed quite easily
  • At the end check that no wires are doubling back on themselves unintentionally, this can sometimes happen inadvertently when adjusting bundles
  • Many sensors will have some length of wire between them and their connector, the harness length should be reduced by this amount if possible to avoid unnecessarily long connections
  • Wait to add wire lengths until after CAD is frozen, it is difficult to change them after the fact if accounting for sensor wire lead lengths

Timeline:

  • Summer → Preliminary harness CAD created
  • Design Season → Edit preliminary harness CAD based on team changes as much as possible
  • Post Main CAD Freeze → Finalize remaining items such as wire lengths, AWG, color, etc.
