Purdue Formula SAE : Track Map - Course Setup Notes

Created by Unknown User (jpeerbol), last modified by Unknown User (agarw162) on Feb 21, 2023

Accel: 75m (246.06ft) long, +4.9m (16.08ft) wide, spacing approx. 6m (19.7ft), .3m (1ft) roll out
Skid: 18.25m (60ft) center to center, 15.25m (50ft) id, 21.25m (70ft) od, 3m (9.8ft) wide path, 16 cones inside each circle, 13 cones outside. Out lap on right circle, timed right, left, left timed
Autocross: The Autocross course will be designed with the following specifications. Average speeds should be 40 km/hr to 48 km/hr
    1. Straights: No longer than 60m (197ft) with hairpins at both ends
    2. Straights: No longer than 45m (148ft) with wide turns on the ends
    3. Constant Turns: 23m (75.5ft) to 45m (148ft) diameter
    4. Hairpin Turns: 9m (29.5ft) minimum outside diameter (of the turn)
    5. Slaloms: Cones in a straight line with 7.62m (25ft) to 12.19m (40ft) spacing
    6. Miscellaneous: Chicanes, multiple turns, decreasing radius turns, etc.
    7. Minimum track width: 3.5m (11.5ft)
    8. Length of each run should be approximately 0.80 km (2625ft)
Endurance: The Endurance course will be designed with the following specifications. Average speed should be 48 km/hr to 57 km/hr with top speeds of approximately 105 km/hr.
    1. Straights: No longer than 77m (253ft) with hairpins at both ends
    2. Straights: No longer than 61m (200ft) with wide turns on the ends
    3. Constant Turns: 30m (98.5ft) to 54m (177ft) diameter
    4. Hairpin Turns: 9m (29.5ft) minimum outside diameter (of the turn)
    5. Slaloms: Cones in a straight line with 9m (29.5ft) to 15m (49ft) spacing
    6. Miscellaneous: Chicanes, multiple turns, decreasing radius turns, etc.
    7. Minimum track width: 4.5m (14.8ft)
    8. Designated passing zones at several locations

DRS Test Track: The DRS track to correlate with Lap Sim will be design with the following specifications. Three straights of varying length connected with various radius turns forming a triangular track.

  1. Straights (1,2,3): 77m (253)ft, 60m (197ft), and 23m (75.5ft)
  2. Constant Turns: 9m (29.5ft) outside diameter
  3. Minimum track width: 3.5m (11.5ft)
