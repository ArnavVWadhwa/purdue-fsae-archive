Purdue Formula SAE : FW-14-17

Created by Zhang, Chenkai on Jul 11, 2023

Simulation: CZ-046

Changes/Justifications

Description of the Change

Picture

Justification

Increased root chord

Increasing root chord decrease outwash angle, which should be good for front wing performance. However, loss of outwash will negative impact undertray. There is also a slight bit increase in outwash angle, since root chord also contributes to the height of the flaps. So that will make rear wing a tiny bit worse, shouldn't be noticeable though.

Another worry is that with increase chord, the flow conditioner is not in a good place anymore. It's increasing adverse pressure gradient for front wing flaps quite a lot now. So we will see.

CFD Results Analysis

CFD Picture

Description

UT inlet flow tiny bit worse than before, but it's not a 5 pt loss. I was already suspicious of the 7 pt UT gain from last sim. From this I conclude previous sim UT was wack.

Flow conditioner pressurization is not as big of deal as I thought, but it's definitely not ideal in its current position. Need to be moved downstream.


Not really a discovery for this iteration, but I found that reduced tip size is contributing to the pressurization of front tire on downwind side, leading to loss of load in the downwind side front wing.
