Purdue Formula SAE : Mapping Formula Network Drive

Created by Unknown User (jkrawitz), last modified by Unknown User (jpeerbol) on Aug 27, 2021

For Windows:

  1. You need to either be connected to PAL 3.0 or connected to Purdue's VPN. Instructions for that here: https://www.itap.purdue.edu/newsroom/200316-webvpn.html
  2. Go to File Explorer, 'This PC' → 'Map Network Drive' 
  3.  Make sure you choose 'Z:' to represent this drive, a lot of our software already uses that address. In the 'Folder' box type \\project.ecn.purdue.edu\mefsae. 
  4. When prompted enter your Purdue username(first part of email) and password(the one for your email, not 2FA)



For Mac:

  1. You need to either be connected to PAL 3.0 or connected to Purdue's VPN. Instructions for that here: https://www.itap.purdue.edu/newsroom/200316-webvpn.html
  2. Open a Finder Window and Command K
  3.  Type smb://project.ecn.purdue.edu/mefsae/ into the connect to server 
  4. When prompted enter your Purdue username(first part of email) and password(the one for your email, not 2FA)
