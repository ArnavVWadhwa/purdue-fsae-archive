Purdue Formula SAE : Yearly Exhaust Updates

Created by Unknown User (kruege18), last modified on Nov 01, 2019

2020

In 2020 the exhaust was completely redesigned to satisfy new packaging requirements, and target a new RPM.

This year, due to our move to dual radiators positioned in the side pods, exhaust had to exit the rear left side of the car. 

This year we chose to use a 4-1 collector with a 12 degree taper. The reasoning for this is that we had no way to validate that using a 4-2-1 system would be better, but we knew a 4-1 would certainly be lighter, and easier to package. The taper was chosen because it was the steepest recommended taper angle in "Four Stroke Performance Tuning" by A. Graham Bell. 

Since we were unsure of the specific effect that each individual runner length has on performance, we made them all the same length, 20.3 inches. The length calculation used was based on the same equation used to calculate intake runner length. This has not been validated. Our RPM target was 10000. This decision was based on a histogram of average RPM while under 85% throttle during the fastest 2018 Autocross run. This run was chosen because it was a fast run performed by a very talented driver. This run represents ideal "pushing" conditions for the car on track, similar to what we would hope to see during autocross/endurance runs at future competitions.
