Purdue Formula SAE : FW-14-15

Created by Zhang, Chenkai, last modified on Jul 07, 2023

Simulation: CZ-044

Changes/Justifications

Description of the Change

Picture

Justification

Endplate gurney size reduction

Since there is no longer a vortex channel here, endplate gurney width can be reduced. This increases main wing planform area which should be some downforce (partially offset by reduction in footplate area since it still does produce some downforce). More importantly, this moves all wing tip and strake vortices outboard, which should be good for UT performance.
Reprofiled all elements

Since last setup lost quite a bit of downforce due to reflex and bad flow alignment on the downwind side, this should help reduce those effects. Local concave curvature is significantly reduced to reduce reflex. This also increased planform area, which again is good.

On the upwind side, due to increase area of low diffuser angle region, strake vortices can be better controlled, which should again lead to improved UT performance.

Due to the reduction in local outwash angle, the third element camber on the x slice is effectively reduced. I don't know if this will be enough, but probably not.

Top element slot overlap significantly increase and angle of attack slightly increased to further reduce reflex caused by the flappage (yes flappage is now a word).

CFD Results Analysis

CFD Picture

Description

Pretty much as what I was afraid of, the entire outwash section is now getting pressurized.

Moving endplate and strakes outboard really didn't do too much, and that's because the initial losses formed early on where the difference is is upwashed. There is an improvement in high front wing wake, but that hardly matters.
