Purdue Formula SAE : Microsoft Project

Created by Unknown User (jpeerbol), last modified on Apr 27, 2020

Microsoft Project is available on the Microsoft Azure Dev Tools for Teaching program (formerly called Microsoft Imagine or Microsoft Dreamspark).

The software is available at:

https://azureforeducation.microsoft.com/devtools

Sign in using their Purdue username and password.

The available Software is under the Software link in the center blade.

Search for project professional 2019. Download the software. You will need a product key to register the software, this can be found where you downloaded the software, you need to click view key.



This software can only be used for educational activities.

This is current as of 4/27/2020.

We currently use Microsoft Project Professional 2019.


Microsoft Project is available to your students under the Microsoft Azure Dev Tools for Teaching program (formerly called Microsoft Imagine or Microsoft Dreamspark).

Your students can access available software under this program at:

https://azureforeducation.microsoft.com/devtools

They will need to sign in using their Purdue username and password.  The available Software is under the Software link in the center blade.   This software can only be used for their educational activities.
