Purdue Formula SAE : Adams File Setup

Created by Kurisko, Keith Gregory, last modified on Dec 04, 2021

File Locations: Z:\Knowledge_Base\01-Suspension\06-Adams_Car_Vehicle_Files

The primary files are private2.cdb and fsae_2012.cdb


Add the desired folders above by:

  1. Go to Tools/Database Management/Add to session

2. Navigate to the file location Z:\Knowledge_Base\01-Suspension\06-Adams_Car_Vehicle_Files

3. Select one folder and name the alias exactly as seen in the folder as shown below


You MUST select a directory path to where all your simulations will be saved to:

1: Go to File/Select Directory

2. Select your desired folder where all your simulations will be saved to
