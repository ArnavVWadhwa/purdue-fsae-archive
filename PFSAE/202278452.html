Sensor Levels

  1. Critical to run the car
    • Cannot finish a worst case competition endurance without it
  2. Car will still run but still critical for safe driving
    • Will be difficult to diagnose an engine issue without this data
  3. Can run without but data will not be as useful
    • Advanced features like launch control will not work when running without
  4. Optional Engineering data

Bosch 0 261 230 217

Milspecwiring.com M 16-4360

 

see calibration notes

We are not using these Temperature calibration values, We are using the MoTeC 0 261 230 099.0.1 Bosch calibration for temperature and pressure.

We have decided to run the temperature and pressure calibrations for the Bosch 0 261 230 099.0.1 sensor. The temperature sensor calibrations between the race grade spec sheet and Bosch 0 261 230 099.0.1 motec calibration, however the Bosch 0 261 230 099.0.1 motec calibration has more points and the difference in voltage is very small. We are going with the motec calibration because we trust that more than a manually calculated sensor (even though the calculations should be correct), there are more data points, and you cannot accidentally change a table value if you run a selected motec calibration.

The pressure calibrations are exactly the same between the given race grade spec sheet and Bosch 0 261 230 099.0.1, we are selecting the Bosch 0 261 230 099.0.1 calibration for continuity. 

See notes: https://wiki.itap.purdue.edu/x/0oYOD
