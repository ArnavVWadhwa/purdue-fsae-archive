Purdue Formula SAE : FEA Steps

Created by Nowakowski, Justin Thomas on Sep 24, 2023
  1. Add part / assign material properties
  2. Determine loading needed (torque transfer / tcp loads)
  3. start with first load case
  4. go to mesh / method and geometry selection and try hex dominant first
    1. change element size to small value (roughly 0.1 to start (and work downwards)
    2. under 10% pyramids is ideal
    3. jacobian above 0.66
    4. aspect ratio between 1 & 5 (try below 3)
  5. If hex dominant doesn't work, try multizone
  6. Once mesh is complete move onto boundary conditions
  7. For center, and initial torque transfer
    1. pin joints in hub
    2. shell holes are fixed
    3. as if torque is delivered and wheel doesn't move
    4. Keep step in force control at 1s
  8. Insert static structure
    1. start with cylindrical support on outer holes
    2. keeping all fixed
    3. use moment and define magnitude and direction
    4. solve
  9. 79.62
  10. 91
  11. Ensure analysis is accurate before changing model
