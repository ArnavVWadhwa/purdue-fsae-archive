Purdue Formula SAE : Jigs

Created by Unknown User (ehoerl), last modified by Unknown User (jpeerbol) on Jan 20, 2020

Having two jigs per tube is great for welding support, especially critical for longer and more critical tubes.
Jigs should be places at least 3-5 inches away from the node if possible to make welding easier.


Tab jigs are incredibly important. Sharing the same jigs for similar tabs is convenient, but also have multiple copies of that same jig is important to decrease wasted weld time waiting for a weld to cool.
