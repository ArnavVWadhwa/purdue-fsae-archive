Purdue Formula SAE : Teamcenter: How to Change Item Name/ID/Description

Created by Liang, David C, last modified on Aug 27, 2023

POC: Liang, David C 

Prerequisites:

This guide assumes you already have Teamcenter downloaded. Below are links to the page on how to use Teamcenter:

  • Teamcenter: How to use on personal computers
  • Teamcenter: How to Install


Step 1: 

Right click on the item you want to change and click on Edit Properties.




Step 2:

This window will pop up. Here you will want to click OK, which will check-out the item to you. 


Step 3:

Once this page opens up, click on All.


Step 4:

If you scroll down a little, you can edit the description. 


Step 5:

Scroll down until you see the ID box which allows you to change it. 


Step 6:

Copy the changed ID name and past it into the Name category. Finally click the Save and Check-In button and click Ok on the next pop up. 



Note: You can check if the name on Teamcenter or if you open it up in NX. It should look something like the the name below:


If you see something like this name below:

The ID and the name of the file are not the same.
