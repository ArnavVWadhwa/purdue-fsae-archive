Purdue Formula SAE : Rear Wing Structures Notes

Created by Unknown User (aroychow), last modified on Apr 08, 2021

This is more of a flow-of consciousness page for the time being until the structures work is entirely complete. Afterwards, this'll (hopefully) be compiled/organized into a resource for whoever is doing future RW structures, just as a reference.

Structures Loop:

  • Rough out initial design in CAD
  • Hand calcs to find approx loads on bolts, tierods, certain parts of ribs/swans
    • Do FEA for anything you have hand calcs for
  • Adjust the pass based on what you see in FEA
    • Add or remove material based on what you see, change some shapes, etc.
  • Most of your repetition comes from Carbon
    • ex. how many plies?

Below are the expressions used in the LAYOUT file for RW mounting, (just for record keeping):

NOTE: This table is outdated as of 8-April-2021. If needed, can go back through and parametrize most of the CAD.

ExpressionDescription
MP_CarbonOffset

Value of carbon thickness of RW mainplane. Used to offset the airfoil curve inwards for proper rib sizing.

Current placeholder value (CPV): 0.04in ( approx 1mm)

Swan_BoltHoleSizing for the bolt holes on the swans/ribs (Use the NAS Bolt Calculator below)

Rib_Mounting_Thickness

2*Swan_Bolthole. Makes sure that the sticky-out bit of the rib that attaches to the swan is double the width of the bolt in it.
Swan_Wall_Thickness

Width of outer perimeter of mainplane ribs (Symmetrical offset, so it gets doubled)

CPV = 0.125in

Swan_Truss_Thickness

Thickness of trusses on swans (Symmetrical offset, so it gets doubled)

CPV = 0.05in

Swan_InnerFillet_Radius

Radius of internal fillets in swans. Not actually fillets, they're circles tangential to the corners, which break fewer things for some reason (credit to Chase)

CPV = 0.3in (NX defaults to diameter, so this is the diameter value)
