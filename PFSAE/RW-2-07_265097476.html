Purdue Formula SAE : RW-2-07

Created by Matthew James Lamear, last modified on Jul 07, 2023

Simulation ML-018

Changes/Justifications

Description of the ChangePictureJustification

Increased chord of ME to 23 in, decreased chord of 2nd el to 7.5 in, decreased chord of 3rd el to 2.65 in.

Redesigned main element to allow for 23 in.

Prefacing this design by saying I'm very worried about flow sep on the main element. The amount of flow turning is pretty extreme, but if it doesn't separate the DF generation should be very substantial.

2nd and 3rd elements are primarily for pressure recovery. Note that the 2nd element is at 57.5 deg AoA.
