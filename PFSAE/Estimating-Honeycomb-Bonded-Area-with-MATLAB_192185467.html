Purdue Formula SAE : Estimating Honeycomb Bonded Area with MATLAB

Created by Unknown User (hammerly), last modified on Oct 14, 2021

This tool is useful for estimating the honeycomb bonded area for SES.

Scan the pre-crushed side of the impact attenuator

Any scanning application (e.g. Genius Scan or Microsoft Lens) should work. Scan the pre-crushed side of the IA as shown below.

Figure (1) Using Genius Scan to scan the precrushed honeycomb

Export the scan as a .jpeg or .png. Name it something meaningfulÂ  and create a folder in the drive for this.

Open the Matlab file below. Save it to the aforementioned folder.

honeycombReader.m

Open the .m file and edit the inputs so the script can read your image.

Figure (2): Inputs for the script

This script automatically locally brightens darker areas so the cutoff needs to be adjusted. The script displays the final 'crunchy' image so you can adjust the cutoff variable and see how it changes which pixels turn white/black.


Figure (4): Command window Matlab output

This script outputs a few values to the command window. bondedAreaCount is the number of white pixels in the image, nonbondedAreaCount is the number of black pixels in the image, and bondedArea is 32in^2 * white pixels / total pixels.

Figure (5): Fully processed 'crunchy' image
