Purdue Formula SAE : Brake Discs

Created by Wright, Trenton Scot, last modified on Jan 17, 2022

wrigh577_BRAKE_DISC_FIXTURE_BLOCK_dwg1 (1) (1).pdf


High Level Steps:

Stock Prep, Waterjet

  • Start with sheet metal stock (4130 steel) was waterjet to make donuts to attach to fixture block during CNC later. Make each donut about 0.25 in larger than the final diameter of the rotor, and the inner diameter 1.00 in. The general idea is to use a fixture block with a central bolt to hold the stock for the first operation (drilling holes), then insert four more screws to secure it for outer and inner radius milling. 

Create fixture block

  • I started with a block of aluminum some width and length greater than what's pictured on the drawing, then cut it down to ~1.7 inches in height using a band saw. The thickness itself isn't really important, as long as it's >1 in so a bolt and disc can be fixed during the cnc process.
  • Drill and thread the center hole. This is going to be used to clamp the disc on the mill and drill out all the holes
  • Drill out all eight holes on the fixture block, of which four in a standard + pattern fit with the front rotor, while the x pattern fits the rear
    • This ended up not working because the holes were drilled when the coordinate axes were not aligned with the central hole. Instead, John used the CNC mill to drill four holes based on the cam, which were then tapped for 10-24 bolts. 

CNC Mill

Drill Vent Holes

  • First, clamp the disc to the fixture block using the central hole. Instead of using washers, which can change where the rotor is secured drastically, make a specific piece to fit in the 1.00 in diameter inner hole for the 1/2 in bolt that extends slightly inwards. It's easier to explain by seeing it in a picture.  
    • The first operation is cutting all the outer vent holes, done using a (I forgot the name) drill, 25 size for 0.185 in holes. 
    • No peck drilling was required because of the bit being used, which drastically speeds up the process.

Inner and outer circumference

  • After the first op, the mill should stop on its own. The rotor should already be aligned with the previously drilled and tapped holes for the vent hole securing locations.
  • John recommended using a coated carbide bit which needs EXTREME CAUTION when ramping in the inner radius. Slow the feed rate for this step to no greater than 7.5
