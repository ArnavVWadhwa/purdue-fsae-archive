Critical inputs for an accurate analysis:

  • Camshaft

What was done to ensure the critical inputs were accurate? 

  • Camshaft - read when engine runs
  • crankshaft - read when engine runs
  • stator - use AD2 to read output voltage when dyno running
  • starter - use multimeter to check what input to this wire would be when turned on
