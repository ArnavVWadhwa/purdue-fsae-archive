Purdue Formula SAE : Knock Input

Created by Paradise, Hunter, last modified on Nov 24, 2023
  • Digital Signal Processed knock detection.

  • 12-bit analog to digital converters with -0.562 to 5.562V input range.
  • Can also be used as high-speed AV inputs.
  • Input impedance equivalent to 98k with 2.5V offset.
  • 50KHz bandwidth.
  • Transient input +/-200V.
  • Continuous no damage input voltage range -30 to +35V.
  • 2 Analog Knock pins on the M130.
