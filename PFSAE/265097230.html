Project Scope (Present problem statement and stakeholders):

Problem Statement

  • A reliable dash that provides safe and easy access to controls and data needed by the driver while also allowing engineers to access the harness easily.


Stakeholders

  • Chassis
    • Mounting - tabs, depended on FRH
    • c125 data
  • Aerodynamics
    • Carbon plate
  • Suspension
    • Data logging through c125
  • Powertrain
    • Start button
    • launch control

Design Requirements (Present known requirements, ie. Vehicle Performance, rules, and define unknown requirements):

Dash

On front

  • C125
  • Estop - cockpit main switch - Rule
  • starter button - Rule
  • launch control rotary
  • auto shifting switch
  • log button
  • fan switch
  • Plate

On back

  • USB to CAN
  • Bulkhead type connector
  • Back cover

Other requirements

  • Low number of connections inside dash
  • Water resistant
  • Labels for controls
  • Must be within front hoop - Rule
  • room for brake bias adjuster - Not responsible

Maximize Competition Points:

  • Start Button
  • Estop
  • Inexpensive
  • All controls must be contained within the cockpit
    • Display can be outside but preferred to be inside

Driver Safety

  • Estop
  • All data/controls are easily visible
    • Accessed easily
  • Driver does not need to reach outside cockpit at any time

System Reliability

  • Start and kill switches work without fail
  • Displayed data is current and accurate
  • Dash components are securely mounted
  • Display Data is stored and can be accessed at later times

System Serviceability

  • All switches/controls must be easily accessible from within the cockpit
  • Conrols must be labeled well

Driver Confidence

  • MoTeC c125
    • Can Only display limited amount of Data
    • Decide what data driver needs

Design Overview (Rough concepts for solution + Trade study with a recommended path forward):

MoTeC c125

  • CAN buses/RS232
  • Inputs
    • 6 x analogue voltage inputs
    • 2 x analogue temperature inputs
    • 2 x digital inputs
    • 3 x speed inputs
    • How many to display?
  • Find out what is possible to display within budget

Kill Switch

  • Connected to shutdown circuit
  • Minimum 24 mm diameter

Master Switch

  • Grounded Low Voltage Master Switch
  • Tractive Master Switch
    • Included in Dash?
