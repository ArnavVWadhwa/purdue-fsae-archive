PDM15

Software: PDM Manager

Connection: OBDII (if present), CAN port.

Overview: takes power from the battery/rectifier through the master switch, safely manages, and distributes power to the rest of the car. Built-in short protection drops the voltage to a consistent ~13V to avoid fluctuation in voltage levels.

M130 GPR

Software: M1 Tune, M1 Build

Connection: OBDII (if present), Ethernet (black)

Overview: governs the engine based on sensor input, provides data to feed to the display, and logs data.

Software: Dash Manager, Display Creator (for the adventurous)

Overview: displays data, reads some data lines, and passes them to other modules

LTC 4.9

Software: LTC manager

Connection: OBDII (if present), CAN port

Overview: reads and parses input from the lambda sensor
