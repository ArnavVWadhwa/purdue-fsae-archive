Purdue Formula SAE : Water Brake Fluid System

Created by Unknown User (reyno105) on Sep 05, 2020
  • Reservoir: the big tub. (Note it might actually need to be that big so that we dont need fancier water cooling)
  • Pump: https://www.waterpumpsdirect.com/FloTec-FP4012-10-Water-Pump/p72386.html
    • Go with the FP4012. 
    • This chart shows what flow rate it provides at 40 psi with various amounts of elevation (head) supplied:
    •  
    • At 0', which is essentially what we'll have, it can give 8 gpm at 40 psi
  • Water hammer arrestor: the below pages indicate we'll need a water hammer arrestor.  This prevents water hammer (a shock) if the control valve is suddenly shut by having an air compartment that can be compressed:
    •  https://www.amazon.com/Everflow-Supplies-HAWM0034-NL-Hammer-Arrestor/dp/B07PC6W941/ref=psdc_3226897011_t1_B000BO6G8K
      • something like this, just make sure the pressure rating > 77
    • Since this is before the first regulator, it needs to be able to handle the max pressure the pump can supply
      • This is the max pressure shutoff which is 77 psi for this pump, 
  • Pressure Regulator:
    •  https://www.amazon.com/Hourleey-Pressure-Regulator-Screened-Trailer/dp/B085HCXPQL/ref=asc_df_B085HCXPQL/?tag=hyprod-20&linkCode=df0&hvadid=416796348214&hvpos=&hvnetw=g&hvrand=688423801507290000&hvpone=&hvptwo=&hvqmt=&hvdev=c&hvdvcmdl=&hvlocint=&hvlocphy=9016722&hvtargid=pla-904506718778&psc=1&tag=&ref=&adgrpid=93124985239&hvpone=&hvptwo=&hvadid=416796348214&hvpos=&hvnetw=g&hvrand=688423801507290000&hvqmt=&hvdev=c&hvdvcmdl=&hvlocint=&hvlocphy=9016722&hvtargid=pla-904506718778
    • One of the dynomite pages says the pressure should be no more than 60 psi, so we need to cut to 60 psi or lower here
  • Load control valve:
    • looks like this works by splitting flow (some thru water brake some back to tank)
    • Should maybe just buy the one from dynomite?
