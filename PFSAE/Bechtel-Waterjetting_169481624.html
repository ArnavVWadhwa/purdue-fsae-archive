Preparing parts:

  •  in the BIDC no longer supports using flowPath for pathing (will not be approved), thus use Fusion 360
    • Join BIDC channel in Fusion (have to have a TA set it up) so that you always get the most recent tool updates
  • Fusion can be a pain for importing parts, either lay parts out (not DXFs) in NX where they can be constrained to a plane and then open that assembly (Fusion can open .prt files) in Fusion. If parts need to be moved or added this can be done in NX and then re-opened in Fusion. Fusion has a terrible constraint system (so don't even try it it's just a waste of time)
    • OR export dxfs and layout parts in dxf editor and then import that dxf into Fusion as a sketch and extrude
      • Sketches are usually broken so this method is not recommended
    • Fusion is a pain at first but it works much better for generating pathing than flowPath
  • This webpage is very helpful for creating pathing: https://www.purdue.edu/bidc/resources/waterjet-cutting/steps-to-a-waterjet-project/
    • If you have a layout of parts, do each part one after another
      • Start with interior elements like holes or cuts, and then move to path the outside. This will prevent the part from moving during cutting of small elements (the part can break loose if the outside is cut first) and will make sure all elements are made properly. 
      • Waterjet will cut in order of geometry/corner selection so start by clicking inside features, then clicking on outside, then moving to the next part
        • if this does not work just split each part into a separate process
      • For tabs include 2+ when using thin sheet metal (1/16th or less), this will keep the part from breaking off, bending, or hitting the cutter (or all the above) during cutting
        • If any bigger only one is needed (if you have more part will be impossible to break off)
      • If cutting small features, lead-in radius will need to be small (less than 0.05 inches) to prevent feature from becoming distorted
        • if need be this can also be adjusted in flowPath on the computer after .ord file is created.

Waterjetting:

  • If cutting thicker steel, no clamping is needed due to its weight
  • If cutting thin sheet metal use clamps and weights to secure stock to the table. Stock should not be able to move if jostled. Make sure that weights don't get in way of cutting head
  • Open part in flowCut on the computer, select metal, and thickness
  • Create work origin based on where you want to cut (TA will help with this), also set head height using shim that they have there.
  • Make sure to move head manually, tracing out the cutting area to make sure there's no interference with clamps and weights if installed.
  • Always check that work origin was set by moving head up and away (up so that it doesn't potentially crash into the table) and clicking 'move to work origin' on computer
