Changes/Justifications

Description of the Change

Picture

Justification

Four element

Four elements to better take advantage of maintain better suction throughout all elements.

First element runs a lower angle of attack than PF23 rear wing due to stagnation point and pressure side separation encountered with PF23.

First element is also higher due to reduced size, which should allow the first element to generate more of the suction, increasing efficiency.

Elements decrease in size going from first to fourth element to ensure efficiency.

Suction peak moves forward and pressure recovery more drastic going from first to fourth element since the priority changes from generating local load to control pressure recovery. The fourth element very much resembles Liebeck airfoil due to the fact that I want all the suction at near the leading edge close to the previous element, since this element does not generate downforce and its sole purpose is to control pressure recovery.

Endplate cutouts

Top cutout to reduce endplate area reduce moment generated by crosswind for rollover margin.

Leading edge to reduce drag - proved quite effective by Matt's rear wings.

Taking inspiration from Matt for the lower cut to generate some downforce. However, instead of making it a continuous curve that goes from horizontal to vertical, there is a sharp end to the curvature, which allows flow to separate there and act like a gurney, which increase downforce and efficiency.

CFD Results Analysis

CFD Picture

Description

Separation across the endplate leading edge

Stagnation point on the third element too high for my liking. Although this is a flap, so it's not a big deal.

Second element can do with better flow alignment to avoid the reflex. That means more angle of attack since I don't want anymore slot overlap than I do already.

As far as loading goes, main element can do with more loading. It's hard to see but I think there are two suction peaks on the main element which I want to avoid. So that means higher camber near the center.

Third element has too much suction, which at this angle of attack is mostly drag. The solution is probably smaller slot gap to reduce energy delivery to the suction side of the third element. The second element angle of attack is going to be increased to resolve the reflex issue, so there will be so energy loss from that as well.
