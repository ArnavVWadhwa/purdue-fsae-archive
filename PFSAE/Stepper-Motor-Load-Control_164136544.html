Purdue Formula SAE : Stepper Motor Load Control

Created by Unknown User (osborn42), last modified on Dec 06, 2020

There are two ways to control the stepper motor:

  1. Manual
    1. Set Automated Control to "Not in Use"
    2. Set the position of the stepper based on the percent load
    3. Use Page Up or Page Down to adjust the percent load by 1/2 a percent to fine tune the position
  2. Automated Load Control
    1. Set Automated Control to "Enable"
    2. The stepper motor will operate in 'Manual' mode until the engine is running then it will try to adjust the load to keep the engine at the "RPM Target"
    3. Adjust Kp, Ki, and Kd. as necessary to tune the system
    4. Be sure to reset the position of the stepper to 0 when you reset the ECU. See Design Notebook for details on how to reset the stepper motor position
