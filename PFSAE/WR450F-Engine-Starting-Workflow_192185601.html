Things needed for crank positions this weekend

  • spark wiring
  • crank position sensor wiring
  • starter wiring
  • dash wiring

Things needed for cam position

  • cam pos wiring
  • cam sensor installed
  1. Ensure crank timing is set correctly
    1. Ensure Sparkplug has the capability to fire
    2. Set Spark timing to 0 degrees BTDC on M1
    3. Using a timing light on engine and crank
      1. aim timing light at the location specified
        1. light should flash on 0 degrees BTDC
        2. sparkplug should ignite at the exact same time
