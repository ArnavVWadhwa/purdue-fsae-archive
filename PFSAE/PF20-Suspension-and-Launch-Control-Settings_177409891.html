Purdue Formula SAE : PF20 Suspension and Launch Control Settings

Created by Unknown User (reyno105), last modified on Apr 19, 2021


Suspension and Aero Setup

Dry Weather

  • Tire pressures:
    • Usually cold set to 10psi, looking to hold around 12psi hot, which tends to be what we get from 10. 
  • Brake bias: 54% front (by pressure)
  •  Camber:
    • Front: -0.5
    • Rear: 0.0
  • Toe:
    • Front: -0.5 (toe out)
    • Rear: 0.5 (toe in)
  • Springs:
    • Front: 350 lbf/in
    • Rear: 350 lbf/in
  • ARBs:
    • Front:  5.5 (full soft)
    • Rear: 5.25 (full soft = horizontal blades)
  • Dampers:
    • Front:
      • Jounce: 6-1
      • Rebound: 6-1
    • Rear: 
      • Jounce: 6-1
      • Rebound: 6-1
  • Wings:
    • Front:
      • Position: highest setting
      • Flaps: fully closed (NOTE: this balance was about right for a 40mph corner. Any faster, and you would want to shift aero balance about 5% rearward)
    • Rear: 
      • Position: forward-most and lowest hole
      • Flaps: fully closed

Wet Weather

  • Same as dry except for ARB. Tire temperatures were good with camber from above. Did not perform higher speed maneuvers, so unsure on aero. May want to go 5% rearward of all closed to play it safe.
  • At 50% damp, wet tires performed better around skidpad. For autox and skidpad, go to wets if there are any damp spots. For endurance, not sure on when wet tires would overheat, so play it safe and only use if 50% or greater damp.
  • ARB setting below was good for running both LC0s and rain tires in wet conditions


  • Tire Pressures:
    • Cold set to 10, minimal fluctuation because minimal heating
  • ARBs:
    • Front:  5.5 (full soft)
    • Rear: 3 (about 30 degrees from vertical)



Reference from: ARB and Damper Rates Tool - Purdue Formula SAE - Confluence


Launch Control

  • Dry, hot / sunny: setting 2 or 3 if any lug is observed
  • Dry cold: setting 2
  • Damp: setting 2 launch area is mostly dry, setting 1 if launch area is damp
  • Wet: setting 1
