Getting room temperature material properties from MMPDS

NOTE: for some reason MMPDS no longer comes up in Purdue library search results. Here's the link: Metallic Materials Properties Development and Standardization (MMPDS-12) - Purdue University - West Lafayette (exlibrisgroup.com) . If Purdue ever stops providing access to MMPDS, MIL-HDBK-5 can be used. MMPDS replaced MIL-HDBK-5 when the military no longer wanted to deal with maintaining the info. MIL-HDBK-5 should still have properties for the pretty typical materials that we use and can be found online for free (MIL-HDBK-5 J METALLIC MATERIALS ELEMENTS AEROSPACE (everyspec.com)).
