Purdue Formula SAE : UT-1-04

Created by Matthew James Lamear, last modified on Sep 02, 2023

Simulation ML-061

Changes/Justifications

Description of the ChangePictureJustification
Reduced main UT inlet width.

Reducing the main inlet size increases the side channel width initially, this allows for rapid compression when the channel decreases laterally and in height.

Ideally this generates a low-pressure region strong enough to make the vortex spool properly. Very possible that the height decrease is too drastic in this channel, so that may be the next alteration to the design.
