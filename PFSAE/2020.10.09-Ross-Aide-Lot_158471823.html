Before Run

  • Look for brake leaks
  • Check for compliance (camber and toe wobble, linkage movement)
  • Set corner balance if springs or other recently changed
  • Turn car on with steering centered to provide offset to apply when postprocessing data
  • Turn car on flat surface to provide offset for shock pot data.
  • Set tire pressures
  • Record environment conditions in testing data (temp, humidity, state of car)

After Run / Before Next Stint

  • Check compliance
  • Visual check of bolts/torque stripes
  • Measure tire temps and pressures
  • Maintain desired tire pressures

Powertrain:

In Shop

  • Pressure Test Cooling System to 15PSI and check for leaks, mounted securely
  • Verify Oil System is filled and no visible leaks, oil pump belt not loose, mounted securely, fittings tight
  • Intake / throttle mounted securely
  • Fuel system mounted securely, no leaks 
  • Exhaust / muffler mounted securely
  • Drivetrain mounted correctly, differential filled, CV joints "gooked" up 
  • Drivetrain chain appropriate slack (may require tensioning)
  • Verify powertrain and non-powertrain components are safe from exhaust heat 
  • Cycle Lock

Testing Results

Failures / Issues / Notes

  • Sus:
    • Small amount of play developed in front left upper control arm. Play at camber piece
    • Front left hub had 1 stud backing out. likely is titanium stud and needs 4140 (ETD 150 replacement)
    • Should check brake discs + pads as first time with 4130, but no complaints
      • Moved brakes bias to ~53% front (by pressure). Confirmed lock up on all 4
      • Car rotated nicely on entry braking. Seems like current brake bias + damper setting is reasonable
    • need to either proof carbon fiber tierods or make nicer steel ones.
  • Chas:
    • Clutch handle could use tightening so it doesn't need as much travel Unknown User (reyno105) Unknown User (ehoerl)
      • Could flip around and add stop for now.
    • For updated design, avoid needing to pull all the way. Have a stop
  • Pow + electronics combo:
    • Engine cuts: 
      • intermittent engine cuts. Response to fast throttle input was to bog. First blamed traction control and launch control doing stuff they shouldn't. Went to fully disabled. Seems to have been small improvement
      • Then blamed bosch module. We swapped bosch module. A wire came loose on the bosch module but unclear if it was already intermittent or came loose from swapping bosch module. Seems like it probably wasn't
      • Also leveraged full return on throttle to adjust idle throttle back down to 5% (from 9%)
      • After replacing bosch module we figured out that fast throttle input was the cause - kind of.
        • With car sitting we could get bog from fast throttle input, but not all cases seen were with fast throttle input  
        • Not clear if this was just caused by new driving style - Dominic was used to smashing throttle from miata driving while past drivers have typically had only sim experience which trains for slow throttle roll in
        • Swapped spark plugs and noted that they had carbon buildup. They did not have this after the grand prix testing: 2020.09.12 Grand Prix Track
          • Black coating = carbon build up from fuel being injected but not burned
        • After swapping spark plugs and focusing on slower throttle roll in, problem was reduced but not eradicated.
      • Next steps:
        • Make list of what changed in powertrain + electronics between 2020.09.12 Grand Prix Track and this testing session Unknown User (dnocon) Unknown User (jkrawitz)
          • Especially check things affecting amount of fuel delivered: physical injector installed, injector tables in MoTeC, lambda tables
          • Should not that throttle return springs had been messed with and it now returned to full 0% not ~4%
          • oil pressure was not in use because replaced by gear position
        • Compare throttle input between dominic and previous drivers  Unknown User (dnocon)
        • Review lambda between past and current data.
        • Test behavior on dyno. Could then compare difference between dyno and car engine and software
    • Launch control failure:
      • Using new launch control that held at 8000 rpm, car would stall. Same launch starting rpm had been used here: 2020.09.26 Ross-Ade Parking Lot.
      • Could not do successful manual launch in launch setting 0. Unclear if launch control still hampering. Could not go further because ecu to computer box broke
        • Stall in response to fast throttle input made lifer harder
      • Double check updated launch control table
  • Powertrain:
    • Throttle body plastic bit for throttle return spring broke prior to testing. Was able to use hole on aluminum plate below and fix
    • In general, stacking the two springs could cause interference between them and relying on plastic for any kind of real loading should be a last resort. This should get redesign for next car.
    • Get analog oil pressure sensor Unknown User (dnocon) Williams, Carson Kenneth
  • Elec:
    • Added gear position
    • Computer to ECU / dash box broke. 
      • Fix current one Unknown User (jkrawitz)  
      • Assign new / returning member to make backup dash / ecu box Unknown User (jkrawitz)
    • Color dash needs new nuts Unknown User (jkrawitz)
    • Oil pressure needs to be added back in with new sensor that comes with calibration Unknown User (jkrawitz)
  • Aero:
    • rear wing right end plate came lose because bolt backed out. if loc-tite isn't being used here it needs to be. didnt see any on the bolt thread Unknown User (agarw162)
      • Maybe go back to being able to tighten clevis bolt on stackup including end plate
