Baseline

Coast Down Testing:

Test will be conducted by reaching velocity of 50 mph and engaging clutch and coasting down to 10 mph. Note down air temp, ground temp, local wind speed + direction

Procedures:

Clearly mark out starting places and coasting regions of both sections such that car always runs on the same place

  1. Stop prior to every launch in pre-defined launch box (4 cones) 
  2. Launch the car and accelerate to top speed of ~50 mph
  3. Engage clutch and coast down to approximately ~10 mph
    1. Press data marking button when clutch is engaged and when brake is engaged
  4. Repeat test to have at least 4 runs per setup, 2 in each direction

In between the runs of different setups, the car should sit for a while to let the tires cool
down.


Steady State Runs w/ Pit Speed Limiter Test:

Test will be conducted by reaching steady state to quantify lift coefficient

Procedure:

  1. Use a decently long straight for steady state run
  2. First start with full Aero package as baseline and do a steady state run at the top of 2nd gear
  3. Remove RW and do another steady state run
  4. Remove SW +UT and do another steady state run
  5. Remove FW and do final steady state run
