Purdue Formula SAE : How to Setup Endurance/Autocross Track Cones

Created by Skrzyszowski, Mateusz, last modified by Paradise, Hunter on May 07, 2023

This is a guide that will have plenty of examples of how cones are setup in an FSAE Michigan track. Use this guide to setup the track more efficiently and make it better for the driver to understand where they are going.

General bend gates

This example shows how to make gates that around bends.

First thing shown is the corner previous to the example gate. This is important because it will show what the driver will be looking at when they exit this corner. Being able to see and find the next corner easy is very important.

This corner is decently slow, so the driver will have plenty of time to find the next gate, especially with how far away this gate is. You will see in a further example in this section as to why this is relevant.

This is the gate itself. Standard setup for one of these gates is a down cone on the inside pointing towards the inside "apex" cone, and then three cones on the outside. If you look ahead you can see the next right turn is set up the same way.

This image is going through the next gate. Because this corner is much more bendy than the previous, there are down cones on the outside of the bend to guide the driver to the next gate.

Remember when I said that the previous corner thing was relevant? This is why. In the following corner after the right hander, there is a clump of cones in the distance past a decently tight (not really, but still tight enough to make it confusing) bend. If there were only one down cone on this left hander then it may not be so obvious about whether or not to turn all the way right or to go straight towards the clump of cones. This seems obvious when looking at it from a still image, but when you are going 40+ mph pretty much laying on the ground it gets extremely confusing.

Tight corner gates

This section will describe tight corners. This guide picks up right after the final image of the previous section.

This is a great way to set up a 90 degree turn. One cone on the inside with a down cone pointing to it, and many cones on the outside. It is clear where the track limits are and which direction the car needs to point through the corner.

This is another example of the same thing.

This is an interesting section. It is a quick left followed by a medium right turn, however the outside of both corners are marked by a 90 degree cone setup. This makes the limits clear, but still keeps the path not a box by using the inside cones appropriately.

This is a fantastic example of a section that could be confusing. If those cones straight ahead on the outside of this right hander were not there, it would be really easy for the driver to think that the next gate they need to hit is all the way down in the distance, almost treating the section like a straight. However, the cones create a barrier that makes it clear that this is a right turn, and then the down cones make it clear that it will be a left at the next gate.

Slaloms

This section is all about slaloms. Slaloms are super simple to set up, but if you set them up incorrectly they could be very confusing as to where to go. 

This is a very clean slalom entrance. The start of a slalom will always be marked by a single cone with a down cone next to it. The down cone indicates which side the slalom should be started on, so in this case the driver would start on the right side of the first cone. After that you just space out the cones evenly in a straight line.

The exit of the slalom should be very clear, as shown in this picture. It is clear that the next section is straight ahead, as there are no cones that could make it confusing as to otherwise.

Chicane

Chicanes can be very confusing for the driver if not set up correctly.

This is an image from a distance coming at a chicane. As you can see, lots of cones can make it super confusing. That is why the only cones that are down cones should be the ones straight ahead of the entrance to the chicane. They should be pointing towards the exit of the chicane. Otherwise, all other cones should be making a 90 degree bend at each corner of the chicane as shown.

Here is another great example.

This example has the down cones pointing towards the entrance. This is also acceptable, but just make sure that you are consistent throughout the whole course one which one is used. Fun fact: This was probably done because the autocross course was slightly modified and then run backwards for the endurance event. It just made it easier on the organizer, but all of the chicanes had the same format.

Passing zones / straights

The straights in the endurance event are just passing zones. This is where one car passes another slower car. 

This is a pretty simple setup. We probably shouldn't use this many cones just to save on the amount of cones we actually have for making the rest of the course clear. We also shouldn't actually make the passing lane since there will be no other cars on the track.
