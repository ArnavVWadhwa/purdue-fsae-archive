Purdue Formula SAE : Vinyl Sticker Making

Created by Unknown User (jpeerbol), last modified by Thai, Khoa Dang on Apr 10, 2023

The SAE teams have a Silhouette Cameo Plus to share. This machine can take a 15" wide piece of vinyl.


To use the cutter you will need the silhouette studio software which can be found here https://www.silhouetteamerica.com/software

This software is intended for scrap booking, but it gets the job done. There are good youtube videos you can find explaining how to use the software in more detail.


High level you need an image to trace with the software.

The top right post it note looking icon is you sheet size, make it as big as your material (15" wide, really long), choose no cutting mat.

Import the image, I like PNG's. If you want to be really cool you can layout all your images in a powerpoint to scale and export it.

Open the image in the software.

Scale the image to the size you need it to be.Â 

Find the toast looking icon on the right side, this is the trace function which will make the cutting lines, adjust the threshold to your liking. For more difficult images I had to adjust the contrast in another software to make a good cut line in the silhouette software.

I usually use the regular trace function.

This will make your red cut line, you can resize your cut lines now as well.

I like to make boxes around smaller cuts to make removing the 'negative' vinyl easier.


Once you have your cut lines, you need to send the cut to the machine.

Connect via usb, don't mess with bluetooth, it's not worth it.

Different vinyls need different cut settings, so you'll want to test before you do a full cut.

These settings are a starting point, test and adjust the as you see fit. Fill out the chart as you use different materials and play with the settings.
