What is ADAMs Car?

  • Multibody modeling environment specialized for vehicle models
  • It allows for the creation of prototype subsystems and analyze them
  • Model hierarchy is comprised of Parts, Assemblies, Subsystems, and Full Vehicle Models which are stored in Databases which we have uploaded and will maintain on GitHub for version history
  • Able to simulate real live maneuvers to analyze vehicle behavior
  • Able to study kinematic designs and tire curves (don't recommend tire curves as there is more capability with custom MATLAB scripts)
