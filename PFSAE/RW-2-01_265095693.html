Purdue Formula SAE : RW-2-01

Created by Matthew James Lamear, last modified on Jun 29, 2023

Simulation ML-012

Changes/Justifications

Description of the ChangePictureJustification
Altered 2nd element to a more DF-producing airfoil.

Testing to see the effects of the 2nd element airfoil. Also, 3rd element slot gap increased, and the chord decreased. Airfoil shape slightly altered.


Endplate changed to Chenkai's endplate design. Aim is to reduce/eliminate the near-endplate separation.

CFD Analysis

CFD PictureDescription

Loading is considerably larger (1st image is old config, 2nd image is new config). Downforce level did not change despite the loading difference. Probably an issue with the dive plane endplate losing some performance. Overall I think the old config is better due to the drag reduction. 


Flow sep is still near the endplate, would be nice to get rid of this. Maybe look into reducing AoA near the endplate using a sweep. (1st image is old config, 2nd image is new config) 

Seems like the diveplane section can get some additional camber to add some DF. Vortex structure is lower resulting in less upwash provided, need to increase camber for this reason too. (1st image is old config, 2nd image is new config) 

The endplate additional drag is certainly substantial, but this may have been the reason why the stall went away in this iteration compared to last. (1st image is old config, 2nd image is new config)
