HAAS ST20 Y



how to cam:

  1. get added to bechtel team and formula sae folder
  2. turn on cloud libraries in personal preferences
  3. bring part and stock into fusion
  4. select the right holder 
    1. set parameters to match round stock OD
    2. set stock depth (theres a ratio for this)
  5. set wcs to indicated point
  6. roughing toolpaths
    1. facing 
    2. profile roughing
    3. groove roughing
  7. finishing toolpaths
    1. profile finishing
    2. groove machining
  8. live tooling
    1. drilling
  9. axial drilling
    1. drilling
    2. boring
  10. subspindle grab
  11. part off
