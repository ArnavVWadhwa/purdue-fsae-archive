Post Testing & Debrief

Failures

  • Suspension:
    • Rear hubs 2x studs loose on right hub, 1x loose on left
    • Front brake discs came off of hub:
      • Unknown User (xia107)  + Unknown User (reyno105)  to check data
  • Chassis
    • Throttle pedal backing bent. Fixed by clamping it with some metal
      • This led to throttle overtravel = 0% throttle reading on Motec. This issue is good guess when engine cuts
    • Swivel joint failed in manual shifter
    • Clutch cable solder joint failed
    • Seat came out. Dual lock ain't it. Replaced with bolts. Going to need to add strength to bolted regions of seat
      • Unknown User (ehoerl)  to find go kart seat thing and sent to Arpit
  • Powertrain
  • Electronics:
    • Coolant sensor 0 voltage pin pulled out from solder bundle
  • Aero:
    • Rear wing got pretty loose


Other notable:

  • Tested manual shifter:
    • Easy to use with good feel: was workable for fast laps
    • Don't need clutch on downshifts
      • But should keep clutch on shifter for accel event
    • Some gear grind on shift to 2nd from neutral: either clutch didnt fully disengage or
    • Some gear grind on certain downshifts: Probably should use clutch on non-braking downshifts
  • Attempted to test traction control:
    • Pretty much killed the acceleration. Need to understand the system better
  • Tested launch control:
    • settings 2 and 5 seemed solid
  • Found cooling reach steady state of 110 C.
    • Seems likely that new sidepod design is necessary especially to have some fault tolerance
  • Check lists for car + stuff to bring was helpful
  • We have cones. The people of west lafayette dont know what a cone means
