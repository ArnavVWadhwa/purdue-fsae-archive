Purdue Formula SAE : 2. Engine Synchronization Basics

Created by Unknown User (dnocon), last modified on Dec 09, 2021

Now that you understand 1. Engine Speed Reference Basics, this page will teach you about how to setup and troubleshoot sync.

Full Engine Synchronization is required unless you want to run the engine in a wasted spark configuration. When running MoTeC in full sync mode, full synchronization will be required for the engine to start.

If you only have a crankshaft position sensor, you only know what the crank is doing. This means you are unable to distinguish intake stroke vs power stroke, and exhaust and compression stroke.

A camshaft position is required for full synchronization. 


Semi-Sequential (Wasted Spark) 4-Stroke (Otto) Engine

  1. Intake (TDC->BDC)
  2. Compression (BDC->TDC) SPARK!
  3. Expansion (TDC->BDC)
  4. Exhaust (BDC->TDC) SPARK!

Wasted spark means both the spark and fuel injector fire every single time the engine is at TDC. Fueling wise, half of the fuel is injected each time the piston reaches TDC so you still intake the correct amount of fuel when the intake valves open. There is no downsides to running wasted spark besides the chance of combustion through the intake when you are first attempting to calibrate the engine. In addition fuel film adjustments must be made to ensure that the first shot of fuel (the shot that hits the closed valves" is fully atomized by the time the intake valves open during the intake stroke.  

The main benefit of running wasted spark is that you do not need a cam position sensor. 

Wasted spark is commonly used in Harley-Davidson bikes and Mitsubishi EVOs. 


Conventional (Full Engine Synchronization) 4-Stroke (Otto) Engine

  1. Intake (TDC->BDC)
  2. Compression (BDC->TDC) SPARK!
  3. Expansion (TDC->BDC)
  4. Exhaust (BDC->TDC)

This is how almost all non-variable valve timing 4-stroke engines operate. You will need a method to trigger a cam position sensor. 

  • Not for FSAE: Variable Valve Timing (VVT) can not just be a trigger, it will need to know the cam position much more precisely. 

Common Methods to measure cam position (non-VVT)

  1. Trigger wheel on the cam
  2. Pressure sensor measuring intake pressures (you know there is a vacuum when the intake is sucking)

The team prefers a trigger wheel on the cam because it is more reliable. Temperature, pressure, altitude, and intake changes will not affect it. 

Once the engine gains synchronization it will no longer need the sensor. The sensor will only be used again if the crank position is lost.  


When should the Sync Trigger??

Unfortunately in MoTeC, the point at which the sync triggers can not be adjusted manually. This means in order to get proper synchronization, there is a window in the 4-stroke cycle in which the sensor must trigger. 

THE SYNC SHOULD TRIGGER AFTER THE PISTON STARTS TO COME DOWN DURING THE INTAKE STROKE BUT BEFORE THE TDC COMPRESSION STROKE.

AKA SOMEWHERE BETWEEN THE MIDDLE OF THE INTAKE STROKE AND THE END OF THE COMPRESSION STROKE
