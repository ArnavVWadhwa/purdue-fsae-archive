Purdue Formula SAE : Control Arm Loads Calculator

Created by Unknown User (reyno105) on Oct 20, 2019



This is a spreadsheet for going from ADAMS Car outputs to two-force member forces in tubes. Math is below. It's a 2d statics problem.


Input points for right side of suspension (front and rear upper and lower control arms). Names come directly from Adams model


Input loads directly from Adams result file. It is very important to use the values under jorrev_lca and jorrev_uca. The sign convention is based on this and this removes the vertical force of push rod on lower control arms (this is reacted through the upright, not the tubes). These terms can be searched in the Filter section of the Adams car post processing window.


Tierod loads come from jorsph_tierod_outer in the y direction (technically should take RSS, but negligible difference). You must reverse the sign of the y component to match the standard tube convention of "+" is tension and "-" is compression.




Pushrod loads come from jorsph_prod_to_lca. Take the RSS. These loads will always be compressive, so assign "-" to result.



There is a dynamic load added to the static loads (from steady state simulations). This dynamic component is estimated using strain gauge data found here: 9/14/2019 Grissom - Control Arm and Tierod Strain Gauge Loads Testing. The dynamic component is then added to increase the load magnitude. This accounts for changes in surface that may cause a wheel to momentarily load up above predicted static load, yielding a larger peak force.




Documenting which simulation was used in the "SIM" column is likely to be very helpful should there by doubt about a given load in the future.
