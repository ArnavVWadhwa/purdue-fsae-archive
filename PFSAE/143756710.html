Purdue Formula SAE : Save a Copy of a Layout File Assembly (use for revisions)

Created by Unknown User (ehoerl), last modified by Unknown User (reyno105) on Aug 28, 2020

(Superseded by Clone assembly section here: NX Tool Specific Tutorials)

Using the layout file method can make life a little more confusing when you want to create a revision. You've got a layout file that defines parts that are in your final assembly that is actually in the vehicle CAD. You want to end up with new part numbers for the layout file, the individual parts, and the assembly; however, you want to avoid breaking any of the dependencies in the old part numbers so that you actually have an old version saved off. Here's what to do:

  1. Open the layout file, all the individual parts, and the assembly
  2. Save as the layout file to its new number. This will update the links in each of the parts that are open to the new layout file. It is critical that you don't accidentally save one of those parts now, because the old part will become attached to the new layout file.
  3. Save as each of the parts to their new part numbers. This will update the parts in the layout file and in the assembly to the new numbers. Now it is critical that you don't accidentally save the assembly, because it would save the old part number assembly with the new part number parts swapped in.
  4. Save as the assembly to the new part number. This will save the new assembly with the new parts, without permanently affecting the old assembly
  5. Now you can save everything again and call it a day


Clone Assembly Tool: 

This tool is useful for cloning an assembly in order to save a revision which keeps no links to the active assembly. See: https://www.youtube.com/watch?v=C3s1hFWMJHg
