Purdue Formula SAE : BIDC Gantry | AER

Created by Augustine, Joseph Thomas on Nov 05, 2023

1) BIDC Access

  • Before CAM and manufacturing, you will need access to BIDC if you plan on using the Gantry Router (SR100) and other machines. Here is the link to get the Member agreement done, and the Manufacturing HazCom so that you are eligible to use machines in Bechtel: https://www.purdue.edu/bidc/os1
    • Step one: log into BIDC Project Management System using your Purdue Login.
    • Step two: Go to the "Resources" tab in the top left corner and scroll down to "Wood" - click on it
      • You should see 2 red items that are required for you to "reserve a machine"
      • Take the membership agreement quiz and the Manufacturing Safety HazCom quiz (it will redirect you to Brightspace)
    • Step three: Ask for AER Lead to add you to the "FSAE AER 24" Project (this will allow you to see what times we are reserving to use the Gantry)
    • Step four: When we do Layups, you will want to have the HighBay Safety Hazcom Completed as well - please do so in Brightspace; similar to the Manufacturing Safety Quiz

2) BIDC and FSAE Fusion Team

  • For CAM, you will need access to both the BIDC and FSAE Fusion Teams
    • In order to become a member of either Fusion team, you will need to sign up for a Fusion Autodesk Student account (this is free): https://www.autodesk.com/education/edu-software/overview?sorting=featured&filters=individual
    • Once you do so, take this quiz to get onto the BIDC Fusion Team: https://forms.office.com/pages/responsepage.aspx?id=Ob0wQVN8nEGx5YdY1tY_IUsoBm8nTvdGuRFnvTKraURUMEJZVUJPTFo4TkxENEFSUU84UVU1U1NSWSQlQCN0PWcu
    • Now you can talk to Augustine, Joseph Thomas to add you to the AER portion of FSAE Fusion Team

3) CAM (STEP, STOCK, CAM)

  • Ask AER System Owners and Augustine, Joseph Thomas to give you a STEP file for CAM
  • In Fusion, locate the correct folder under →Formula SAE → Aerodynamics to save your STEP file, Stock, and CAM
  • Steps for creating the STOCK
    • 1) Importing the STEP file (as mentioned above)
    • 2) using the Measure Tool (in Design, tools, measure) measure the Length and Width of the piece (this is a baseline for stock)
    • 3) Create a new part titled AER_PartXXXX_Stock with the correct Length and Width you just measured
    • 4) Your Height will be measured from the bottom of the part to what is the upper-most surface/corner, edge, etc.
      • Round up the nearest Inch or Half-Inch of the measured Height
    • 5) Now you should be able to EXTRUDE your Rectangle of Length and Width to the Height
    • 6) Save your Stock
  • Steps for setting up File for CAM (NOT CAM)
    • 1) Create a new part titled AER_PartXXXX_CAM
    • 2) Navigate out main workspace (home button next to arrows near top left of screen)
        • Scroll down to select "Workholdings, Stock and Machines" - click into it
        • Sroll down to select "Tables" - click into it
        • Drag and drop SR100 into the part
        • BREAK THE LINK ATTACHED TO THE SR100 / TABLE
        • (optional) delete the "Sacrificial Plate" if you do not plan on using one
    • 3) Navigate back to Formula SAE → Aerodynamics → #DesignatedFolder# to find AER_PartXXXX and its corresponding Stock file
      • Drag and drop them both into the current part (CAM)
      • (optional) Break the links to both the Part and the Stock
    • 4) Keyboard shortcut for jointing objects together (J)
      • joint the corner of the part to any 'bottom left' corner of the SR100 Table
      • joint the corresponding corner of the stock to the same corner of the SR100 Table
      • Select any component/part - drag it around to see if it moves, it does, "Ground" the SR100 Table
    • 5) Change the mode from Design to Manufacturing
      • You will now begin CAM Setup
      • (If you have not Enabled Cloud Libraries in Preferences, do so – Top right corner, under profile Icon)
        • (In the Manufacture section of tree, check "Enable Cloud Libraries", and "Show Operation Machine Time") - press apply and OK
      • Select "New Setup"
      • Under "My machines" Cloud → HAAS SR100 – select
      • a) MANUAL PROBING Work Coordinate System (WCS) – orientation ("Select coordinate system"); Stock Point (top corner of Stock closest to WCS Coordinate Arrow)
      • b) AUTOMATIC PROBING WCS – orientation ("Model Orientation"); Stock Point (top corner of SR100 Table - WCS Arrow)
      • Model – select the Part you want (AER_PartXXXX)
      • Fixture – Turn On
        • Select the Table in  SR100 (Tree area under Manufacture - Models, SR100, Table)
      • Continue to Stock selection
        • select Stock
      • Continue to Post Process
        • Change "Program Name/Number ~default 1001~ " to "XXXX" (Part number you have)
        • MANUAL PROBING - Machine WCS -change from 0 to 1
        • AUTOMATIC PROBING - Maching WCS -change from 0 to 2
      • Setup Complete, select OK
    • 6) GANTRY AER purposes
      • select 3D - parallel - *roughing pass
        • select Tool - (Tree on left) "BIDC IS030 (SR100)" - 1/2" (Carbide 1/2" D Flat Compression End Mill) - select
        • *Coolant Should be disabled
        • Feed & Speed - Foam (unless otherwise noted)
        • "Geometry" tab - Machining Boundary - "selection"
          • select the bottom, outermost chain of the Model
        • "Passes" tab - Passes - change "Stepover" to 0.25
        • "Stock to Leave" - check - number should be 0.02
        • First operation complete - select OK
      • select 3D - parallel - *finishing pass
        • select Tool - (Tree on left) "BIDC IS030 (SR100)" - 3/8" (Carbide 3/8" D Ballnose Compression End Mill) - select
        • *Coolant Should be disabled
        • Feed & Speed - Foam (unless otherwise noted)
        • "Geometry" tab - Machining Boundary - "selection"
          • select the bottom, outermost chain of the Model
          • Disselect "Contact Only"
        • "Passes" tab - Passes - change "Stepover" to 0.05 (unless noted otherwise)
        • "Stock to Leave" TURNED OFF
        • Second operation complete - select OK
      • Simulate both operations
        • "Stock"
          • decrease Accuracy
          • Turn ON "Stop on Collision"
        • Note collisions on toolpaths, talk to Augustine, Joseph Thomas , Fava, Nicola Bernardo , John Cheng Yu Chang 
      • Go to Tool Libraries in the "Manage" Tab to change Tool Number
        • *Default right now is 1/2" at 1 and 3/8" at 2
        • Renumber 1/2" to 3 and 3/8" to 4
      • Assuming everything above is done correctly, CAM is complete
        • Reach out to John Cheng Yu Chang for CAM approval
        • notify Augustine, Joseph Thomas or part distributor about completion
          • discuss Stock sizes and operating day
    • 7) **MetalShop AER purposes - TBD
      • talk with Augustine, Joseph Thomas Crooks, Troy A

4) Operation at BIDC

  • Be sure to complete 1) before requesting to machine at BIDC
  • Make sure CAM is approved before operating
  • Make sure John Cheng Yu Chang Crooks, Troy A and Augustine, Joseph Thomas/ Fava, Nicola Bernardo (Trusted Users) are there for supervision
    • Create a reservation for machines needed at BIDC
  • Go down to Stockroom or FSAE shelf in HighBay to retrieve stock
    • Cut down stock to size (Panel Saw, Bandsaw)
    • *Note that level 3 machines need a STAFF of BIDC to operate, not just John or Troy; communicate/coordinate machine needs/usage properly
  • Read the Gantry SR100 Standard Operating Procedure:
    • Gantry SOP.pdf
    • Ask Questions as needed
  • Retrieve correct tools/endmills/drills from ENDMILLS drawer in Woodshop (1/2" named: LMT ONSRUD, 3/8" named: LMT ONSRUD **we do not have this, use Kennametal's**)
  • *Loading Tools - Follow Length Below Holder measurement in for 1/2" EndMill (default 1.8"), 3/8" BallNose (default 1.8")
    • Use Torque Wrench - set to 80lbs/ft to tighten tool
  • resume following SOP
  • Right-click on Setup - Post Process
    • check operation tree to confirm number of operations 
    • MANUAL PROBING - deselect "Tool Probing" & "WCS Validation" - post
    • AUTOMATIC PROBING - post
  • upload finished CAM file to BIDC Net Share: https://upload.bidc225lnx1.ecn.purdue.edu/

5) Clean Up/Part Storage

  • Be sure to clean up the workspace, BIDC will not accept future reservations if you are found responsible for a messy workstation/machine after usage
  • discuss with present team members where to store the part and leftover, usable stock
