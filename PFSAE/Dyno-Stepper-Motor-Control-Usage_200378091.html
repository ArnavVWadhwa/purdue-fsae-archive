Purdue Formula SAE : Dyno Stepper Motor Control Usage

Created by Unknown User (jpeerbol), last modified on Dec 10, 2021
  1. Make sure dyno is powered off and reset switch is in the off position and stepper motor is unplugged
  2. Move control valve to 0% and then plug in the stepper motor
  3. Turn on main power switch, stepper motor should move to 100%
  4. Then in M1 Tune, Tuning > All Calibrate > Dyno, go to percent load under Water Valve Stepper PID and set it to 100% and then turn on the reset switch
  5. The stepper motor load control is now ready to use
