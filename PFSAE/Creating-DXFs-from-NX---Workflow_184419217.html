Purdue Formula SAE : Creating DXFs from NX - Workflow

Created by Unknown User (xia107), last modified by Unknown User (hammerly) on Aug 10, 2021

DXF Creation

1. Start with the .part in NX, you can have as large of an assembly as you want, but a single waterjet part must be it's own independent body.

2. Enter the sheet metal application. Realistically, all tabs should be designed in the sheet metal application.


3. Select convert to sheet metal, and select the base face of the tab to convert. This gets more complicated for bent brackets and is only applicable if the 3D geometry has an uneven wall thickness. This article goes deeper into how to convert to sheet metal.


4. Click Flat Pattern and select the appropriate base face. NX then creates a new feature called a flat pattern. I think this is useful in drawings.

6. Click Export Flat Pattern. If you have multiple tabs in a part file, you'll have to create a flat pattern for each tab and export them individually.


DXF Preparation and Rearranging

(This is the best way I know of for doing this. Again, if someone has a better way, please update cause this is not ideal.)

1. Fire up AutoCAD, available on ME lab desktops (via RDC if needed), open the .dxf file you just created and hit the green checkmark on import diagnostics if it offers. It's gonna look like this:

2. Rearrange and organize dxfs as needed

This is also the time to think about designing in tabs, resizing holes for tolerance, etc.

You'll need to be able to use the basic commands in AutoCAD (move, copy, mirror, rotate, etc.). This guide won't explain those, so just google it.

Basically you can use AutoCAD to copy all your parts (of the same thickness and material) into the same DXF file for waterjetting, rearrange to be efficient with stock usage, etc.

The following only applies to FlowCut/FlowPath. That would likely be waterjetting at BIDC, WJCI uses OMAX.

3. When you have all the parts rearranged and organized, you'll want to run these 3 commands to avoid issues in FLOW later:

  • Purge (Removes unused items, such as block definitions and layers)
  • Overkill (Removes of overlapping, repeat, lines)
  • Explode (Separates all lines/arcs into individual segments)
