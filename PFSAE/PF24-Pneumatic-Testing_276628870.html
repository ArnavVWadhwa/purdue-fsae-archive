Test List

  1. Hydraulic Cylinder Validation -- Goal: validating cylinder performance
    • https://www.terex.com/docs/librariesprovider3/default-document-library/testing-and-validation-hydraulic-cylinder.pdf?sfvrsn=7d578733_0 
  2. Paddle Control w/Actuator – Goal: learn how the paddle controls the actuator/air inside
  3. Load testing with different pressures – Goal: validate that safety measures with lines running below normal pressure can still deliver minimum required force to shift
  4. Max Shifting – Goal: check that when running 1-2 endurances there is still air in the air tank for use (e.g. if a leak was to occur)
  5. Timing - Goal: time to actuate
    1. slow-mo actuation with timestamps how long to be released
    2. control over timing of being released (playing with prop valve and pressure)
