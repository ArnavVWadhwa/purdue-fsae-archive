Changes/Justifications

Description of the Change

Picture

Justification

Main element reprofile, third element reprofile, second element angle of attack adjustment

Copy and paste from RW-06-01:

Second element can do with better flow alignment to avoid the reflex. That means more angle of attack since I don't want anymore slot overlap than I do already.As far as loading goes, main element can do with more loading.

It's hard to see but I think there are two suction peaks on the main element which I want to avoid. So that means higher camber near the center.

Third element has too much suction, which at this angle of attack is mostly drag. The solution is probably smaller slot gap to reduce energy delivery to the suction side of the third element. The second element angle of attack is going to be increased to resolve the reflex issue, so there will be so energy loss from that as well.
