Purdue Formula SAE : Dyno Planning

Created by Unknown User (jpeerbol), last modified on Jul 26, 2020

Things that need to be taken care of before running

Need to either size smaller pump or better flow control

  • needs better resolution
  • Find land and sea charts
  • figure out calculations and pump sizing
  • electronic controlled pump
  • Think current pump can provide 40 gpm


Minimum Needs:
calibrated torque output, sae calibration, need to be able to accurately compared different components at different times


Eventually want:

Make getting the dyno setup as easy as possible

  • better casters on the cart
  • extension cord
  • Hose crank
  • better rollers for tub
  • computer table/chairs 

Automated load control

  • rpm hold
  • motec or stand alone unit

Better throttle handle/ holder (something tool-less, better than just a screw)

Automated timing adjustments (probably really hard, but could save a ton of time)

  • auto timing changes
  • log torque output and auto adjust load to fix rpm
  • https://www.motec.com.au/forum/viewtopic.php?f=39&t=4347

3d torque curve based on throttle position to use for lap sim

easy way to measure fuel usage

virtual endurance runs (requires electronic throttle)

 


Would like an easy to move engine frame

  • need to backwards compatible with vibration isolators and different exhausts
  • at least two of them
  • could be used as permanent engine storage
