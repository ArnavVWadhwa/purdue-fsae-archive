i2 Standard

  • Download is directly found on MoTeC's website for free
  • Use the most recent version of i2 Standard
  • Do NOT Download i2 Pro as it will not work with the provided data.

NX 1946

  • There are multiple ways to access NX, with the best option being to access a lab computer in person
