Problem:

Design a reliable dash panel for the driver to be able to easily access relevant data and and have access to necessary controls. 

Requirements:

VehicleDash
Maximize Competition Points
  • Start Button
  • Kill Switch/Shutdown Button
  • Inexpensive
  • All controls must be contained within the cockpit
Driver Safety
  • Kill Switch/Shutdown Button
  • All data/controls are easily visible
    • Accessed easily
System Reliability
  • Start and kill switches work without fail
  • Displayed data is current and accurate
  • Dash components are securely mounted
System Serviceability
  • All switches/controls must be easily accessible from within the cockpit
Driver Confidence
  • MoTeC Display Logger?
  • Data Display
  • Any other data that driver needs
