Notes When Developing Model:

  • There are some weird things with adams to look out for. One is when saving a model make sure to go to the highest level of the assemvbly and go file save or saveas to ensure all updates are made. It seems like ctrl+s does not actually save changes.
  • Look out for adding or modifying tire properties/springs/damper files. You might need to go into the subassembly file → open in notepad → edit by adding the file path to correctly load and let the model use the file.
