i2 Pro (Data)

Very useful thing to verify first, make sure the ECU has the correct name for the vehicle. Having this correct makes it much easier to sort data later.

Make a new folder for each testing session, 2022.03.03 - Grand Prix 1

Put all of that day's data in that folder. I know you can use data bases to search through later but the folders are a great redundant feature.

Short Comment

Year.Month.Day - Rough description of event

Example: 2022.03.03 - JP & Dom endurance 1

Long Comment

A description of the data. Include a reference to the associated confluence page if there is one.

Driver

Full Driver Name (First and Last). If it is an endurance put the first driver's first name first, second driver second. (data can and will be split after the fact, the driver's full name will be on that specific data file)

If Dyno tuning, put the name of the person operating the dyno/tuning.

If warming up the car or "bench testing" put diag

Venue

Location that data took place. (i.e. Grissom, ME Loading Dock, Ross Ade, Grand Prix)

Event

Event that is taking place. (i.e. Endurance, AutoX, Comp skidpad, Comp Accel, 50ft skidpad, Dyno Testing)

      • Warmup
      • Preflight
      • Shakedown
      • Testing
      • Testing Endurance
      • Testing AutoX
      • Testing Comp Skidpad
      • Testing Comp Accel
      • Testing 50ft Skidpad
      • Comp AutoX
      • Comp Endurance
      • Etc.

Session

The number corresponding to the data pull for that day. If this is the 3rd time the data is being pulled, this should be 3.
