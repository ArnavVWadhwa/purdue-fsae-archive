Purdue Formula SAE : NX units conversion

Created by Unknown User (jpeerbol), last modified by Unknown User (agarw162) on Sep 29, 2022

For some reason NX makes it extremely difficult to convert part units.

This was extremely finicky but I think I finally got it to work somehow.

Need to use NX Command Prompt which is a separate application

  1. Go to NX command prompt
  2. Type " ug_convert_part -in " in the command line (no quotes, put a space between the part and -in) and then drag in the part you want to convert into the window, the file path will populate. Then press enter.
  3. It should show some command prompt stuff and say successful.
  4. Check your displayed part properties to see if your units actually changed.


Directions taken from here: https://www.prolim.com/how-to-convert-part-units-in-nx/
