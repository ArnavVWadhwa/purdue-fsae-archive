The introductory project for chassis will be to design a front crash structure that meets or exceeds the pre-defined project goals/design targets (see below). The goal of this project is to continue to progress with CAD skills learned in the initial CAD training videos by incorporating more independent and more complex design problems. In addition, this project should provide you with opportunities to learn simulation techniques in ANSYS (the FEA software used by the team) as well as get familiar with the documentation and hand calculation tools provided. 

This project will be done by all new members to the chassis team alongside any current members who wish to participate (this is not required but highly encouraged).

As a general rule for this intro project, we don't expect you to have any knowledge of car design, CAD software (other than concepts covered in the introduction), or ANSYS. The goal of this project is to let you get comfortable with the software and design process used by the team through a challenging design problem. The purpose of this project is NOT to be frustrating or extremely difficult or time-consuming, because of this make sure you are asking as many questions as you need to (I would rather you ask than progress in the wrong direction and get frustrated and stop working on the project). 

In addition, resources, including the drawings, Formula SAE rules, and tutorials for processes in this project are included at the bottom of this page. Please make use of these.

The project is expected to take around 10 hours to complete. Plan accordingly.

Project Goals / Design Targets: 

  • Simulate the bulkhead, bulkhead tabs, supports, and anti-intrusion plate
  • Get deformation results which are close to those seen in the physical testing of the bulkhead

Deliverables: 

  • CAD and simulation model of the bulkhead, bulkhead supports, and anti-intrusion plate
  • Simulation of the full assembly in ANSYS (Just take some screenshots in ANSYS once you've finished)
  • Plot of Deflection as a function of mesh size
  • Answers to the questions at the bottom

Steps:

Model the crash structure in NX

Build the assembly hierarchy shown below.

Engineering diagrams are provided down below in the resources section.

This will help us evaluate your assembly, and when working on a large project it is important to subdivide systems logically so other people working working on the same systems know where to find everything. We also want you to have some experience with assemblies in assemblies. The engineering diagrams are provided in the resources section.

Note: When extruding the impact attenuator and supports, extrude as a sheet

Wavelink geometry (Where separate parts meet)

This allows parts to interact with one another, meaning you can copy geometry from one part onto the other. This way you can make the holes on the tubes, and then project them directly onto the anti intrusion plate. This means that if you were to change the dimensions of the bulkhead, the changes would carry over onto the anti intrusion plate, and your holes would still be aligned. Good practice is to link one body at a time. 

A tutorial for this is in "NX Tool Specific Tutorials", which can be found in the resources section at the bottom of this page.

Midsurface by Face Pairs

This turns the object it is applied to into a set of faces instead of a solid. This allows Workbench to make sense of the part. Use 0.5 thickness ratio, meaning the face that is created is half way between the outer edges of the object you are applying this to. Do this for every part, but NOT the supports and impact attenuator.

Hint: If the apply button is grayed out, click "Automatically create face pairs".





Divide Face (Supports and Impact attenuator)



Divide the face of the actual part you are testing (Bulkhead/Anti intrusion plate) by the part delivering a force (Impact attenuator/tube connections). This leaves an area which can later be selected in ANSYS when selecting supports and the location of the applied force. Make sure you are selecting the midsurfaced faces, NOT the body because you are going to export surfaces, not bodies, so ANSYS won't recognize any command that has only been applied to the body.


After this step you can export your assembly as a STEP file and import it in Workbench. When doing this, make sure to export surface bodies specifically. This can be found in the menus in the export screen. If we don't do this, we aren't sending anything of value to Workbench since the whole part is a set of surfaces.

Analyze the structure in ANSYS Workbench / Mechanical

In Workbench drag a Static Structural analysis system from the bar on the left to an open space in the Project Schematic.

 

Import the step file geometry (right-click Geometry>Import>Browse)

Open DesignModeler (right-click Geometry>Edit in DesignModeler)


Suppress Excess Geometry

At this point, if you get an error saying you have 0 bodies and 0 faces, right click the line in the tree outline where the error is. The generate button in the menu will load in the imported surfaces.

In the parts tree you'll see all the faces you created by midsurfacing in NX. Select the ones which correspond to the supports, right click, and choose suppress. This is because they overlap with other parts and don't represent a physical component, but a face through which forces act on the part. If you haven't discovered how great ctrl clicking is by this point, try it. Also, just for laughs, try pressing ctrl while panning around the part. 

Group components into parts

You can see this in the previous photo. When you have many components selected, like all the bulkhead parts, right click and choose "Form New Part". This'll let you hide an entire part at a time when convenient later on in the model editor.

Open Mechanical

This is back in the Workbench main page. Right click model and choose model editor.

Apply thicknesses

This converts your parts back from a set of faces, into a solid part that can be generated into a mesh. Refer to the engineering diagrams for thickness values.


Generate No Separation Contacts between the bulkhead and AIP 

This is where the bulkhead touches the anti intrusion plate. It lets workbench know that the parts are touching each other so that that can be taken into account in the analysis. Remember to also select the Filleted corners of the tubes here.


Connect the welded inserts to the anti intrusion plate by a fixed joint

To do this create named selections for both the welded inserts and the AI plate. Then you can create a fixed joint between the two named selections instead of individually going in and selecting welded inserts one at a time, hiding the AI plate and then showing it again every time. Find the named selection tool in the search bar at the top. This tells the program that the parts are fixed to each other. If you want to understand why it is a hinge, ask Christian.


Add fixed supports

Now you'll go in and select the supports. This simulates the connections from the bulkhead to the rest of the chassis. The circle is the end of a tube, which will resist against the bulkhead pushing backwards in the event of a crash. You can select all 4 supports at once.


Apply the force.

This is done almost exactly the same way as the supports, just where the impact attenuator is instead of the connections to the Chassis. Select the direction vector based on common sense. Apply the steady state force from the F-D plot.


Tell the simulation what to measure

We care about total deformation and Equivalent (von-Mises) Stress. When you simulate the crash you'll see a distribution of these values along the part, similar to mesh quality. Also parametrize the maximum deflection. To do this, go into results in the total deformation menu and click the box in front of Maximum.


Parametrize the mesh size

If you were only doing one simulation you would care a lot about mesh quality. You are measuring deflection in the middle of the anti intrusion plate so it is important that the mesh quality is high on that area. However, you will be parametrizing the mesh size, meaning you'll never even look at the mesh.

To parametrize the mesh size click on the highlighted P in the design editor for mesh size, you find a new menu has popped up back in the home page in Workbench.

Go into it and add in new rows for the mesh size, decreasing exponentially. You can go back and edit this later after running a pass of simulations. This will also add rows for the max deflection value which we parametrized in the last step.

Running the simulation

Below is an explanation for how to run the ANSYS simulation. You are not able to do this due to lisencing issues (You are only able to simulate parts with extremely few mesh elements, making the simulation useless). When we get together to assess your project, someone with the correct lisencing will run the simulation you have set up. The only part you are not able to do is click run and get results. (Sad, I know)


To run the simulation, right click the table with the parametrized values and select "Update Selected Design Points". This will run the simulation and fill in the table for max deformation. During this proces DO NOT CHANGE ANYTHING. You don't want to cause issues with the simulation. Either do nothing and talk to your friend in the computer lab, or go into another tab and do schoolwork.

Once the simulation has run it will say ready in the bottom left corner. At this point you can right click your table and export it into Excel as a CSV file. Plot total deformation as a function of mesh size.

After this, Duplicate your Analysis System, go back in and suppress the diagonal tube and remove the contact between it and the AI plate. Set the mesh size and run the simulation again, this time in the deign editor since you now know the optimal mesh size from your last sim. Observe the deflection without the diagonal tube. According to point 8.2.4 of the rulebook (linked in resources)"All Non Crushable Items inside the Primary Structure must have a minimum 25 mm clearance to the rear face of the Anti Intrusion Plate". This means, deflection must be less than an inch. This is why we decided to add the diagonal tube.

Questions

  1. At what mesh size does the mesh size cease to be a relevant variable?
  2. What is the deflection with and without the diagonal tube?
  3. Where is the highest stress on the part?
  4. What's another way that the Bulkhead and AI plate could be changed, as opposed to the diagonal tube, to keep deflection under 1 inch while minimizing weight? (This is open ended with no right or wrong answer, I just want you thinking in terms of the design decisions Chassis makes)

Compile your answers and the other deliverables on a pdf and send them to Aaltonen, Emil Johan .

Anti Intrusion Plate, Aluminum 6061. (Note, the other parts are steel and this is Aluminium. This is because the volume of this part is massive in comparison to the other parts and Aluminium is light)

The rectangle in the center represents the impact attenuator. It does not matter how far out you extrude this rectangle, as it is only relevant as a surface. For the purpose of this analysis, the input force works through the impact attenuator, and the effects of this force on the rest of the structure are analyzed.
