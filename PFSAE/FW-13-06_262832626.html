Purdue Formula SAE : FW-13-06

Created by Zhang, Chenkai, last modified on May 31, 2023

Simulation: CZ-011

Changes/Justifications

Description of the Change

Picture

Justification

More strake changes

Leading edge stayed the say. Extended height back to FW-13-04's height. Removed the curl because I don't understand what's going on. Increased inwash angle so that the strake is now geometrically inwashing at the trailing edge. This is to power down the inner vortex even more. This is a balancing act between powering the vortex down and inwashing the outer vortex wake.

CFD Results Analysis

CFD Picture

Description

Inner vortex formation is slowed, leading to higher front wing downforce.

Size of downstream wake did not change much. If anything, this is ever so slightly more inwashy, which caused by the inwashing nature of the strake.
