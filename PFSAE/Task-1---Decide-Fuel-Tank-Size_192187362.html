Purdue Formula SAE : Task 1 - Decide Fuel Tank Size

Created by Skrzyszowski, Mateusz, last modified on Oct 11, 2021

Why is fuel tank size so important?

Simply put, without fuel the engine can't run. The primary reason we need to size our fuel tank using data is so that we have enough fuel to finish the endurance event of competition. This is the most important event in competition as it is worth far more points than any of the other events. We also don't want to size our fuel tank to be too big. If the fuel tank is much larger than required, we are adding unnecessary weight to our car. When designing the car, we need to make everything as light as possible. Weight adds up with each component on the car, so each system should be designed with weight management in mind. This will not be a design requirement in this onboarding, but should be something to keep in mind while designing your system.

Using i2 to find the fuel used

Below is the data file containing data from the 2021 FSAE Michigan endurance event. Download this and put it somewhere that you can access.

MichiganEndurance2021.ld


If you are unfamiliar with i2, it is the software that we use to access and look at data that was pulled off the ECU after the car was running. First thing we need to do is set up and navigate i2. To start out, open i2 and click where it says circuit. This is just one of the default workspaces that i2 comes with. Don't worry about this, we have our own workspace that we use that I will link to later. Just use this one for now. You are now in the main i2 layout. This is where you will be able to look at and analyze all the data from running the car. In order to look at data in this layout, you need to add a channel using the channel tab on the left size of the screen. Knowing this, you can figure out what channel to find that will tell you how much fuel was consumed during the endurance run.

Finalizing Tank Size

The amount of fuel used during an endurance can change from run to run. There are a number of variables that affect this, including track design and engine operating conditions. This is why we need to apply a safety factor to our fuel tank size before we start our design. Once you have chosen and applied a safety factor to your size, prepare a small paragraph that explains what you did to select your tank size and why you did it.
