Purdue Formula SAE : 1. Engine Speed Reference Basics

Created by Unknown User (dnocon), last modified on Dec 03, 2021

Engine Speed Reference is one of the first sensors you need to figure out on a new engine. Engine Speed tells you more than just RPM of the crankshaft. It also allows you to know the location of the piston, TDC, BDC, and everywhere in between. 

For every 2 rotations of the crankshaft on a 4-stroke engine, the camshaft will rotate once. You can not get synchronization from only the crank sensor. See 2. Engine Synchronization Basics for an overview on sync. 


Magnetic Pickup vs. Hall Effect

Before we get into M1. You will need to understand the difference between types of trigger wheel sensors. 

A Hall Effect sensor is superior to a magnetic pickup for our purposes because the output waveform is easy to use. 

A magnetic pickup (or otherwise known as a M Hall) will need a moving magnet to trigger a signal.

A hall effect will only require a ferrous material (like steel) to trigger the sensor. 


Depending on your build these are the parameters you will need to adjust to allow MoTeC to read the sensor corrrectly: FOLLOW THE HINTS GIVEN BY MOTEC IN M1 TO TUNE THE PIN VALUES!

It will require a lot of cranking and trial and error to tune these values correctly. To speed up this process and to view the actual waveform of the sensor see this page: 3. Capturing Inputs in MoTeC


Overview

These parameters are things you will likely need to adjust.

Much of the rest of this page is explaining what MoTeC already tells you in M1.

Mode

When you open M1 Tune (Engine → Speed → Reference → Mode) there are several options for different trigger wheel setups. Below is a screenshot of the Enumerations relevant to FSAE and the engines we would be using.  You will likely need to open your engine up to see the triggering mechanism. This will likely be located on the flywheel, a gear on the timing chain, or on the AC Magneto. Do your research diligently and select the appropriate mode.



Tooth Count

On the crank trigger wheel count the number of teeth. If there are missing teeth, pretend that they exist and include them in your counting. 

  • i.e. If you have a Trigger that has 11 physical teeth showing and one location that a 12th tooth would go to make the wheel symmetrical, you should input '12' into "Tooth Count" in M1.
    • This type of trigger wheel is known as a 12 tooth 1 missing. 


Offset

Attached is a screenshot of Engine Speed Reference Offset. Follow these directions to figure out what number to input. You will need to take crude measurements inside the engine of the trigger wheels location in comparison to TDC. There will likley be a marking you need to line up to determine TDC. Or you can put a dowel inside the sparkplug hole and rotate the crank until that dowel is at TDC. Don't worry about getting this perfectly correct. We will fine tune this offset number later so timing will be exactly correct. See 5. Manually Verifying Ref and Sync


Blank Ratio

Attached is a screenshot of Engine Speed Reference Blank Ratio. You will need to view the waveform and set this number. This number is not crucial to the performance of the engine and can be set to 20%. 


Wide Pitch Threshold

Attached is a screenshot of Engine Speed Reference Wide Pitch Threshold. Set to 150% as suggested by MoTeC.


Narrow Pitch Threshold

Attached is a screenshot of Narrow Pitch Threshold. The team has followed the 60% recommendation by MoTeC in the past.
