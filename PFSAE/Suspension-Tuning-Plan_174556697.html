Small skid = 15ft ~17 mph

Med skid = 25ft (comp skidpad)  ~24 mph

Large skid = 55ft ~35-40 mph (aero package)

Small slalom = 25ft (autocross min)

Med slalom = 37.5ft

Large slalom = 50ft (endurance max)

Step steer maneuver = steering input from medium skidpad applied 5mph faster than medium skidpad

Medium speed braking / corner entry = eyeball GP setup

Increasing Radius corner =  eyeball GP setup


Setup

  • Remove aero package
  • Set front and rear ARB to middle settings (by rate)
  • Set dampers to baseline (~0.6 ratio)
  • Set toe to:
    • Front: 0.5 out
    • Rear: 0.5 in

Steady State

  • Run large Skidpad:
    • Make large change to prove effectiveness
  • Run tight Skidpad:
    • Find best arb setting and record
  • Run regular Skidpad
    • Find best arb setting and record
  • Add aero package - fully closed
  • Run regular skidpad:
    • Find best arb setting and record (mostly making sure aero isnt changing behavior at this speed much)
  • Run high speed skidpad:
    • Move aero aft if stability is problem


Transient

  • Medium slalom
    • Make large damper adjustments to attempt to feel effect
  • Small slalom:
    • Find best settings and record
  • Large slalom:
    • closer to steady state, just see how the car feels
  • Try step steer maneuver matching design case:
    • Go to min and max damper settings and run 3 each
  • Medium speed braking / corner entry:
    • Get feel for rotation and tune w/ brake bias or rear damper low speed rebound

Differential

  • Increasing Radius corner
    • Test with min and max settings to feel change. Pick best or go to middle setting

Bonus

  • Try crazy rear toe setting in skidpad
  • Try 0 toe in slalom - see if we get the large effect predicted
