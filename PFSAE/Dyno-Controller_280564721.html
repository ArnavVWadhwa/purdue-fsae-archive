Full Dyno Controller

VI of current dyno controller (yet to be tested).

Two modes of operation are controlled by a case block, explanation is below.Â 

Currently, the PID controller directly reads engine RPM and adjusts the frequency of the input PWM signal and direction to control stepper speed, and therefore controls the flow to the brake and load to the engine. A better approach may be to determine the ramp response of the stepper-valve system from fully closed to fully open, and create a relationship between stepper position and flow rate. Then, implement a PID controller that uses the flow rate as the process variable to adjust engine RPM.


Automatic Load Control Block

Uses Automatic Tuning PID block.


Automatic Pull Block

Uses encoder position to fully open stepper motor, and then smoothly close stepper motor.
