Purdue Formula SAE : ADAMS vs. MoTeC Data Plotter

Created by MacNab, Zachary Robert on Nov 10, 2022

WIP - This page currently serves as an idea board / temporary place to store my thoughts

I believe this tool will be very useful for correlating our ADAMS models to the car. It is not really valid for transient maneuvers where driver error is a large concern, but I think it could be valid for SS. 


Technicalities

  • Coding Language: Python
    • Matplotlib used for plotting
    • Numpy for parsing
  • IDE: Spyder, Anaconda distribution


Ideas

  • I would like a UI where the user is able to select their data channels they want to look at
  • UI also houses file uploader 
    • Maybe I want this infinite, so you can even overlay multiple ADAMS datasets or multiple MoTecC datasets
  • Maybe a % different or stdev between the two uploaded channels
  • Up to the user for what values of the tire model to change for a better correlation
