Purdue Formula SAE : Manual Lap Generation | i2

Created by Unknown User (jpeerbol), last modified by Paradise, Hunter on Jun 01, 2023

This is how you manually create laps within i2 Pro using the transponder data from competition.

0. Make sure you are logging good data, you need to be logging GPS and have GPS time in the data

  1. Find the live results online, the current link is results.fsaeonline.com
  2. Find the event and time it started timing
    2. The time is in GMT
    3. The lap time starts with the event before, so the start of driver 2's stint in the end of driver change lap

  3. Open your data in i2 pro and find the time in the data.

    2. you want a worksheet with GPS time of day across the top, zoom in on the data to find the resolution you need
  4. Go Tools > Lap Editor > Beacon (across the top) > Insert (it will auto populate with where your cursor is in the data), press OK
  5. Type up your lap times into a notepad document, MM:SS:sss
  6. Go Tools > Lap Editor > Laps (across the top) > click the lap you want to build off of (probably the last one) > Insert, then type in your lap times, then OK
    1. helpful hint if the car turns off (driver change) it will probably mess with the lap times so you will have to restart the lap building procedure for that section
  7. Click laps you can't trust (i.e. driver change or the first lap of skidpads), click edit, then uncheck trust for lap time comparison.

For something like an endurance you probably will also want the option to compare two different driver stints, do this after you generate the lap times in the full file. To split data, seeÂ Splitting Data | i2
