Purdue Formula SAE : Evaluate Simulation Results

Created by Kurisko, Keith Gregory, last modified on Nov 14, 2023

How to look at results from a simulation that you ran in ADAMs Car

  1. Load Postprocessing Window
  2. If you ran a simulation in the environment it should automatically populate the results file
  3. If the result file is not loaded or was ran at another time you can select import a file.
  4. Select from the bottom options to configure what you want to plot and view.
