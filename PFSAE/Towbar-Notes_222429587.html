Purdue Formula SAE : Towbar Notes

Created by Panchal, Aarnav Nilesh on May 24, 2022

Given that we integrated the towbar design into the uprights, essentially creating another load case for the uprights, we noticed a couple of things:

  • The Towbar was easy to create a pushbar/quickjack around
  • The Towbar CAD was not satisfactory and contained circular references
  • The Towbar cross bracing needs further analysis as we have noticed it broke when it was 1/16 and not when it was 1/8, maybe we can save weight on it with different geometries
  • The side braces held up quite well however we can also further optimize these by looking into lug geometry on the uprights
  • We also were able to mount the undertray's rear points and secondarily constrain the chain guard to the towbar, this proved to be useful as we passed rules and avoided cantilevering the guard (it broke beforehand though haha)


Design Process:

  • Requirements
    • Needs to follow rules while also supporting vehicle on quickjack
  • CAD Process
    1. Process should begin with towbar location, this is to avoid the circular reference and simplify geometry modifications
      1. The towbar location is going to be constrained by the location of the differential, it is a point in space centered on the car axis
      2. You should create a datum for the towbar part, this will serve as the governing datum for the modifiable geometry
    2. Next step would be to define the location of the Upright Lugs, create those and make sure you are able to move them in the upright parts
      1. Ensure that you have a generally okay location (ie don't put one on the middle of the bottom walls, you're putting that section into bending then)
    3. Once the locations of the lugs and the datum have been decided you can start to make the side braces
      1. This is the core of the design, make it so that the side braces connect to the origin of the datum point and the center of the lug holes, using this we can simplify the side brace arms as a truss for hand calcing the geometry
      2. Once you've created the brace sketch you can add required features onto the face of the sketch (CGT163 skills basically) however I'd wait until the other parameters are defined.
    4. For the side braces you need certain details, sketch on face and create these features so that you can have a simple design
      1. How it attaches to the tube
      2. How it attaches to the cross brace
      3. How you can support the undertray
    5. Next is the cross brace, here we need to use the features on the side brace to create the cross brace, for this load it sideways and make a bracing structure, we want to avoid making this thing a four bar linkage
    6. After all this add your edge blends and then chuck it into ANSYS to verify your hand calcs and to make sure it actually works.
