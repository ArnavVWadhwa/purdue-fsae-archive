Opening A Part

Search your part. You can do this in the search box on the left of NX.

Tip

Want to see every item relating to PF24? Use a wildcard to represent a string of characters that is missing. For instance, entering PF24* in the search bar will yield the following window.

Entering PF24_CHA_* will show all CHA related parts. So on and so fourth


Make sure your part is selected

Click the NX button

Complete


Making A New Part

Easiest to do this in NX 

Open NX through Teamcenter

File → New → Item

or 

Ctrl + N

  1. Make sure it is a Model, and only a model! 

2. Make sure to SET YOUR UNITS PROPERLY: WE USE INCHES

3.  Name your part file with accordance to the CAD Naming Convention - Purdue Formula SAE - Confluence (Link)

4. Go ahead and do the Double-click to assign command

5. Name  

Deleting A Part

You are only able to delete parts that YOU create. Even with admin permissions. 

  1. Make sure you are the owner of the file
  2. Make sure the desired part is selected to delete
  3. Click the red X button

Copying a Part

Hit CRTL + SHIFT + A in NX with your part open

  1. Make sure that your work part is the part that you want to copy
  2. Make sure you select new item - currently, we don't use revisions
  3. Make sure to select the folder that you want your files to be housed in
  4. Enter your part ID with accordance to CAD Naming Convention - Purdue Formula SAE - Confluence
