Purdue Formula SAE : Traction and Launch Control in MoTeC M1 Tune

Created by Unknown User (reyno105), last modified by Unknown User (xia107) on Sep 23, 2020

Figuring out how launch and traction control in M1 Tune took some work. It's pretty simple, but the M1 Tune explanations are wak (imo).


Note: to be able to find these settings, go to the tuning tab, then all calibrate. Turn off the check in Normal. The check in normal only shows you channels marked as normal, which does not include relevant traction control settings.

Traction Control

Here's how traction control works:

  • There is a map of target slip ratio vs throttle position: (Traction → aim → main)
    • You can use one of the rotary switches to select entire separate maps
  • The target slip ratio from this map can be adjusted by:
    • Lateral acceleration and / or speed (Traction → Aim → Compensation → Vehicle Acceleration Lateral):
      • The slip ratio target is increased or decreased by a percentage specified as a function of speed and lateral acceleration
    • Gear position (Traction → Aim → Compensation →  Gear)
      • Each gear can adjust the target slip ratio. Would mostly be used to disable in certain gears, or if to make sure it is disabled if gear position is unknown)
  • This target slip ratio with modifiers becomes a target rpm.
  • Once the target rpm is exceeded, an ignition / fuel cut starts.
    • This is controlled by traction → range table
    • This table is a percentage as a function of speed. If your target rpm is 10000, and this is set as 5%, ignition and fuel cut will start at 10000 rpm and then be fully cut by 105% of 10000 rpm (10500 rpm)
      • The entire table gets swapped with each position on a rotary switch (if mapped)
    • Fuel margin (traction → engine speed limit → fuel margin) is a value in rpm that specifies how much later fuel cut should begin than ignition cut
  • There is a minimum speed for the system to turn on (Traction → activate → vehicle speed)

Launch Control

This one's less opaque. Pretty much a max rpm vs speed is set. Use it to limit rpm below where TC takes over for a faster transition. rpm is limited via ignition retard (with a PI loop). There is an rpm range that can specified that is how much above the target rpm you can go before ignition / fuel cut is used instead of just ignition retardation.
