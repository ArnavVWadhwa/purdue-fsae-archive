What is NX and Why Does the Team Use it?


Accessing NX

To access NX, remote desktop into a Purdue computer that has it. This page has information on how to do that: https://slic.ecn.purdue.edu/public/remoteavailability

Use computers in KNOY, ARMS, HAM, and BIDC labs as these should have NX (the version of NX is not important) and be accessible to members of any major. To find these, just enter one of those terms into the search bar on the page linked above

When opening the program, be sure to select "NX 12" as opposed to "NX 12 Layout"

Modeling Primitives (START HERE): 

Modeling primitives is a good first step into CAD if you have never used a CAD package before. The goal of this process is to create the base shapes using as few operations as possible. See the below video for examples of the Cube / Rectangle and the Cylinder. For the others use tools learned from the video to generate the shapes on your own.

  • Cube / Rectangle
  • Cylinder
  • Sphere
  • Cone
  • Pyramid
  • Ring

More Advanced Sketching: 

These activities will build on skills learned in the Primitives activity. The attached video goes through the first two examples. Work through the remainder on your own should you feel like you need more work in this area. All examples are given as 2D drawings, just extrude these to 3D before saving to show the sketch had no gaps.

Bottom-Up V Block Modeling: 

This is the next step from the Primitive model and will use skills from those exercises as well as introduce the concept of assemblies.
