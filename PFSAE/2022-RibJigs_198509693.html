Purdue Formula SAE : 2022 RibJigs

Created by Unknown User (aroychow), last modified on Jan 13, 2022

This page is going to cover a new method we're trying to get better dimensional accuracy on our airfoils (FW, RW, and SW).

Old Construction Method

  1. Place lower skins in their respective molds (to help them keep their shape)
  2. Attach ribs to the skins using epoxy, and use the set time to get them positioned as accurately as we can.
    1. We position them by eye, measure and see if they're where they need to be.
    2. Note: we also had some waterjet foam "ribs" that were attached between the skins, but they were not visible externally. (Only there for dimensional accuracy and additional attachment between skins).
  3. Add some more epoxy on top, and place the other skin.
  4. Lift the top skin to see if there's enough epoxy contact for a proper bond.
  5. If so, place it together, maybe with some weights on top, and let it cure in the mold.

The main thing here is the initial placement of the ribs on the skins, which is done by eye. This year, I'm making a set of jigs so that we have better accuracy for rib placement, both across the span and in the lengthwise direction.

The concept:

Make a jig + baseplate system that lets us accurately position the ribs (relative to each other). Using this, we can position the skin more accurately, and once we trim the excess, we should end up with rib positioning accurate to the mm (approx. 0.04").

New construction method:

  1. Attach the correct jigs to the baseplate using the pre-spaced mounting holes.
    1. (Everything uses #8 screws, which are the same ones used in the actual wing construction.)
  2. Attach ribs to the jigs (including #8 spacers)
    1. <TODO: Add spacer dimensions / part #>
    2. We use the spacers so that any excess epoxy doesn't glue the skins to the jigs.
  3. Attach one skin to the ribs (epoxy, etc.) Make sure it is positioned correctly on the ribs.
  4. Let dry (enough).
  5. Flip the wing and attach to the jigs again, with the already-attached skin flipped <TODO: Insert sketch/image>
  6. Attach the other skin in a similar manner.
  7. Theoretically, we now have the most dimensionally accurate rib positioning we've ever had.

The rib hole spacing has been consolidated to minimize the number of unique jigs to be made. As of now, there should be:

A pair of  "GIGA" Plates, with holes for:

  1. FW Inner Flaps
  2. FW Outer Flaps
  3. SW Flap 1
  4. SW Flap 2
  5. RW Mainplane
  6. RW Flap 1
  7. RW Flap 2

And some extra, bespoke plates for the inner ribs of the FW and RW mainplanes.

Quick notes about the design:

  • All span measurements are from the outside of their respective ribs.
    All chord measurements refer to the furthest distance between two screw holes for any given rib.
  • Screw holes are as follows:
    • Baseplate gets 0.1360" holes (#29), 8-32 screw tapping drills
    • Gigaplates get 0.1695" holes (#18), close fit clearance drills.
    • Number Machine Screws Tap Drills Chart (csgnetwork.com)
  • Spacers are 0.25" thick (assumed, we have some lying around apparently)
