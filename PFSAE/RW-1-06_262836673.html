Purdue Formula SAE : RW-1-06

Created by Matthew James Lamear, last modified on Jun 19, 2023

Simulation ML-007

Changes/Justifications

Description of the ChangePictureJustification
Added a fourth element that is chopped off at the TE.

To try to reduce flow separation, I removed part of the TE as the low-pressure region generated by the gurney should now be larger. Additionally, the airfoil should still act as if that portion is still there, so pressure recovery should still be in check.

Simplified the endplate airfoil to be constant along the whole endplate.

To improve the ease of manufacturing, this part has been simplified. Also increased the thickness of the airfoil to not have flow separation.

CFD Analysis

CFD PictureDescription

Note: Cl monitor was not fully converged, but it was close enough to just get the scenes, would have certainly increased the Cl as the UT was gaining prior to stopping the sim.

Cd is a big problem with this iteration, as expected when adding an additional element. Need to do some fine-tuning but I think this strategy makes sense.

I think reducing the size of the gurney will improve the high drag, seeing as a lot of the drag is coming from the flow sep at the end of the airfoil.

Reducing the chord of the exit flap is the next step here, going to try to eliminate the gurney in favor of an additional flap.


The endplate airfoil worked perfectly, this is going to be what moves forward to future iterations. Also no low energy flow above the main element!
