Purdue Formula SAE : Fuel Injector Setup M1 Tune

Created by Unknown User (jpeerbol), last modified on Jul 15, 2020

I had a difficult finding this information so I am including it here. To setup fuel injector calibrations under 2: Initial setup > 0 All calibration > Fuel Injector Primary Calibration, is a drop down menu for fuel injector calibrations, these are calibrations by Motec so they are accurate (make sure your M1 has updated calibrations, go to Help > Check for Updates), you most likely will not have an injector that does not have a calibration, so you will need to select manual calibration. Values for the injector calibration will then be able under 7 injectors tab. I read somewhere that Motec will make characterizations (calibrations) for injectors if you send them the injector and they don't have it yet, (I think for free) I'm looking now and can't find any details... bummer.
