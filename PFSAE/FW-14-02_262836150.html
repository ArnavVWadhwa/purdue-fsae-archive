Changes/Justifications

Description of the Change

Picture

Justification

Main element reprofile

Reduced S2 ride height to 2 in while keeping S3 ride height at 2.2 for additional downforce.

Reduced center section ride height for additional downforce. The concern is losses going into undertray.

Increased S0 suction side camber for additional downforce.

Second element reprofile

Increased root chord increase downforce.

Fourth element reprofile

Increased mid section chord to continue to use the entire rulebook despite main element changes.

Increased gurney span for additional downforce.

CFD Results Analysis

CFD Picture

Description

A lot more inwash due to loading closer to the center, limiting undertray performance.

Increased losses in the center.

The outboard vortex is visibility being inwashed more.

Increased center loading upwashed a lot more wake, limiting rear wing performance.
