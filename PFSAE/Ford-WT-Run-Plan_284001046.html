Purdue Formula SAE : Ford WT Run Plan

Created by Matthew James Lamear, last modified on Feb 28, 2024

Test Date: March 8th

Key Tests

  1. Use tire gurneys to match the separation line to the moving tire separation line in yaw 0, 5, 10, 15.
  2. Verify mini-wing effects.
    1. If detachable → try one run with and one run without, then compare Cl Cd of the full car.
    2. Smoke wand at minimum to verify its existence. 
  3. Verify strake flow field/separation characteristics.
    1. Use flow vis to test wing separation off the innermost strake.
    2. Potentially find a particle location that gets inwashed from the side to visualize strake system in yaw 15.
  4. Re-verify RW performance and separation characteristics.
    1. Use tufts and/or flow vis for separation at various AoA.
  5. Ensure no UT diffuser flow sep at kick points and side tunnels.
    1. Test steadiness using tufts, use flow vis here as well.
  6. Map Cl/Cd at various AoA for RW and FW

Test Plan

  • 4 runs per hour for 8 hours (32 runs)
  • 30s of data record
  • Run number is in order of priority.
    • means undecided if I want to keep this the way it is.

Sweeps without having to change vehicle configuration are faster (3 minutes in between data records to reach steady flow)

Definitions

Baseline - All aero components at standard AoA, and components used in CFD are used in wind tunnel (i.e. Miniwing on, RW swan neck fairing attached, tire gurneys attached).  Fan off unless specified.

alpha (α)- The most common yaw angle the car sees based on grip-limited speed weighted yaw cases using the yaw lap sim Chenkai made. (0 deg)
