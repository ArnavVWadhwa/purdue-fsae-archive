Purdue Formula SAE : How to Make a Connector | RapidHarness

Created by Paradise, Hunter, last modified on Dec 06, 2023

Background:

Most connectors have already been created in RapidHarness however, it is possible that more will need to be created as new components are used and harness design evolves. This is not difficult to do, but there is a procedure to do so to ensure consistency and that the appropriate information is included. This tutorial also applies to many other components such as heatshrink, ring terminals, etc.


Basics:

Step 1: Locate Connector

For this example we will be using a 4-pin DTM plug found here: https://www.milspecwiring.com/DTM-4SK_p_64.html.


Step 2: Fill In Part Number, Manufacturer, and Description in RapidHarness


Step 3: Add Component Link


Step 4: Declare Connector Cavities

For most connectors this is as simple as indicating how many pins there are 1-X. For more complex connectors it may be necessary to utilize more complex identifiers.

Typing 1-X in the cavities box will automatically create cavities 1, 2, 3... X for you.



Step 5: Locate Suitable Connector Image

This image should represent the connector variation being used. For example, do not put an image of a non lipped connector if the part name and link both lead to a lipped connector. If this does not create confusion for you, it will confuse someone later down the line.


Step 6: Remove Image Background

This will allow the connector image to be overlaid with the actual connection in RapidHarness which makes for easier identification of connectors and connections.


Step 7: Add Image to RapidHarness


Step 8: Create Version

Versioning allows the connector to be used by other people
