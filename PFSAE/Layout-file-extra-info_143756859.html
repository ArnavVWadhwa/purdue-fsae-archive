Purdue Formula SAE : Layout file extra info

Created by Unknown User (reyno105), last modified by Unknown User (agarw162) on Jun 08, 2022

Video overview:


A layout file structure is incredibly useful for two reasons:

  1. It allows you to design all the parts that interact in an assembly at once in the same file
  2. It allows you to include all the reference geometry you want without ever needing to get it all hidden properly
  3. It creates parts that are very lightweight - they do not include any reference geometry or even all the various features like sketches, extrudes, assembly constraints, and fillets that actually make up the part. This allows larger assemblies to be opened and looked at without bogging a computer


How to make an assembly with the layout file structure.

  1. Per the naming convention, the layout file should be 5100 for example
  2. Make this part,Â 
  3. Assemble into it all the reference geometry you could ever want. No need for linked bodies, but just use add part.
  4. Design all of your parts using the reference geometry as well as any bottom up methods (sketches, extrudes, etc.)
    1. To use reference geometry directly, be sure to change the selection scope FIXME
  5. To organize the various separate parts you made, use folders in the feature tree
  6. Once you're done designing the parts in the layout file, for each individual part:
    1. Use "new part" to make the new file as a part in the layout file assembly
    2. Double click the new part in the part tree. This makes that part the active part. That means that you are editing that part, but the stuff in the layout file is still kind of there.
    3. Select wave geometry linker. Select body and then select the body in the layout file that corresponds to the part.
    4. Double click on the layout file part name at the top of the part tree to make it the active part again
    5. Right click on the name of the part you just made in the part tree, then select "open in new window"
    6. Select reference sets. Add the body you've added to the part to the reference sets.
    7. Set the material properties
    8. Now repeat for each part
  7. Once you've made the files for each individual part, just make an assembly out of them as you usually would. You can add any hardware to this assembly as you usually would. If the parts aren't showing up in the assembly, it's probably because you didn't add the body to the reference set.
