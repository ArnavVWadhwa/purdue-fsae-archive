Purdue Formula SAE : Updating Interpart links

Out-dated links

Parts that are "not fully open" in an assembly when changes are made will have out-of-date links (greyed out check mark)

Parts with a non-boxed red check mark are "open partially" and could contain out of date links

To make sure links are updated, make sure parts are shown with a boxed red check mark in the assembly navigator

Delay Link Updates

When working with a layout file that references geometries from other parts using WAVE Geometry Linker, any changes made upstream can lead to immediate update to downstream components / child parts. This could be either desirable or undesirable.

Reason why you want links to update immediately:

  • Makes sure all related parts and features are up to date
  • Allows you to quickly view the effects of changes to a parameter on the entire model

Reason why you don't want links to update immediately:

  • Linked features are prone to breaking if not modelled properly the first time
  • Constant updates after every small change takes more time if several changes to upstream components need to be made

The feature "delay link update" does what is says it does - delay update of links


This allows you to make changes to the model without effecting linked features or geometry

Updating all links

There are 2 main ways to update parts with links: "Update Links" and "Open Component Fully".

The difference between these 2 is that one way allows you to control which links are updated and the other enables all links to be updated. 

To update links for selected components, use "open component fully"


To update all links, use Tools → Delay and Update → "Update Links" 

Note: this will update ALL links within your assembly, doing this for big assemblies can take a hot minute.


This is not exhaustive, take everything i said with a grain of salt
