Run a tap through the jig a couple times after you've formed the thread


Prioritize keeping a base layer of jigs that won't be removed after the frame is welded

this sucks! now we've got to remove the base jig on the green tube.

Make doubles of the tab jig plates if possible / if you have the stock.

It saves a lot of time because people can continue jigging while a welder is working on a tab.


Diff Tabs don't need to be jigged. The diff itself will be used to jig these tabs with the engine in place.
