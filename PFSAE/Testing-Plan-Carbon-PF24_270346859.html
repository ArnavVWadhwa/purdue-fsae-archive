Purdue Formula SAE : Testing Plan Carbon PF24

Created by Vasilev, Dimitar Kirilov, last modified on Sep 16, 2023

Testing Parameters in order of importance

  1. Bond Gap
  2. Bond Diameter
  3. Bond Length
  4. Resin Choice

Prior to Testing

  1. Manufacture inserts
  2. Prepare inserts 
    1. Sand and clean
    2. Coat in alodine
  3. Prepare carbon tubes
    1. Sand and clean the inside of the tube
  4. For each carbon-insert pair measure and record the OD of the insert and the ID of the tube
  5. Label the sample so we can identify the sample and its ID/OD parameters
  6. Resin prep
    1. Identify bonding temperature and bonding time 
    2. Identify the amount of glass beads to be mixed with the resin (grams per ml) 
    3. Mix resin and hardener
    4. Add and mix glass beads with the resin
  7. Bonding
    1. Clean both the insert and the inside of the tube
    2. Identify the approximate resin amount per bond
    3. Bond the insert to the tube paying attention to the amount of resin used (don't use too much or too little)
    4. Leave resin to cure at the appropriate temperature and duration

Testing Procedure

  1. On the first sample
    1. Load starting at 0 up to the point of failure
    2. Identify the point of failure
  2. Record graph and data and make sure it is saved and ready to export
  3. Load each consequent sample starting at 60% of the max load
  4. Record the graphs and data of each sample and make sure it is saved and ready to export
  5. Export graphs and data for each specimen
  6. Repeat for each sample case/ testing parameter

Post testing

Preparing the data

  1. Average out the ID of the carbon tube and OD of the insert
  2. Average out the max load capability
  3. Find standard deviation
  4. Repeat for each test sample

Plot the data for each testing parameter

Identify manufacturing consistency

Identify trends in performance and evaluate optimal bonding parameters
