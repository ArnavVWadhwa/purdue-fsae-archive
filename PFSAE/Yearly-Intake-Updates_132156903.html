Purdue Formula SAE : Yearly Intake Updates

Created by Unknown User (kruege18), last modified on Nov 01, 2019

2020

The 2020 intake is largely based on the 2018 intake designed by John Foster. 

Updates include:

Shorter (3.9 inch) runners to target 11000 RPM, instead of 6900 RPM. this decision was based on a histogram of average RPM while under 85% throttle during the fastest 2018 Autocross run. This run was chosen because it was a fast run performed by a very talented driver. This run represents ideal "pushing" conditions for the car on track, similar to what we would hope to see during autocross/endurance runs at future competitions. 

Runner profile was also straightened to improve flow characteristics through the runners into the ports.

Plenum volume, diffuser design, throttle body design, and trumpet design were all kept the same in the interest of time. 

This design requires validation across the board.

Modification order should be: Runners>Diffuser>Plenum>Throttle Body>Trumpet
