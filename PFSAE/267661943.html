Purdue Formula SAE : Layout Files, Part Files, and Assembly Files

Created by MacNab, Zachary Robert on Aug 07, 2023

Here are the differences between layout files, part files, and assembly files. 

All files referenced on this page are .prt files in NX! These files are team standards, not NX standards. The only thing technically distinguishing them from each other is only our naming convention 


Layout

This file is where you start when making a part. Layout files are meant to be where all of the "messy" CAD work gets done. In actuality this work should be very neat and parameterized, but when even when you do CAD right, for a complex model these files can just look messy.

In your layout file, you should have things like your sketches, datum planes, hole features, extrudes, revolves, etc. in this file. 

Part

Once you have your part made from your layout, we want the solid body that the part makes, but we don't want all the messy sketches and extrude features. If we kept these features in there, not only would the whole car look really messy with everyones reference features, but we would also be waiting ages for the car to load because the computer has to go through and perform each extrude, revolve, hole operation, etc. when we reopen the car.

Ideally, we would have a tool that takes the solid body from our layout file, and stores exclusively the body. That is exactly what WAVE Geometry Linker does. What's nice about this feature is that it doesn't slow all our computers down and only loads what we need in order to see what we care about, the geometry of each part of the car. Note that you can't edit the part from this file! You can only edit the geometry of your part from your layout file

Assembly

An assembly is just a collection of part files! Its super useful for organizing our CAD and keeping everything very streamlined. 

Reference Geometry

Need a file to store your chassis templates? How about aero keepout zones? Reference Geometry files are there for that reason. These are kind of like Part files, but are not meant to be actually made or produced.
