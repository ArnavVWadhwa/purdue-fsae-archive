Purdue Formula SAE : Using the Track "Server" - WIP

Created by MacNab, Zachary Robert, last modified on Apr 08, 2023

I used my old laptop to create a "server" for the track. It's useful for sharing data quickly at the track so multiple laptops can be used to view data, and we don't need internet to use it. Here's how to use it. 

Overview - Readme

Everything is based on a simple windows feature with sharing files if you're on the same network. It's not on OneDrive, and it does not rely on the internet at all. The simplest way to think of this is that all the laptops are just simply connected via the network switch, meaning they can communicate with each other. This creates a local network amongst the computers. Then, one person shares a specific folder they want over the network. Everyone can then connect to the "host" computer, and read the files. 

Details

Host

Ideally, the host laptop has a PurdueFSAE login. I made one on Matt's Acer laptop. Login details are below.

Username: PFSAE

Password: yamahar6letty

The recipients will login to this account to access the folder.



Recipient

  1. Make sure network sharing is turned on. If not, you should be able to go to the network tab in your file explorer and there should be a banner at the top offering to turn it on.
  2. Map a network drive. 
    1. Replace "MyReadyNAS" with the laptop's name in "About My PC". Replace "Media" with the folder name that you want to connect to. 
  3. Done.
