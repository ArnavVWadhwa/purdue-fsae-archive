Purdue Formula SAE : Ergo Test Stand PF23 | Design

Created by Unknown User (mgal), last modified by Dien Quang Nguyen on May 27, 2023

Remember this is not an exact chassis reference, it is more a tool to get data of relative measurements. Also before manufacturing this stand there still is a lot that can be done. Some examples of this can be adding a shifter and other components, reducing weight and cost, etc.

One critical thing that needs to be done is to finalize the dimensions. Most of the dimensions currently are based on PF22 and are arbitrary. Please refer to this confluence page (Dimensions) and the component design pages to finalize the dimensions. The CAD model is fully parametric as long as all the parts are fully loaded.
