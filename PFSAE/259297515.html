Error NullReferenceException

This error could be due to a few reasons, but I have only encountered 1, so below is how I resolved it.

This is likely due to the script being run (via Alt + F8) without any part file loaded. To resolve this, create (or open) the desired part file for suspoints, then with that part selected as "work part" (the part name on Assembly Navigator tab should be bolded), run the script using Alt + F8.
