Purdue Formula SAE : MIS Weather Analysis

Created by Zhang, Chenkai, last modified on Jul 29, 2023

Method

A python script using meteostat API was created to investigate historical weather at MIS in May and come up with estimations of the kind of weather we should see at comp. The python script is included in the AeroSensitivitySim GitHub repo, since that's the motivation behind this study.

Wind speed and direction from May of past 11 years (2013-2023) were obtained with the API. Although wind speed data is complete, there is a significant portion of wind direction data missing (220/341 data points are available). The complete data of wind direction was only available from 2020 onwards. I am not too concerned about the missing data though since it's the overall trend we are looking at.

Results

From the wind direction histogram, we can see that there is no real representative wind direction we can be designing for. It looks like there are dips in the 150-330 deg axis, but the dips are too small to be meaningful from an aero design stand point.

Wind speed data is a lot more interesting. The data has a mean of 7.14 mph and median of 7 mph. From the wind speed histogram, we can see that the most probable wind speed is between 5.5 and 7.5 mph. We can also calculate the likelihood of wind speed, the table is shown below. I would like to do a formal confidence interval calculation with this, but the data is only 83% normal (tested with Shapiro-Wilk test).
