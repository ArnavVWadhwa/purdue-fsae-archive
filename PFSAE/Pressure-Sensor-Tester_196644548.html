Purdue Formula SAE : Pressure Sensor Tester

Created by Unknown User (dnocon), last modified by Unknown User (jpeerbol) on Nov 18, 2021

Problem: 

Sensors are relatively fragile. Often in the car runtime data that we collect we see measurements that are out of the normal range we would expect. This could indicate a serious problem in the powertrain. However, it is usually a broken sensor. A standard pressure sensor on the car is https://www.milspecwiring.com/75PSI-A-MOTEC-PRO-SENSOR_p_375.html

We need a method to test questionable pressure sensors so we can make sure they are reading correctly.

The idea is to make a manifold that can adapt to pressure sensors. This manifold should have a way to be pressurized. An appropriate analog pressure gauge should also be incorporated so we can see the actual pressure inside the manifold. This analog pressure gauge will be used to manually check the measurement displayed by the questionable sensor.

Things to consider: (i have not thought of a full solution, so it will be up to you to do research and own this)

  1. Can we pressure test the sensors using air?
    1. these sensors are used to measure the pressure of liquids (in our use case, oil pressure and fuel pressure)
    2. if we pressurize with air will it be similar?
      1. my guess is yes, but i want to confirm this with further research
  2. How do we plan on pressurizing the manifold?
  3. Which fittings will we need to test sensors?
    1. need to adapt the fitting the sensor uses to the manifold
      1. research the sensors we would be using
      2. find the best method to quickly interchange sensors to the manifold
  4. Choosing an analog gauge
    1. which range of pressures will we need to be able to test?
    2. get a gauge that will be in the range we are looking for
  5. make a case for this
    1. we'll probably be using this tool for a long time so we should make it nice
  6. Try to find off the shelf components and just custom design the minimum things possible.
    1. A good place to look for fittings and such would be https://www.mcmaster.com
    2. We typically purchase sensors from https://www.milspecwiring.com
    3. A good place to find motorsport fittings would be https://www.summitracing.com
