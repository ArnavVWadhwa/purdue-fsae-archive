Purdue Formula SAE : Keizer Wheel Shell

Created by Unknown User (xia107), last modified on Dec 05, 2021

We get our wheel shells from Keizer. Because of shortages this year, we weren't able to get the hole pattern we needed so we ordered the blanks. and machined it ourselves.

There isn't much remarkable about this setup or CAM other than the fact that there was a lot of them to run:

2 shells per corner X 4 corners X 2 sets of wheels = a lot of shell blanks to run.

To make this go faster here are some guidelines:

  • There is no need to touch off and re-set G54 for each part. Just set 2 toe clamps in the mill table and dont move them until you're done. Push each shell into those 2 toe clamps and then hold them down by the flange each time.
  • You will have to re-set G54 between short and tall shells


Other Notes:

  • This is a spun aluminum part, which means it is very soft. Use a 2 flute endmill with lots of coolant and slow feeds, otherwise the mill will gum up with aluminum
  • Use a contour instead of a pocket for the center circle otherwise this will take forever. The cutout will just fall through. Use a big wrench to make sure the mill doesn't flip it out when the cutout is about to separate.
