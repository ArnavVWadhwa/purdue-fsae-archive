Purdue Formula SAE : Poly V6-1 First Order Upwind Initialization

Created by Zhang, Chenkai, last modified on Apr 17, 2023

Background

  • First order upwind is highly diffusive, which means it's very stable but doesn't produce accurate results
  • The stability of first order upwind can be exploited to "initialize" the flow field before second order upwind takes over to achieve a converged solution
  • The stability allows for high relaxation factor, which increases the rate of convergence drastically

Settings

  • First order upwind (for both segregated solver and SST model)
    • Iterations: 200
    • Velocity relaxation: 0.9
    • Pressure relaxation: 0.3
    • SST relaxation: 0.85
  • Second order upwind
    • Velocity relaxation: 0.7
    • Pressure relaxation: 0.3
    • SST relaxation: 0.6

Results

First order upwind (final CL: -3.644)

Final converged results (final CL: -3.692)

The difference between this final converged results and the regular V6 is caused by relaxation factor. Ran a sim with the old relaxation, it converged in more ~3013 iterations with an average CL of -3.747
