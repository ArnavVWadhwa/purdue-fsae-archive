Good Tungsten to order

For Steel: (chassis welding, anything steel basically)

2% Ceriated 1/16" and 3/32"

1/16" provides more control but is rated for less current

typically we weld the whole chassis with 1/16" because it provides the best control


https://weldfabulous.com/ck-t1167gc2-2-ceriated-tungsten-electrode-1-16-x-7-pkg-10/

For Aluminum:

1.5% Lanthanated (Gold) 3/32"

Ck makes some quality stuff, can't go wrong

https://weldfabulous.com/ck-t3327gl-1-5-lanthanated-tungsten-electrode-3-32-x-7-pkg-10/

*you can probably use different tungsten types, but we have been using these two types for years without issues. If you want to experiment to see if something is marginally better, be my guest, but we know these types work
