Purdue Formula SAE : Bulkhead Studies

Created by Unknown User (green338), last modified on May 18, 2020

Bulkhead Lift

PF21 sweep of bulkhead from 0 to 3 inches resulted in the following data. These studies used the 2020 FW.


Based on these studies the ideal bulkhead lift is 0 inches (flat). Mass, Cg, and Cd*A are constant for all positions, so the highest downforce option (flat bulkhead) will maximize points.


Bulkhead Size

PF21 sweep of bulkhead sizes used 4 cases. 



Used the same parts as the above simulation (2020 for all parts except for sidewing and chassis). Radiators are in 2020 position and are partly inside the sidewing with no ducting, so mass flow numbers are much lower than expected.

These studies show the ideal bulkhead size is 12x12 or 12x14. These options complicate pedalbox mounting, so bulkhead size will be kept the same for 2021. This will also allow for the reuse of nosecone molds.
