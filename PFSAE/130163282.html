Purdue Formula SAE : Welded Tab / lug Sizing

Created by Unknown User (ehoerl), last modified by Unknown User (reyno105) on Oct 13, 2020

This is a guide for steel sizing tabs / lugs that need to be welded. It comes from team experience of how thin of a part we can successfully weld.


Structural 

All structural components should begin with an 0.071 tab and should be sized from there for the expected load. Example of these components include control arm tabs, shock tabs, wing mounting tabs, etc. 


Note: high load components such as diff uprights and regulated components such as antisub tabs may require thicker tabs and should be looked at more closely.

To size tabs, use the Lug Margin Calculator.


Non-structural

Non-structural components such as catch cans, fuel tank, etc. should be mounted using 0.05 thickness tabs. 


Note: all tabs should have a semi-circle as the outer profile
