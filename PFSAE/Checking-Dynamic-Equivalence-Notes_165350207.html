Purdue Formula SAE : Checking Dynamic Equivalence Notes

Created by Unknown User (xia107), last modified on Dec 17, 2020

Typical and Troubleshooting

  • Select local machine for working directory
  • Open former and updated models for comparison
  • Adams may be unhappy if you don't have the subsystems in the "private" folder that it automatically made on your computer
    • Copy over the one from the drive if necessary

Checks

  • General once over of all suspension points, is stuff that's supposed to have changed different?
  • Static Vehicle Characteristics
    • Check Wheel center rise is small, proves preload is set correctly
    • Check that some of the values match PF20 design binder
      • Roll Center Height, Roll Camber Coefficient, Camber change, etc
    • Check preload, has to do with spring installed length
      • Use installed length of spring (200mm for the ohlins ttx25), but include preload (corner weight minus unsprung mass)
    • Compare .svc files
      • Check wheel rise, should be close to zero (less than 80 thou or so)
      • For SVC, change unsprung mass percentage until desired corner assembly weight (~40 lb)
