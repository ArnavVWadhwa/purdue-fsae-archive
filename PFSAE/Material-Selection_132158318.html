A Couple Notes:

  • https://ntrs.nasa.gov/citations/20120001798 - NASA spec about material selection
  • The values listed are approximate for quick comparisons between the materials. The final design values should come from MMPDS. This videos on this page (Structures Analysis Documentation Main Template (and MMPDS instructions)) has info on how to access MMPDS and use its values.
  • Young's modulus is the exact same as Elastic Modulus, denoted by (E).
  • The units for Young's Modulus listed are ksi*10^3. This matches how MMPDS provides these values.  10^3 ksi = 1000 ksi = 10^6 psi
  • The vendors are listed in order of preference. We try to make a few big AED Motorsports orders each year and pick it up all at once to save on shipping.
