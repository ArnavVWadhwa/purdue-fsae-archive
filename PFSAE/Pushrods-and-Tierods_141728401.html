Purdue Formula SAE : Pushrods and Tierods

Created by Unknown User (reyno105), last modified by Unknown User (xia107) on Dec 05, 2021

Tierods

Following the carbon tierod failure on PF20, we are going back to steel tierods until better proving can be done on carbon and aluminum bonding.

  • Machine Inserts, combination of lathe and then milling with the collet block.
  • Cut tubes to length
  • Make friends with a welder
  • Only significant detail is to make sure each tierod has a LH and RH insert so it's actually length adjustable.


Pushrods

We still deem that it is acceptable for pushrods to be carbon for the weight savings. This is because the pushrod is primarily under compression (duh) and that load should be mostly taken by the flange interface rather than the bond.

  • Machine Inserts, combination of lathe and then milling with the collet block.
  • Cutting the carbon tubes and bonding has usually been an aero responsibility.
    • Give Aero the inserts and the lengths for the push rods
