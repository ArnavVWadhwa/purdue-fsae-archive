Adams Notes

Adams files: Z:\Knowledge_Base\01-Suspension\06-Adams_Car_Vehicle_Files

Open Assembly, then repeat name in database management

Select directory, where it runs sim, and saves, needs to be on C drive

  1. Run simulations at slowest speed of vehicle (14.5 mph)
    1. Constant velocity cornering, name vehicle and test, 1000 steps, 2nd gear
    2. Set Gs to be a bit more than limit
    3. it'll say that it falied because it didn't hit velocity, but it'll say the accel that it hit
  2. Run straight line accel
    1. name appropriately, 14.5mph, 1000 steps, 2nd gear
    2. 1.3 g, slightly more than limit
  3. Braking sim
    1. Same menu as forward accel, Rename appropriately
    2. -1.9g, set decel method to braking

Now we've run the right starting files

For every run: acf file and adm file, open in Notepad++

adm: all the into about vehicle, should never have to mess with

acf file: commands that run certain simulation

open newly made acf files and match with template, compare for braking, accel and cornering

Change the template to match, save as new revision when doing so

  • comments are lines with !, don't worry about changing that
  • Make sure the lines starting with control match
  • also the functions below that
