Purdue Formula SAE : PF23 Toe Adjustment Project

Created by MacNab, Zachary Robert, last modified by Paradise, Hunter on Aug 16, 2023

Paradise, Hunter owns this. 

Goal: 

Make a robust and accurate toe adjustment system for PF23. 

Timeline:

  • Finish Preliminary Cad31 Mar 2023
  • Design Review 06 Apr 2023 

Design Requirements:

  • Under $150 USD 

  • Needs to function properly with the car off the ground at any height

  • Needs to return to the same position each time
  • Measures toe to 0.1˚ precision
  • Needs to be able to make a toe change in 3 minutes or less with 2 people. 

Ideas:

String Based Adjustment

Strings were considered similar to professional motorsport practice.

Claude doesn't like that, and for a pretty good reason. It doesn't seem terribly accurate, and it takes a bit of time to set up. Setups should be quick and easy, otherwise, we wouldn't want to do them. 

Longitudinal Laser Based Adjustment

Lasers in place of strings were considered because they would allow for a single mounting to the chassis.

Same issues as strings but with a slightly easier setup and possibly reduced cost.

Plate Based Laser Adjustment

Laser pointers mounted to the hub of a given wheel point rearward at a chassis-mounted plate that has etched marks for every 0.1˚ of toe adjustment based on pre-done calculations. Benefits from the cost reduction of single chassis mounting while being significantly faster than the alternatives.

Assumptions:

  • Ignore effects of caster and camber on laser path as the toe is changed
  • Assume PF23 CAD is consistent with the manufactured car in full droop
  • Assume the manufactured device is consistent with its CAD
  • Assume the purchased laser pointer points straight relative to its body
