Purdue Formula SAE : Pedal Box PF23 | Design

Created by Rumsey, Helen Nicole, last modified by Dien Quang Nguyen on May 27, 2023

Minimum Viable Product Pedal Box

  • General
    • Pass all rules
    • Has working brake and throttle (duh)
    • No 'break by wire' systems
  • Size adjustable design
    • Must work for anyone between 95% male to 5% female heights
    • Mounted on the floor on rails
  • Weight
    • Must be either fabricated aluminum or steel, or machined Aluminum, steel, or titanium.
    • Aluminum preferred when it is strong enough
  • Cost
    • Machine our own parts
    • Don't use titanium
  • Throttle and braking responsiveness
    • Design for desired pedal ratio
  • Reliability
    • 2 parallel fluid circuits (If one fails, other must still work)
    • Protected with scattershields
    • Both pedals should be able to withstand the max force of the driver repeatedly
    • Must have brake over-travel switch
  • Throttle 
    • Mechanical or electrical actuation
    • Must return if not pressed (return springs in throttle body)
    • Must have smooth operation with no possibility of binding or sticking
    • Must be protected from driver
    • Requires a positive pedal stop
  • Forces
    • The brake pedal and associated system components design must withstand a minimum force of 2000 N without any failure of the brake system or pedal box. (Max force by design judge may be different)
    • Break pedal must be able to provide enough force to breaks to lock breaks

Improvements to make:

Make adjustability work without compromising rigidity

Work with the driver for heel tray/ toe restraints stuff

Use less tabs / bulky tabs by making the tabs be in compression rather than tension.

Place tabs in where the pedal will be in compression to reduce tab size (look at PF20?)

Break pedal did not return to starting position so position the master cylinders to insure return and possibly add a return spring if all else fails.

Throttle: Mount cable without getting in ways of rules and without clunky support

Break bias cable should be strapped down well and make sure the breaks return to start to allow cable to be adjustable

see if we can make the break bias electronically controlled (servo motor)


Lessons to understand before

Do more research before and understand vehicle dynamics

Understand torque bias on axles

Look at Design docs for PF21 for breaks
