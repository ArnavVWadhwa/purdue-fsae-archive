Purdue Formula SAE : Matlab

Created by Unknown User (jpeerbol), last modified by Unknown User (xia107) on Sep 25, 2020

Matlab is available through "sponsorship". Typically Matlab will offer licenses to competition teams (FSAE), but because Purdue already has licenses they give us licenses (this is available to anyone with a Purdue email and says they are researchers(Matlab groups competition based teams and researchers together)).


Go to this link. 

You must make a Mathworks account with your Purdue Email.

Then, when creating a MathWorks Account, for the question "How will you use MathWorks software?", select "Teaching or research in school."

You'll have to confirm you email address and stuff but it should tie a license to your account and you'll be good.


If you already have a MathWorks account tied to you Purdue email instead of using the above link use this one to associate your Purdue email with the Purdue research license. https://www.mathworks.com/licensecenter/licenses/add


You should not have to fill out information about what your team does and stuff, if you're that deep you went too far. Either you already are on the Purdue license or you need to use the second link, see directly above.


If you already have MATLAB installed and just need to start using Purdue's License:

1) Go to Mathworks.com, click on your profile picture in the upper right corner and go to My Account

2) Click the tools icon to Manage the license that you want

3) Under Install and Activate you should have the option to Update License File under Related Tasks, click that. Be sure you are downloading the file for the correct version

4) Download the License File

5) On your computer, open the MATLAB License Activator

6) Select Activate Manually without the Internet and proceed

7) Browse thorugh files and select the license you just downloaded

8) Restart MATLAB, in the command window type License and ensure that the number matches the one you intend to use
