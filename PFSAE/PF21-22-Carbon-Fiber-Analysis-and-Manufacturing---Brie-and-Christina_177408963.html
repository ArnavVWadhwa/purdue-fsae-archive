Purdue Formula SAE : PF21-22 Carbon Fiber Analysis and Manufacturing - Brie and Christina

Carbon fiber analysis for the floor pan, front floor pan under pedal box, and firewall using Tyler's CLT (classical laminate theory woah!) calculator. 

Requirements: 

  • Factor of safety = 1.4
  • Less than 10 mm of deflection
  • No ply failures
  • Quasi-isotropic 
  • Use woven at 45 and 0 degrees
  • Want uni-directional along the 0 (0 is along the x)

Assumptions:

  • Weight (using 95th percentile male) = 102 kg
  • Bump acceleration = 3
  • Modeled as 3 point bend test

Floor Pan:

Variables 

F_max = 102 kg * 9.8 m/s^2 * 3 * 1.4 → (95percentile_male_weight * gravity * bump_acceleration * factor_of_safety)

base = 0.45466 m

length = 0.7366 m 


Ply schedule based on best combination (~10mm of deflection and no failures in any ply)

Number of plies: 5

Ply schedule: 0 90 0 90 0

Ply order: [2 1 2 1 2] - where 2 is unidirectional type plies and 1 is woven type plies

Explanation of how we came up with this ply schedule: Used Tyler's CLT calculator on a vector combination generator with 0 90 45 (code will be in the Formula drive). The lowest deflection was obtained when combining ply types - and making sure there were 0's when using ply type 2 (woven). Then, we used the combination generator on the CLT calculator, increasing the number of plies until we obtained a combination that had a deflection lower than 10mm.
