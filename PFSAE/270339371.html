Getting your files

First, check if the part is already in Teamcenter by looking up the part number. 


Next, make sure you have your part in a STEP or STP file. 

 

For this example, I'll be doing the multimatic dampers. 


In NX

File → Import → STEP242

Do the proper settings per above

Hit this button to reset ID fields and such

Name your part in accordance to the part number that is associated with it. If it doesn't have a part number, put your best name in accordance to PascalCase (defined based on CAD Naming Convention - Purdue Formula SAE - Confluence)

Make sure to fill out the fields as such on the left.


Hopefully you see this message 


Complete
