Purdue Formula SAE : Ergo Test Stand Design Objectives (Please Delete)

Created by Rumsey, Helen Nicole, last modified by Unknown User (mgal) on Jun 12, 2022

One project that would be foundational to chassis development would be to make an ergonomics test stand out of 80/20 framing extrusion and steel tubing (similar to the dyno) or wood. This could be a more robust/detailed version of the cockpit mock up made here.

This cockpit mock up would be useful in determining the following:

  • Head restraint position e.g. how far behind the head restraint is from the MRH (remain rules compliant) 
  • Steering wheel angle
  • Steering wheel position
  • Steering wheel (Momo vs other)
  • Seat belt tabs position e.g. distance from MRH so we don't guess the position the night we decide to weld them
  • Shoulder harness bar height
  • Driver back angle
  • Pedalbox Adjustment
  • Steering wheel force input (at different angles)
  • Maximum Main Hoop Width/ FRH width ??? not as relevant (likely can do without having side bars - mostly to see driver elbows) 
  • Dashboard height (mostly location of color dash) 
  • Shifter Position

Requirements:

  • Fit under the shop table (~30 in.)
