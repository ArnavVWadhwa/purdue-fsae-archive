Purdue Formula SAE : Powertrain Subteam Onboarding Project

Created by Skrzyszowski, Mateusz, last modified on Oct 13, 2021

Pre-Requirements:

  1. Please read Onboarding before starting this project
  2. Please read Powertrain Onboarding before starting this project
  3. CAD Training should be complete

Tools:

  • i2 Standard (Data Analysis Software)
  • NX 1946 (CAD)

Goals:

  • Learn to make data driven design decisions using i2
  • Understand how to navigate i2
  • Learn advanced tools within NX
  • Familiarize yourself with the FSAE Rulebook

Deliverables:

  • Selected fuel tank size with supporting evidence
  • Fuel tank model made in NX
