Purdue Formula SAE : Analog Voltage Input

Created by Paradise, Hunter, last modified on Nov 24, 2023
  • External, calibrated, inputs that measure absolute voltage.
  • Typically used with ratiometric 3-wire sensors such as throttle position, pressure, etc.
  • The power supply for ratiometric sensors can be specified for improved accuracy.

  • 12-bit analog to digital converters with 0V - 6.098V input range and a precision internal reference.
  • Input impedance 100K.
  • 230Hz bandwidth.
  • Transient input +/-200V.
  • Continuous no damage input voltage range -30 to +35V.
  • 8 AV pins on the M130.
  • Supports absolute and ratiometric measurement.
