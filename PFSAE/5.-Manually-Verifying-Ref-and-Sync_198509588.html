Purdue Formula SAE : 5. Manually Verifying Ref and Sync

Created by Unknown User (dnocon), last modified on Dec 02, 2021

Before attempting to start engine, these are easy ways to check that you have setup the engine sensors correctly. It's a nice sanity check.

First and foremost, make sure you are able to get cycle lock. This is required because the following verification tests will require the firing of the sparkplug. 


Verifying Crank Position

Verifying and ensuring crank position is crucial. You do not want to set spark timing to 15deg BTDC and actually have that be 30 BTDC in reality. Things could get crunchy...

Verifying Crank Position is extremely easy. 

Steps:

  1. Connect coilpack to sparkplug (sparkplug can be inserted in engine, it just needs to be grounded)
  2. Set ignition timing to 0deg BTDC
  3. Setup and Point timing light at measurement window. See engine manual on location of this. 
  4. Crank Engine
  5. Check that the light flashes when the engine is at TDC. 
  6. Adjust values slightly so that the light fires exactly when the engine is at TDC. 


Verifying Cam Position

Verifying Cam Position is crucial as well. If you do not get this right, you can run the risk of running the engine "backwards."

Verifying cam position is a bit more difficult. It should be verified after Crank Position is verified.

Steps: 

  1. Disconnect sparkplug from inside the engine an ground it on the engine block. (You will need to visually see the spark)
  2. Set ignition timing to 0deg BTDC
  3. Disconnect valve cover so you can see the cams rotating.
  4. Crank Engine and take a slo-mo video. 
    1. The spark should occur when the engine is in TDC after the compression stroke. Compare the timing of the spark to the position of the cams.
