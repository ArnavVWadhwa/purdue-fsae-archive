Purdue Formula SAE : Bi-Linear Stress Strain Curve for Non-Linear FEA Generator

Created by Unknown User (reyno105), last modified on Apr 15, 2020

This is a simple calculator for creating bilinear stress strain curves that may be plugged into ANSYS for a first pass at analysis including non-linear material properties. The first line goes from the origin to the point where yield would be predicted by a linear model. The second line goes from this point to a point at elongation to failure, ultimate tensile stress. The slope of this second line is called the tangent modulus.

Strain past 2 to 3% for most metals should be avoided both because predictions will be inaccurate and because it's not a good idea in the first place. (The results become inaccurate because the difference between the engineering and true stress strain curves start to matter).
