Why do we use CAD? 

  • Weight tracking:
    • Track Weight (then validate)
    • Track CG (then validate)
    • Track Inertias (then validate)
  • Tool clearances
    • Rivet holes
    • Nut / bolt head access
  • Assembly checks
  • Motion studies
  • Ergonomic studies
    • Driver models
    • Definitive ergonomic metrics
  • 3 View drawings
    • Rules requirement
  • Manufacturing drawings
    • ME machine shop requirement 
    • Validating the part is manufacturable 
    • Keeps track of tolerance stackups
    • Supplier purchased parts
  • Wire harness
    • Routing and mounting considered 
    • Cable lengths for calculations 

Why do we use Teamcenter?

Team center is a PDM (project data management) tool. It is used for CAD file version tracking, implementing processes for error catching, and making sure that we don't step on eachothers toes when working on CAD at the same time. 

  • Part revision tracking
  • Approval processes
    • Used for double checking errors in
      • part interfaces
      • technical drawings
  • File management
    • Specifically making sure that no two people are trying to work on the same file at the same time
  • Easy BOM generation
    • Accounts for every item in CAD – nothing is lost
