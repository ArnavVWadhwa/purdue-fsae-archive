Project Scope (Present problem statement and stakeholders):

  • Problem Statement

    • The team requires a reliable system to convey electrical inputs and outputs for vehicle function, control, and analysis.

  • Stakeholders

    • Powertrain
      • Engine sensors/injectors/coils and engine control systems
      • Fuel/Coolant/Oil temperature sensors
      • Fuel/Coolant pump
      • Manifold/Fuel/Oil pressure sensors
      • Lambda sensor
      • Fan
    • Suspension
      • Wheel speed sensors
      • Shock pots
    • Chassis
      • Steering position
      • Pneumatics wiring/control software
      • Air tank pressure sensor
      • Brake pressure sensors

Design Requirements (Present known requirements, ie. Vehicle Performance, rules, and define unknown requirements):

  • Rules

    • The full list is found on the drive
    • 3 Main Categories
      • Shutdown Circuit
        • BOTS
        • Master switch
        • Cockpit main switch (ESTOP)
        • Stickers
      • Competition Requirements
        • Brake light
        • Transponder
      • Battery Safety
        • A firewall must separate the driver from any lithium battery

  • Known Requirements

    • Maximize Competition Points
      • Minimize weight by minimizing wire run length
      • Vehicle performance data collection and storage
      • Routing that does not interfere with cockpit template testing
      • Minimize CG height
      • Comply with desired weight distribution target
    • Driver Safety
      • Rules-compliant shut-down systems
        • BOTS
        • Cockpit Main Switch
        • Primary Master Switch
      • Crash safety features beyond the rules
      • Highly visible and reliable brake light
      • The firewall must separate the driver and battery
    • System Reliability
      • Fully heat-shrunk and booted
      • All components IP-65 rated or higher
      • Routing that considers interference sources
      • Routing that considers moving/hot vehicle components
        • Drivetrain
        • Driver
        • Suspension
        • Engine
        • Oil/Coolant
      • All components securely attached to the vehicle
    • System Serviceability
      • Quick and easy installation and removal
      • Routing that does not impede other system's serviceability
      • Extra length for each wire to allow for re-crimping if necessary
    • Driver Confidence
      • Push to talk system for all drivers
      • Reliable and easily accessible controls
      • Completely autonomous engine management
      • No critical system failures during the testing season

  • Unknown Requirements

    • Additional sensor/component accommodations
    • Power consumption of components
    • Wire sizing
    • Splice points

Design Overview (Rough concepts for solution + Trade study with a recommended path forward):

  • Architecture
    • Bundled wires
    • Heat shrink
      • Provides more strain relief and protection (especially if properly booted → IP65)
  • Pinout
    • Excel
      • Get an idea of how things are connected so that it is easier to implement into rapid harness
    • RapidHarness
      • Great tool for harness building
  • Software
    • Anti stall implementation
    • DFMEA
    • Organized and fully commented
  • ECUs 
    • MoTeC M130 and PDM15
      • Reliable
      • Owned
  • Bundle Type
    • Concentric Twisting
      • Most compact option
      • Best strain relief
        • Most reliable because of this
    • Straight Bundles
      • Slightly lighter
      • Messy
      • Less compact
  • Connectors
    • Deutsch Autosport
    • Deutsch DT/DTM
    • Component Specific Connectors
  • Battery | SIZING > https://www.motortrend.com/how-to/0804rc-electrical-current/
    • Antigravity ATZ10 Lithium Ion (https://antigravitybatteries.com/products/starter-batteries/restart-oem/atz10-rs/)
      • 2.6lbs
      • 360 CCA
      • 12V
      • 12AH (PbEq - Somewhat Useless) / 6.1Ah (Actual)
      • 5.9 in x 3.46 in x 3.62
      • BMS
        • Over-Discharge Protection, Over-Charge Protection, Thermal Protection, Cell Balancing
      • Any mounting orientation
    • OEMish Lead Acid (https://www.batteriesplus.com/productdetails/battery/motorcycle/yamaha/yzf=r6/600cc/2009/cyla9b4xta)
      • ~7lbs
      • 110 CCA
      • 12V
      • 8AH
      • 5.91 in x 2.72 in x 4.13 in
  • 2 Piece Harness

Manufacturing Overview

Tabs​

  • Laser cut and welded for components + zipties​

Flat Pattern​

  • Guide for bundle positions and wire lengths​
  • Can also show pinout for each connector​

Cut & Twist​

  • Cut wires to lengths specified on RH cutsheet​
  • Concentrically twist harness (ELE-3COP-256)
    • https://www.te.com/commerce/DocumentDelivery/DDEController?Action=srchrtrv&DocNm=ELE-3COP-256&DocType=SS&DocLang=EN​

Splice​

  • Splice wires before going on car to save time (trust CAD)​

Continuity Check​

  • Ensure that all wires were routed properly​

Heat Shrink (w/o heat)​

  • Bundles must be sheathed before crimping for it to fit​

Crimp + Pin​

  • Crimp each wire's respective contact and pin all connectors​

Heat the Heat Shrink + Boot Connectors​

    • Following successful operation sheathing shrunk + connectors booted (ELE-3COP-551)
      • https://www.te.com/commerce/DocumentDelivery/DDEController?Action=srchrtrv&DocNm=ELE-3COP-551&DocType=SS&DocLang=EN
