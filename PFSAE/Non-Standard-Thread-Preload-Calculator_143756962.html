Purdue Formula SAE : Non-Standard Thread Preload Calculator

Created by Unknown User (reyno105), last modified on Aug 14, 2020

This calculator can be used to predict nut factors for non-standard threads. It was developed primarily for analyzing center lock threads, but could have other applications. For normal bolts, however, this calculator should not be used because the nut factor will likely be able to be looked up in Shigley's Mechanical Engineering Design. Those values will be more accurate because they come directly from bolt testing data. This skips the middle step of using friction found via some other kind of test to predict friction between threads. This friction is often different. Preload predictions that are not using nut factors from a credible source should be treated with caution. Even for well-controlled bolts, the standard amount of uncertainty is +/- 25% for nut factors. When this was used for center lock threads, a nut factors of anything from 0.1 to 1 was assumed.


The nut factor is the k in the equation T = kFd where T is torque (in-lbf), F is preload (lbf), and d is nominal diameter of the threads (in). It is useful because it provides the ability to calculate preload across sizes of a single type of bolt.
