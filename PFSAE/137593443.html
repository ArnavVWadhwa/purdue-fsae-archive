Purdue Formula SAE : In Plane Geometry Calculator (For Rocker Design)

Created by Unknown User (reyno105), last modified on Jan 14, 2020

This tool can be used to design rocker and pushrod geometry such that all points fall in a single plane. It is not fancy enough to allow any plane. The plane be a line in one of the planes of the cartesian vehicle coordinate system. (I have always designed rockers this way to avoid doing more geometry...)



2d Plane Calcs

Enter the two points that define the line all other points are to fall along. These are labeled ball joint and shock to chassis because those are the points i generally used to start. Then enter any x or y value in the appropriate regions to get the other value that places the point along the line.


Length Calc

Not fancy in the least. Just finds a 3d distance using root sum of squares. But useful for making sure shock length is gonna work out.


Rocker Axis Def Calc

This is probably the fanciest part of this sheet, because I had to remember a thing from calc 3. You enter three points to define a plane. It then finds the normal vector and then the specific line that is that normal vector through point 1. Enter "t" to create a point along that line that is the normal vector * t away from the point. This is useful for defining the axis of rotation in kinematics software. In Adams car for example, point 1 would be BC_center, point 2 would be damper_outboard, point 3 would be prod_inboard. The created point would then be used as BC_axis to define the line the bell crank (BC) rotates about. The value entered for "t" won't actually matter in this application, but I usually selected a small value for it so that there is not a random point 1 mile away from the car in CAD.
