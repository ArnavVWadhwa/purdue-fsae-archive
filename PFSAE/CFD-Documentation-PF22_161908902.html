CFD Setup:

Fully automated java/python suite for cluster and STAR-CCM+. Only user interaction necessary is importing geometry and entering a python command.

Tenfold increase in simulation capacity

Full true-to CAD model 

  • Physics Models
    • Spalart Allmaras (SA) Turbulence Model
      • Note: Convergence is a concern compared to K-epsilon model
    • Steady State
    • Incompressible
  • Mesh
    • ~50 million Trimmed cell mesh
      • Trimmed cell mesh has a much lower memory requirement than polyhedral 
      • Macro suite removes poor quality cells
      • True-to CAD import adds around 10 million mesh cells but saves a lot of manpower
      • ~ 7 hours to solve
    • Sub-Optimal Mesh to under predict downforce since we don't have wind tunnel data
      • Stretch Factor could be reduced but as of now we have no correlation data so it's better to add a safety factor
      • If stretch factor goes down, Cl would go up
    • Mesh dependency study (*Need to add) Unknown User (agarw162)
  • Wall Y+ 
    • Targeting low Wall Y+ for all Aero Components (*Needs fixing) Unknown User (agarw162)
      • Cuts down mesh usage by almost half
