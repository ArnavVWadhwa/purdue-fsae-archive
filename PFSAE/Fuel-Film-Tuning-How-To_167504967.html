Purdue Formula SAE : Fuel Film Tuning How To

Created by Williams, Carson Kenneth, last modified on Jan 17, 2021
  1. Set all trims to zero
  2. default values in main, scale, offset, and time constants
    1. Main 
    2. Offset 
    3. Time Constants
    4. Trim Engine Speed
  3. Closed loop off
  4. activate film compensation
  5. use default values for film increase / decrease time constants for fast and slow
  6. choose fixed engine speed and fixed fuel mixture aim
    1. 6000 rpm
  7. Put those numbers into reference engine speed and reference mixture
    1. engine speed should be in the lower range as films have bigger impacts
  8. Scale the main table by performing steps in the engine load and checking if the lambda trace in acceleration / deceleration seem stable
    1. If trace in acceleration is too lean then increase scale or the table values by a certain factor and vice versa
    2. usually the slope of the table will be quite similar so scaling the whole table will give satisfactory results
    4. see more detailed thing if not satisfied
  9. Adjust time constants 
    1. smaller = increase in film volume will be processed in a shorter time
    2. determine the offset by resuming combustion i.e overrun
      1. Overrun = turn limiter down to 6000 and bang off limiter (Carson's 1am thought)
      2. look at overrun and maybe make a custom script to accelerate to 6000 and shut fuel off for a split second
    3. if the lambda trace is too lean then increase offset and vice versa
  10. Repeat tests at different engine speeds to calibrate trim engine speed table
    1. if lambda trace when increasing engine load is too rich then go to negative values and vice versa
      1. trim engine speed should be pretty linear
  11. Spot check lambda traces in engine load changes at different coolant temperatures
    1. if lambda trace in acceleration is too lean, increase trim temperature


Note:

When you stomp on throttle: fuel film calculator reset tolerance
