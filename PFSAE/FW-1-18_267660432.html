Purdue Formula SAE : FW-1-18

Created by Matthew James Lamear, last modified on Aug 02, 2023

Simulation ML-037

Changes/Justifications

Description of the ChangePictureJustification
Added a main element upper flap.

Need more forward downforce, so I'm going to start testing this concept. 

The idea is that since the main element and 2nd element are so flat and low, the additional upwash generated by the flaps won't massively affect DF production by the main element. This should be mitigated by the endplate vane. This aims to generate a downwashing outwashing vortex that should reduce the upwash caused by the flaps. 

Additionally, the pressure is relatively high on the corners of the main element due to the elliptical nature of pressure distributions. If these regions are affected by the flaps above, they shouldn't be super impactful.

CFD Analysis

CFD PictureDescription

Upper surface upwash is too large to be viable in this spot. Separation likely causing some issues with DF production of the main wing.

Endplate vortex from the main wing is much stronger, increasing outwash. This is likely why UT saw such a massive increase in performance.
