Purdue Formula SAE : FW-2-01 Y0

Created by Matthew James Lamear, last modified on Aug 10, 2023

Simulation ML-043

Changes/Justifications

Description of the ChangePictureJustification
Altered upper flap orientation to be less draggy.

Drag increased substantially when adding this component, so I am working to reduce the drag addition. 

This may reduce upwash overall, but I think the drag reduction will be worth it when adding more components onto this geometry.

I'm planning to add another vortex generator next to this component to further power up the vortex.

Increased height of final airfoil.

This change should allow the vortex to begin sooner, resulting in no overpowered vortex in the 2nd strake channel. 

Improved sweep after the 2nd element.

(also changed airfoil solid parts to surfaces)

The slot gap issue occurring in the previous issue should be fixed with this sweep change.
