Brainstorming Notes

Using Stock Oil System


Need to figure out sensor options because the OE Engine does not measure Oil Temp / pressure


Possible Oil Temp Sensor Locations

  • Drain Bolt: M10 x 1.25
    • Pros: Easy, get a good average temperature, manual also says to use a probe at this location if you are interested.
    • Cons: Ground Clearance / Diffuser Clearance / Need to change oil A LOT
  • BEST OPTION: Engine Oil Filter Element Cover
    • Pros: Easy to mount (adapter)
    • Cons: also need to CNC a custom one RIP

Possible Oil Pressure Sensor Locations

  • BEST OPTION: Engine Oil Filter Element Cover MIGHT be useful to measure pressure (oil filters are usually in the pressurized part of the system)
    • Pros: Easy to mount (adapter)
    • Cons: Need to make sure that the oil pressure at the filter scales or is close to the pressure in the galley for the bearings
      • also need to CNC a custom one RIP
  • Sensor in oil strainer not possible
  • Sensor in oil level check not possible
