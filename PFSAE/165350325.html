Controller chosen and Pros and Cons List: Gazda, Andrew Michael

ECU - 

Pros - Already generally set up with car, seems like a much easier solution for communicating with other sensors as needed

Cons - Somewhat limited in power supply especially with the one cylinder engine this year

Arduino -  

Pros - More freedom in using Arduino

Cons - Likely more difficult to set up on the car and to code.

At this point I'm planning on going down the ECU route though I suppose if some sort of urgent power issue arises I could make a change, but it seems like the most straightforward solution at this point.
