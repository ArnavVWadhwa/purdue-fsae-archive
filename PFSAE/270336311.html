Pre-checks 

Make sure that tc13prod is on the bottom line: this is the server that you need to connect to

General Comments

Teamcenter largely operates in the background.

Launching NX

Simply hit the NX logo button. Make sure that you have NX already installed for this to work on your personal machine!

 


Making a new part/assembly in NX

Open the New Item menu.

Hit [Ctrl + N]

File → New → Item

Search wildcards

Want to see every item relating to PF24? Use a wildcard to represent a string of characters that is missing. For instance, entering PF24* in the search bar will yield the following window.
