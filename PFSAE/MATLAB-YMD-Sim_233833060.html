Purdue Formula SAE : MATLAB YMD Sim

Created by Kurisko, Keith Gregory, last modified on Nov 19, 2022

Uses 4 wheel model (developed off of bicycle model) to analyze vehicle lateral behavior and characteristics

  • Added fidelity to standard bicycle model
    • Nonlinear Tire Model - Pacejka 2002
    • Ackerman Steering
    • Aero contributions Downforce
    • Lateral Load Transfer Fz each tire
    • 4 Tire Fy Calculation
    • 4 Tire Mz Calculation
    • Front Lateral Force Component based on Steered Angle Delta
    • Kinematics
      • Camber Curves (changes IA)
      • Static IA
      • Static Toe
      • Toe Changes
    • Forward Acceleration
      • Simulate Load transfer that occurs under longitudinal accelerations (assumptions made might not initially allow for the calculation of loss in grip)
    • Compliance
    • Investigate Tire Models and correlate to TTC data for changes in Fy and Mz w.r.t inclination angle (Needed before adding kinematic effects)



Sign Convention:

RH Turn:

  • N +
  • r +
  • beta can vary (swept)
  • vy +
  • vx + always
  • Ay +
  • delta +
  • alpha f -
  • alpha r -
  • Fz + always
  • Load transfers from right side to left side

List Assumptions:

  • Simulation V1
    • *Steady State (Assumes longitudinal acceleration = 0 thus engine force = drag force + rolling resistance)
      • *given snap shot in time of vehicle as having yaw acceleration or a yaw moment indicates non steady state


Inputs to Simulation:

  • Simulation V1
    • Vehicle Parameters File
      • Wheelbase
      • Front and Rear Track
      • Vehicle Mass and distribution
      • Aero cl and distribution
      • Cg height
      • etc... (Many vehicle parameters: Currently can only input LLTD, but a further iteration will allow for LLTD to be calculated based on suspension setup (springs, arb setting, rch, etc...)
    • Vx in mph
    • Pacejka Tire Model
    • Sweeps of Beta (vehicle side slip angles) and Delta (steering angle)


Outputs of Simulation:

  • Simulation V1
    • Yaw Moment Diagrams
      • Labeled isolines for delta and beta
      • Color contour plots
        • Control (dN/dDelta) Nm/deg steering
        • Stability (dN/dBeta) Nm/deg body slip
    • Tire Model Investigation Is Commented out
      • good to use to validate the tire and sign conventions being used
    • Could add other features and convert to GUI

Example Results:

Interpreting Results- Right hand plane is right hand turns (ay +); Top plane is clockwise rotations (N +)
