Purdue Formula SAE : iii. Interpreting LLTD Data

Created by Unknown User (hammerly), last modified on Dec 10, 2021

Exporting Table Data

By entering the Parameters section, we can find this table to add design points (DPs) in addition to the current DP 0. This is useful for taking the frame from extremely springy to fully rigid by varying the Young's Modulus of the tube material. Export the table data by right-clicking the table, then clicking "Export Table Data as CSV".

Interpreting TT Data

With the displacement probe on the front suspension nodes, we can calculate a torsional stiffness value used to compare chassis iterations. Parameterize the X and Y displacements of these nodes, then in excel use some vector math to find the angular displacement of these nodes. This is shown by angle alpha in the image below. Divide the moment applied by the weight on the torsion bar by this angle alpha and we now have a torsional rigidity value. This is useful for the x value of a design point when comparing different frame concepts.

Interpreting LLTD Data

From the 2. LLTD / Torsion Test Design Decisions page, we know how to compute LLTD as a proportion of vehicle weight. To find LLTD sensitivity, subtract the LLTD of the case with 350 lb/in springs from the LLTD of the case with stiffer 400 lb/in springs in the front. This yields a % change in LLTD. Now, for each design point plot this % change against the torsional stiffness. Use this tool to compare designs, but don't lose perspective of how these designs compare to a fully rigid frame and the mass cost associated with adding tubes.
