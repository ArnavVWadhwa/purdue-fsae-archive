Purdue Formula SAE : Camber and Toe Compliance Calculator

Created by Unknown User (reyno105), last modified on Jan 21, 2020

This tool sums camber and toe compliance from the chassis, through the control arms, through the upright, and finally the hub and wheel. It generally assumes everything to be in 2d planes.



Chassis Node Displacement

For chassis node displacement, loads from Control Arm Loads Calculator are applied to a FEM of the chassis. Displacements of three of CA nodes at one corner are compared against one of the nodes as a reference. This is done because the all nodes moving together does not create camber change.

The lesser of the displacements of the chassis nodes for a single control arm is used because this will dictate the y direction movement at the upright. (The fact that the other node may have moved more will just cause rotation of the control arm the primarily moves the node in the x axis. While actually just solving the x and y displacement would not be difficult, this spreadsheet provides a quick estimate at compliance, which is likely all that is necessary.


Control Arm Extension

Again, loads from  Control Arm Loads Calculator are used. The two-force member calculator in Structures Analysis Documentation Main Template (and MMPDS instructions) can be used to determine extension. Like chassis node displacement, the lesser extension of the two tubes that make up a control arm is used.
