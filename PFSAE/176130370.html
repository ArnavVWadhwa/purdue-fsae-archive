Tire Testing Consortium (TTC)

Purdue bought into TTC back in like 2007. Basically you pay $500 and you get tire data. A bunch of other teams do this too and that provides enough money for fancy tire tests. Miliken Research Associates manages the testing, which was perfromed at Calspan.

  • Calspan page (overview): FSAE TTC - Formula SAE Tire Testing Consortium (calspan.com)
  • Miliken page (the official page): Milliken Research Associates, Inc. -- FSAE Tire Test Consortium
    • Forum: Formula SAE Tire Test Consortium - Index page (fsaettc.org)
    • You can register on the forum to get access by being associated with Purdue FSAE.
      • Need to register with @purdue email 
  • Drive Location: Z:\Knowledge_Base\01-Suspension\03-Tire_Data\TTC_DATA
    • Each round has content and runguide
      • Content: discussion of tests run
      • Run guide: list of what data file matches what tire
    • There is also a pictures folder, useful for seeing wheels used


Stackpole Engineering Services

Stackpole provides a software for plotting tire models from .tir files. They also provide pre-made tire models if you're in TTC.
