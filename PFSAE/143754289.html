Notes:

  • To be of any use, this system needs to be bulletproof.
  • Reason for pursuing this system:
    • Due to the nature of our competition events, especially acceleration, skip pad, and some smaller autocross courses, a GPS-based lap timing system is not accurate enough. A beacon-based lap time system should really shine when testing acceleration and skip pad. I doubt we could set this system up at competition (maybe on the endurance track if they continue with the large format endurance course). I would expect to use a GPS receiver in conjunction with the beacon-based lap timing system. My goal with GPS would be to have local time in the log file, extra location data, and possibly a rough shape of the course.
  • An easy way to achieve that required reliability and time savings is by purchasing the MoteC solution.
    • Motec BTX transmitter ~$790 (list) would need 2 for accel measurement, maybe purchase a third so skidpad can happen back to back? (need tripod or something and 12v 10ah batteries for each transmitter) ($310 with MoTeC sponsorship)
    • Motec BR2 ~$694 (list) need one per vehicle ($290 with MoTeC sponsorship)
