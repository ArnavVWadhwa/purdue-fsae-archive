Purdue Formula SAE : Tire Data Graphing Program

Created by Unknown User (reyno105), last modified on Mar 07, 2020

This program comes from my less than spectacular freshman year Matlab skills, but it's still all we've got. It can be used to look at tire data directly.

find_best_FY_v2.m

Instructions:

  • Go to this folder to find tire data: Z:\Knowledge_Base\01-Suspension\03-Tire_Data\TTC_DATA
  • Data for our current tire options (LC0 18-6-10 and R25b 18-6-10) is in the ROUND_5 folder.
  • Look at the Contents_Round5 pdf to understand what data is available
  • Look at the RunGuide_round5 pdf
  • Select a cornering run for a tire and pressure of interest
  • Go to the RawData_Cornering_Matlab_SI and double click the .mat file of the run chosen
  • Matlab will open a window providing import options, just select "Finsh"
  • With the data loaded, run the matlab program attached here
  • Inputs:
    • Enter a pressure in psi. The data will include anything within +/- 0.5 psi of the entered pressure (can change code to change range)
    • Enter an inclination angle. Remember that the sign convention of inclination angle is opposite to that of camber. The data will include anything within =/-.1 degrees of the value input (can change code to change range)
      • To include all inclination angles, just enter a stupid big number. Anything larger than 180 degrees will trigger the program to include all data
    • Enter a normal force. The data will include anything within + / - 10 lbs of this. (can change code to change range)
      • To select a normal force where data is available, read about the tests in the Contents_Round5 pdf
      • Note: temperature sweep was performed at 12 psi. So use that data file and pressure for temperature data.
  • The program will then create the Fy vs SA plot as well as a friction coeffecient (mu) vs temperature plot.
