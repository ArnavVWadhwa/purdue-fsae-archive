Knock threshold - any knock % above these numbers there is a proportional ignition retardation from trim gain

Recovery rate - how much ignition increases per second after an event

Gain - if there is a 50% event with 100% event it would pull 50 degrees of timing

Trim limit - limits the amount of timing the gain can pull

Knock Window Start - 10 degrees and window width 60 degrees
