Purdue Formula SAE : FW 1_1 TB

Created by Tanuj Aditya Basu, last modified on Jan 28, 2024

Simulation TB-001

Changes/Justifications

Description of the ChangePictureJustification

Changed the first element to a teardrop design in the 0 and 1 sections to make it more of a flow conditioner for the main element. Looked at Williams F1 design to try and 

Added as much chord as possible to the main element in the 0 and 1 sections. 

Had to slightly alter the main element angle of attack to fit the nose cone and the MEF connector. Also had to alter angle of attack for final element to fit in the rules box.

Changed all of the trailing edges of all elements to have a 0.1 thickness trailing edge rather than coming to a singular point.

Section 0:

Trying to expand the chord of the main element (the second element) as much as possible. The first element can now primarily be used as a flow conditioner to maximize the performance of the main element.
