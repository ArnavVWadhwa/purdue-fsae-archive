Any helpful notes for future machining of this part: 

  • Okay so main thing: stock should probably be slightly larger than the nominal OD, BUT if you're gonna make a donut, PLEASE size it for a/the threaded bolt hole in the sac block. Will make it much MUCH easier to mount to sac block. The current method is essentially to clamp the donut stock to the SB, throw it in the CNC, and zero it on the ID (i will update this if this goes ahead as planned). It would be easier to use the aforementioned method of using a center bolt.
