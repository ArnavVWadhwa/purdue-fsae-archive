Purdue Formula SAE : Running Models

Created by Kurisko, Keith Gregory, last modified on Dec 05, 2023

Follow these instructions to properly setup your sandbox environment to run models.


SETUP SANDBOX: 

  1. Open Command Prompt
  2. Change directory to formula drive
  3. Change directory to sandbox folder under suspension
  4. Add personal sandbox using your school username
  5. cd into that folder and mkdir called whatever the name of your simulation is
  6. cd into that folder

Now your sandbox is created and has the destination folder for where you will download and install models from GitHub.

INSTALL MODELS/SETUP SIMULATION ENVIRONMENT:

METHOD 1: Through GitHub Online

  1. Check what model you are cloning and determine what input files are needed i.e. tire files, vehicle parameter files, etc.
  2. Example will be for Purdue-FSAE/Yaw-Moment-Model: Yaw Moment Diagram Model (github.com)
  3. Goto Desired Repo
      1. Download ZIP and Move from downloads into sandbox folder
  4. Extract All from ZIP
      1. Make sure to set folder path correctly extract to the Sandbox simulation level as shown above
  5. Delete the ZIP folder
  6. Repeat with other Repositories as necessary
  7. For Yaw-Moment-Model, vehicle parameter files are needed along with tire model so those repos were also downloaded as explained Steps 1-5
      1. Properly Cloned Simulation 


RUNNING MATLAB MODELS FROM SANDBOX

  1. Launch MATLAB
  2. Navigate to Your Sandbox Simulation Directory
    1. Add the Simulation Directory to Path
  4. Open Model Main File and Run once parameter files are configured
