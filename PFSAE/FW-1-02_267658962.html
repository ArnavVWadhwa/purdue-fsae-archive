Description of the Change

Justification

Squirt suppressor change

Implement Anhedral style wings

Main focus is to move the vortex lower leading to the wing shape present.

Also thinking about yaw sensitivity vertical segments as proven before do not survive and therefore reducing the squirt suppressor should improve the yaw performance along with still generating a vortex and effect that we want it too.

  Also this iteration will test the effects of anhedral wing which through information I've found could negatively effect performance by creating more in wash along with generating more lift

AOA and sweep might need to be adjusted for the flow field
