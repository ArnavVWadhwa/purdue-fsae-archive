Purdue Formula SAE : Anchor Nuts

Created by Unknown User (jpeerbol), last modified on Oct 12, 2019

For situations when a regular loose nut will not work due to wrench clearances, an anchor nut is a good option.

There different form factor options, but all military/nas spec hardware listed here is a deformation nut meaning it can be classified as a critical fastener.

Because they are deformation nuts, over a few uses (number of usable reuses is currently unknown) the nuts will no longer retain the hardware properly. As a general rule of thumb all elastic and deformation nuts should be replaced once they can be threaded by hand. Hardware is cheap, when in doubt, replace.Â 

This hardware should be mounted with rivets, so if they need to be replaced, the rivets can be drilled out, the nut can be replaced and new rivets can be mounted to the car. In theory these could be welded to steel, but if they need to be replaced, grinding off and re-welding an anchor nut is not a good option. The heated of welding has not been tested and could mess up the threads, requiring a re-tap and removing the deformation feature.


Anchor Nuts that could be used:

MS21075

These are a floating caged nut and can be used when some give is desired for mounting components. These could be difficult to thread when mounted in any orientation other than horizontally because they are floating.


A typical threading for this nut is

8-32, PN: MF5000-08

10-32, PN: MF5000-3

MS21069

A typical threading for this nut is

8-32, PN: MK1000-08

10-32, PN: MK1000-3

MS21071

A typical threading for this nut is

8-32 , PN: MK2000-08

10-32, PN: MK2000-3

MS21055

A typical threading for this nut is

8-32, PN: MK3000-08

10-32, PN: MK3000-3

These have been modeled and are in the piece parts folder in 8-32, 10-32.
