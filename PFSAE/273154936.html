Purdue Formula SAE : Suspoints VB Script - Teamcenter Integration (NX 1926/1946)

Created by Dien Quang Nguyen on Sep 26, 2023

Topic: Review the functionality and integration of a script made to import suspension points into NX from an Excel file with the new Teamcenter system that the team utilizes. This is current as of September 2023.

Main conclusion: 

  • For now, just set up the suspoints layout file in the local NX, then import the suspoints layout file into Teamcenter, ensuring that the expression link (ug_excel_read...) is kept. 
    • Once in Teamcenter, as long as the expression link remains updated, the "Update for External Change" function can still be used to update suspoints according to the excel file.
  • Still don't really understand how this Teamcenter server thing works and how it integrates with the script, but it works.
