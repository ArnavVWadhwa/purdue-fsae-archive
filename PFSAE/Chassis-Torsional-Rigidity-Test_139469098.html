Purdue Formula SAE : Chassis Torsional Rigidity Test

Created by Unknown User (reyno105), last modified on Jan 18, 2021

Purpose: for validating FEA results.


Lessons learned from 2020 test:

  • Be sure to include the weight of the bar when considering factor of safety. It turns out to be the equivalent of 31 lbf in the weight bucket, which is considerable when the planned load is ~50 lbf
  • Run the test at least twice. This helps to get rid of any big shifts in the rig.
  • Clamp the moment beam to the fixtures for connecting to hubs. In the model, these are treated as rigidly connected to the beam, so it is important to make that assumption hold true. The data got waay better (in terms of hysteresis and model correlation) once we did this. Even a jank c clamp setup seems to do fine.
  • Take pictures of all dial indicators at each loading. This is necessary to work out what's going on when a reading doesn't make sense and for any location where deflection surpasses one full rotation of the dial indicator.
  • Make sure the jig plates are tightened on the table...Â 
  • Grease the pivot
  • Make sure none of the dial indicator stands are touching the chassis anywhere


In general, 2020 test and model was best since at least 2016, so refer to it over others. (2020 model included beam and pivot correctly and the test includes values for loading and unloading and provided fairly clean data)
