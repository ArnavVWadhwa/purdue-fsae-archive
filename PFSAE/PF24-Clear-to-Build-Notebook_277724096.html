The PF24 Clear to Build is a new manufacturing part tracker. 

How it's done

It's all in an excel workbook. Formulated and color coded. It uses drop downs, tool tip descriptions, etc. 

One thing that it doesn't use and shouldn't use it macros. While it would be nice for adding rows and such, I don't believe it is compatible with excel online which would make the program not accessible on mobile.
