To split data so you can more easily compare runs

  1. place you cursor where you want to split the data (i.e. during driver change when the data stops logging)
  2. File > Split Data...
  3. Edit the file names and driver names so it is accurate
  4. Then split
    1. if you split something like an endurance, you will now have 3 files, the original and the two splits. For something like an endurance, I would keep all 3 files, If you are splitting because you weren't able to download data between discrete events (i.e. accel and skidpad), I would keep the two split files and delete the original combined file that you don't really need
    2. I do like to keep the session number unchanged, because you then have a way to know that it was from the same data download session
