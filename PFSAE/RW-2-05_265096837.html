Purdue Formula SAE : RW-2-05

Created by Matthew James Lamear, last modified on Jul 05, 2023

Simulation ML-016

Changes/Justifications

Description of the ChangePictureJustification
Going back to RW-2-00 to optimize airfoils.

Increased LE radius for 2nd and 3rd element, decreased AoA of main element slightly. Increased chord of 2nd and 3rd element and reduced AoA of both.

Decreased main element AoA 2.5 deg to allow for more DF generation in 2nd element.

Increased LE radius should result in better DF generation and pressure recovery.

Decreasing AoA will increase the amount of DF generated by decreasing the drag component.
