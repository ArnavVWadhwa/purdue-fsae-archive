Purdue Formula SAE : PF23 Frame Analysis (& Other Frame Peripherals)

Created by Rumsey, Helen Nicole on Dec 04, 2022

Analysis can be found at Z:\2023\4_Chassis\2_Frame\2_Simulation

Operating Load Cases

The folder has an ANSYS file for Operating load cases (cornering, braking, etc.). Operating loads come from suspension's control arm forces. An excel sheet in the same folder documents the results from the operating load case analysis. The front impact case is why we added back the top nSIS tubes (otherwise the frame can only survive a little over a 7G front impact before the frame yields around the bottom of the MRH).

Rocker & Shock

Similarly, the rocker and shock loading in the rear has an ANSYS file and excel sheet documenting the results. Mostly looking at the little nub in the rear (since the tube is cantilevered). Also looked at the upper rear frame tube under its 'worst' loading (shocks, engine, and drivetrain) to size the tube.

Torsion Test

Torsion test ANSYS file also located in the folder path. This file should be used to determine how much the frame is loaded during the torsion test (likely around 50 lbs).

Impact Attenuator

Data from Plascore about the honeycomb used in our impact attenuator is also located in this folder. Can also reference the PF20 IA test for data on the honeycomb.

Other Calculations

Z:\2023\4_Chassis\2_Frame\1_HandCalculations

Hand calcs located in this folder include all the suspension lugs, sizing of the impact attenuator bolts, and sizing of the bottom rear tube (mainly to confirm that if it's used as the rear jacking point nothing fails).Â
