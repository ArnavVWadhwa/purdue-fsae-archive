Purdue Formula SAE : How to View CFD Results

Created by Zhang, Chenkai on Nov 30, 2022

Un-tar

  1. Download the tar file from OneDrive and save it to a folder
  2. Open up the folder where it the tar file is stored in file explorer
  3. Type "powershell" in the address bar and hit enter
  4. Type "tar -xvf " and hit tab key. The file name containing extension should automatically show up if the tar file is the only file in the folder. If you, you will have to type the name file including extension in manually
  5. Hit enter and the file will be un-tared in to the folder
