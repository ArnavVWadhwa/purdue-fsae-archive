Purdue Formula SAE : 09/30/2022 - Hall Effect Wheel Speed Sensor, Gap Tolerance Test

Created by Unknown User (xia107), last modified on Oct 01, 2022


Objective

The design goal for the PF23 brakes package is to move the trigger wheel to be a feature on the outer perimeter of the brake rotor. This would benefit in removing component count, clearing up packaging behind the hub, and give more teeth to allow for greater resolution.

The biggest concern is whether the ~0.040 in axial float in the rotor would cause triggering issues for the hall sensor. A test was executed to test this sensor's capabilities in handling this gap variation.

Should be noted that 0.030-0.060 is the specified gap from the manufacturer.



Test Setup

A representative rotor with the outer tooth profile was laser cut out of 0.125" 4130 (representing the thinner of the front and rear rotors). This was fixtured in the lathe chuck, and the hall sensor was fixtured in the tool post.

This allowed for very fine (and measurable via the lathe DRO) adjustments to the gap. Also the lathe had a readout for the spindle RPM which could be used to verify the the value in MoTeC. We tested as low as 13 RPM and up to 1400 RPM

It should be noted that testing this requires a harness, so our solution was to bring the dyno into the machine shop.

If this test is to be performed again for whatever reason, the wire routing to the tool post sensor needs to be very organized and neat.

We also attempted to test transient gap changes, ie not just sitting at a constant gap. This was tough to simulate as hitting the sensor head on a spinning saw would be bad. Nevertheless we tried to test this by manually rocking the Z value back and forth, amplitude of roughly 0.015-0.020 and a frequency of 2-3 Hz. This didn't appear to influence the sensor reading.


Takeaway: There is roughly a sweep of positions 0.180 inch in Z and 0.200 inch in radial that this sensor head could be in and still measure ok.
