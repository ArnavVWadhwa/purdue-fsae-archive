Purdue Formula SAE : CAD Setup | Harness

Created by Unknown User (mstarobr), last modified by Paradise, Hunter on Sep 21, 2023

----------------------------------------------------------------------------------------------------------------------------------------

Objective: this page will show you how to get from no electronics CAD to the beginning of harness CADing. Includes:

  • Ideas of what to prep before CADing
  • How we currently organize our Electronics assembly 
  • Contents and function of each of our Electronics sub-assemblies
  • Setting up the electronics sub-assemblies 

----------------------------------------------------------------------------------------------------------------------------------------

Before getting started on setting up an electronics assembly for a new season:

Before starting, be sure to remember, electronics typically is one if not the subteam that starts latest because we require other teams to be finished or nearly finished with their CAD before we start ours. This however does mean that we have a lot of time to prepare. This includes:

  1. Make sure that you know what connectors go to what module pins
  2. See if there are any additions or subtractions in sensors being used on the car
  3. Source all of the sensors you will be using and download their data sheets and pinouts 
  4. Update the spreadsheet to convey all this information   

Start these as soon as possible, you have about 2 week to a month max to complete these to be the most efficient when harness CADing starts. Use that time wisely. 



How we organize our Electronics Assembly:

In the past, we have laid out the electronics assembly into several pieces each integrated into one another via a main assembly. See Figure 1.  

Why the electronics CAD is separated into parts:

The separate files allow for ease of use:

  • allowing only necessary pieces to be visible when working on different parts of the assembly
  • separating the reference geometry from the all sub-assembly and the  harness being routed, preventing obstructions 
  • allowing for visual references to other subteam assemblies without having to open the main assembly
    • allows us to work entirely independently from other subteams


                   

                               Figure 1: PF22_ELE assembly list


How the CAD assembly is laid out: 

In the case of our 2022 CAD, we divided the assembly by into 3 parts. 

PF22_ELE_0000_refernceGeo: our reference document with assembly pieces from other subassemblies 

    1. this assembly must be constantly monitored and altered based on the other subteam updates 
      1. make notes about team updates
      2. keep communications with the other subteams (namely chassis and power train)
      3. add any piece that you think will alter your harness path is worth adding to this assembly 
    2. When integrating PF22_ELE with the other subteam assemblies, make sure this one isn't visible 


PF22_ELE_9000_Harness: This assembly is where you will mainly be working in

    1. this is the assembly you are doing all your sensor and module placement and harness routing 
    2. this includes all your connectors or placeholder sensors 
      1. any sensors outside of this assembly or placed in the ELE assembly by another subteam must them be added to this assembly 
    3. When working on any harness, make sure you select this assembly as the workpiece (or open it independently of any other assembly) BEFORE you start changing or adding anything   


PF22_ELE: The end all be all of the electronics CAD, where all the work is laid out together.

    1. used for double checking features between ELE_0000 and ELE_9000 
    2. this also holds some of the parts that will come in handy to other subteams 
      1. examples include the battery and master switch since Power Train typically designs the tabs for both of these components
    3. When starting an electronics assembly, add all other electronics assemblies to this one   



Setting up the electronics CAD for a new season: 

  • For this step you need a new assembly (This will be the Main ELE assembly, Ex: PF22_ELE, abbreviated to MEA) to be added to the current All Team Assembly (Ex: PF22, abbreviated to ATA here). This will allow you to obtain any other subteam parts necessary for the harness CAD.
  • Add both of the other assemblies (Ex: PF22_ELE_0000_refernceGeo and PF22_ELE_9000_Harness, abbreviated to ERA and EHA respectively) into the new electronics main CAD, so that they can be accessed within the ATA 
  • I would recommend adding parts to the EHA first so that all of the chassis frame is already added before adding other relevant parts to ERA  
  • When adding parts from the ATA to EHA, make the EHA the Main Work Part (to do so, either right click the assembly and selecting "Make Work Part" or by double clicking the assembly) Command is found in Figure 2. 


                             

                                        Figure 2: Work Part Selection   


  • For the parts that need to be added, you will have to use the WAVE Geometry Linker command (Figure 3), using the Body select feature within it (Figure 4)
    • All parts added will have the positional dimensions and their position relative to other assembly parts that it in its original assembly   
      • This command adds all parts selected into whatever the Current Main Work Part is (so please do the previous step) 


                                                         

                                               Figure 3: WAVE Geometry Linker                                                   Figure 4: Body Selection Sub-Feature 


  • Select the designated parts for each assembly 
    • PF22_ELE_9000_Harness:
      • all chassis tubes (especially on the driver's right side, where most of our routing goes)
      • the base plate and heat shield (depending on the year, either of these places are where your modules will most likely be mounted)
      • engine, intake, and dash panel (these are pieces that connectors are directly connected to, can theoretically be deleted later, to prevent overlapping parts in the ATA) 
    • PF22_ELE_0000_refernceGeo:
      • any piece the harness must along and must evade (ideally these are temporary additions that can be deleted later, to prevent overlapping parts in the ATA)      
        • includes but are not limited to:
          • Mounting suspension tabs
          • Exhaust pipes and muffler
          • Steering column/protective plate
          • Catch cans 
          • Fans mountings  
          • Coolant and fuel tubes
          • Drive train sprockets and chains  
  • Once these assemblies are filled out make sure that each of them have what you expect before closing the ATA

If all of the steps are completed, you should be ready to start working on the electronics harness
