Purdue Formula SAE : Mass Accounting and Reference Sets at Top Level

Created by Unknown User (xia107), last modified on Oct 06, 2021

Relevance

At any given moment, we should be able to open PFXX.prt and quickly look at mass of the entire car as well as a mass breakdown of all the subsystems. Each subsystem owner should take these steps to ensure all component masses are reflected accurately at the top CAD level, and continue to update materials and reference sets even as the design changes.

(This is currently NOT the case in the assembly below)

What is a Reference Set?

In higher level assemblies (like PFXX.prt), Reference sets are saved configurations for what is displayed and what isn't. By setting this correctly, we can ensure that all the vehicle components and only the vehicle components show up in the top level assembly. This will eliminate displaying non-vehicle peripherals like Aero molds, Chassis jigs, random Coordinate systems, etc.

To make a Reference Set:

  • Make your assembly of concern the active work part, it will be bolded in the tree to indicate that.
  • Select the Reference Sets menu, use search if needed
  • Create one named PFXX, assuming this is the reference set you want to represent your subsystem at the top vehicle level
  • Select the bodies as needed to add to the reference set. You can do this at a lower assembly too for fewer components, if that makes things easier.


Updating Mass Properties

1. Start by just checking what the current calculated mass is.

Make sure you have your desired reference set selected and show mass properties in the tree by right clicking at the top of the tree and checking the following option:

Update Mass Properties if needed:


2. Now is the fun part:

Do NOT just look at an assembly assume the calculated mass does or doesn't seem right. This is useless. Go down the tree, expand to the component level and make sure that every .prt under your ownership:

         a. Doesn't have repeat bodies (this is especially important with how we use Wavelink and layout files)

         b. Doesn't have missing bodies

         c. Has correct material properties assigned to bodies

Go to steps 3 and 4 if you aren't sure how to check for these details.

This is the tedious part, but make sure you expand every assembly in the tree and make sure there is only the bodies you expect in there, and that each one has a material assigned, and therefore has a mass calculated:


3. If you aren't sure the right bodies are in a .prt and that there aren't duplicates, follow these steps:

Go to the lowest level .prt file in your assembly and open that in a new window. You can see below that FL_UCA_FORE and FL_UCA_AFT are the lowest level .prts here. That is what we where we want to verify bodies.


With the standard by which our team follows for CAD, this .prt should ONLY have a linked body in it. This .prt SHOULD NOT have sketches, extrudes, or any intermediate modelling steps. ONLY THE FINAL BODY IS LINKED INTO HERE. Go to Layout file extra info if you need a refresher.

The only thing here is a linked body:



4. If you need to assign a material to something:

Search "Assign Material" and open that menu. Find the most appropriate material and select all applicable bodies (Hint: Density for metals is pretty much the same across all aluminums, across all steels, etc. Do make sure to select the correct materials for plastics, composites, etc.)




5. Rinse and repeat:

Go down your assembly tree and repeat this process of checking for proper bodies and components in your .prt files. Go back to your main assembly and select Update Mass every once in a while to make sure any corrections you are making are being reflected.
