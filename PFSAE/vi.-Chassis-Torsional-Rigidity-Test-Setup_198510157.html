Purdue Formula SAE : vi. Chassis Torsional Rigidity Test Setup

Created by Klaveano, Alexander Ryan, last modified on Jan 15, 2022

To go along with computer analyses done on the chassis rigidity, we need to develop a test setup that can be used to experimentally validate how close the behavior of the chassis is to our expected results from simulation.

The suspension is designed with the assumption that the frame is rigid, which is an okay assumption to make for design, but when fine tuning the car for characteristics like understeer/oversteer balance, the inherent torsional springiness of the chassis will affect the ability to do so using the ARBs.

  • This test setup must be able to apply an adjustable amount of loading to the front axle.
  • The rear wheels will be held rigid to the table using a fixture yet to be designed. This fixture will attach directly to the rear hubs.
  • A similar fixture will be used to support the front wheels via the hubs, but instead of fixating them to the table, they will be attached to a long steel beam that will extend far out from below the car, upon which weights can be controllably placed to apply a twisting load to the chassis.
  • All suspension dampers shall be temporarily replaced with rigid rods to eliminate damper springs from the test variables.

Test cases shall be compared to those from this page iv. PF22 Design Decisions from LLTD/TT
