Problem:

The team would like to implement a robust electronic throttle control system that meets all redundancy and emergency shutdown rules.


Goal:

  • Select accelerator pedal position sensors and design circuitry to meet the transfer function rule
  • Implement circuitry to have brake pressure sensors meet rules
  • Implement throttle plausibility checks
  • Design and manufacture analog brake system plausibility device for shutdown circuit


Requirements:

VehicleETC
Maximize Competition Points
  • Include necessary testing points for inspection
  • Perform adequate analysis for submission
  • Select components that meet rules requirements
Driver Safety
  • Adequate redundancy with both throttle position and pedal position sensors
  • Reliable BSPD system
  • Reliable throttle plausibility programming
  • Ensure that power remains off until a full reset of vehicle power
  • Inclusion of pulldown/pullup resistors on all critical sensors
System Reliability
  • Manufactured PCB for final BSPD circuit
  • Dustproof enclosure for sensitive electronic components
  • Accurate sensor calibration for brake pressure, throttle position, and accelerator pedal position sensors
  • Quick response from the throttle according to the accelerator pedal
  • Accurate throttle vs braking table to prevent accidental ETC shutdown
System Serviceability
  • Clear component layout on the PCB to allow for the replacement of faulty components
  • Independently disconnectable sensors
  • ETC control circuits do not affect the placement of other critical vehicle components
Driver Confidence
  • Customizable pedal maps for each driver's preference
  • Quick response time from pedal to throttle
  • Clear indication of when system shutdown will occur
