Purdue Formula SAE : Feeds and Speeds

Created by Unknown User (reyno105), last modified by Unknown User (xia107) on Aug 11, 2021

This page is to provide some reference for machining speeds, feeds, and depth of cut. It is obviously overridden by anything the machinists who run the ME shop say or the sound of the machine in front of you, but hopefully provides some reference. Ultimately, the goal is to get a feel for whether the machine and cutter are "happy", but this is a good starting point.

The goal for roughing passes is to get discrete chips to form rather than one continuous chip. This is to avoid launching the massive chip ball into your face... which hurts. Turning up feed rate is usually how to get this to happen. The goal for finish passes is to get a good surface finish, which will require turning the feed rate down. Try to couple this with smaller passes due to said issue of launching chip ball.

Note: can generally face at whatever speed you were turning at.


Lathe - Drilling

Try to keep a steady chip coming out. If this stops, you need to back out to clear chips. Chips will get caught between the bit and side of the part. When this happens, the bit gets jerked and can break easily.

Lathe - Reaming

RPM is lowest rpm in high gear (~100 rpm). Drill pilot hole to size from this chart: https://www.rockrivertool.com/documents/pre-ream-drill-size-chart.pdf (or similar). Pretty much the hole you drill should be about 3% smaller than the reamer. Go slow! Reamers are not for taking abuse.

Manual Mill - End Milling

Always use the shortest end mill that can reach where it needs to. Less moment = less chatter + lower risk of breaking

Use a four flute end mill for steel (I think so that you can use lower rpm and make less heat). Use a two flute end mill for "gummy" materials like 6061 aluminum. Can use either for 70XX aluminum because it isn't as gummy

For steel, or with small end mills, some mobile met is usually a decent idea

Table below is roughing passes for fastest material removal. Can increase feed rate if you're not taking off so much material. The end mill needs to be sharp. Also, check that the tip aren't broken off or you're radii will be larger than intended. For the first pass, start it by hand to feel if there will be issues. It may also help to go a radius length into the material slowly by hand and then using auto feed if you want to take larger passes. This prevents the peak leading from being much higher when it hits.

Manual Mill - Side Milling

Always use the shortest end mill that can reach where it needs to. Less moment = less chatter + lower risk of breaking

Conventional cut (spin against motion of table) on roughing passes. This is because large climb cuts will pull the table into the slop of the lead screw and the cause jerking as the lead screw catches up and falls behind repeatedly.

Climb cut on finish passes. The finish pass won't apply enough force to jerk the table. You get a better surface finish because chips get cleared to behind the end mill. This means they won't get caught between the tool and part causing smearing. (Note, this only applies for manual mills. CNC mills use climb cuts always because they don't have slop and you need to clear the chips properly.)

Use a four flute end mill for steel (I think so that you can use lower rpm and make less heat). Use a two flute end mill for "gummy" materials like 6061 aluminum. Can use either for 70XX aluminum because it isn't as gummy.

In general, goal is to make long chips (the full height of the cut) not dust. If you're making dust, reduce RPM and/or increase feed rate. If the table is shaking, try RPM up and down as may be resonance. If that doesn't fix it, gotta take smaller passes.

Manual Mill - Fly Cutting

ALWAYS CHECK THAT THE SET SCREWS HOLDING THE CUTTING TOOL ARE TIGHT. YOU'RE GONNNA BE STANDING IN FRONT OF THE THING; DONT TRUST THE LAST PERSON TO HAVE CHECKED.

Tends to jostle the table lower on the last passes, so make sure to lock the z axis on you're final cut if you want a flat surface. This is especially true if multiple passes are needed at a single depth.
