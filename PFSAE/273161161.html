Purdue Formula SAE : Useful Keyboard Shortcuts (keybinds) in NX

Created by Liang, David C, last modified on Oct 22, 2023

Add any keybinds you find useful

______________________________________________________________________________________________

Middle Mouse Button (scroll wheel) = MMB

Right Mouse Button = RMB

Left Mouse Button = LMB

Shortcut NameInstructionsWhat does it doPicture (if applicable)
FitCtrl + FIf the CAD file is clipping (as shown in picture), there is a memory problem or smth. Using the keybind will fix it. It will also fit the entire model in the screen.Â 

RotateDrag MMBRotates the camera about the mouse
PanDrag MMB + RMBPans the camera
Open part fileCtrl + OOpens a page, from where you can open a part file (will work with teamcenter)
Open new fileCtrl + NOpens a page, from where you can create a new part file (will work with teamcenter)
UndoCtrl + ZUndo
RedoCtrl + YRedo
DeleteCtrl + DCan highlight something and delete it, or will open a tab, from where you can delete several parts
Finish sketchCtrl + QWill finish the sketch
Normal to sketchShift + F8Will pan the view to normal to the sketch
Switch windowsAlt + TabWill switch between tabs in NX
Snap ViewF8Will snap the view to the nearest one (top, down, or side)

Ctrl + WWill show a list of things to show and hide (datums, points, sketches, etc.)
