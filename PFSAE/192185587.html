Purdue Formula SAE : Start Switch Relocation | PF22

Created by Unknown User (jkrawitz), last modified by Paradise, Hunter on Jul 16, 2023

Why: Want a system with less critical failure points

If dash fails, we can't start the car even though the dash is not completely necessary for the car to function. The PDM is necessary for the car to run and if the PDM fails the car shouldn't be able to start anyway.

Therefore, we are relocating the start switch from the PDM input of the dash to a switch input of the PDM.Â
