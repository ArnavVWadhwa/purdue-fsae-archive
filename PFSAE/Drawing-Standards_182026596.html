Simple Drawing:

  • Likely used for stock prep/ machine setup, but for parts that will end up mostly CNCed
  • Basic dimensions, tolerances
  • Rough idea of tools needed and stock size
  • Accompanies .stp file


xia107_PF22_SUS_3501_WHEEL_CENTER_V1.pdf


Detailed Drawing:

  • For parts where a person will have to do the majority of machining
  • Should be drawn with milling/turning operations in mind
  • Should use multiple views/sheets if necessary to make machine time as efficient and simple as possible.
