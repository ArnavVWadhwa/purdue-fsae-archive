Gear position

Blocker: Need to decide how to read from sensor

pink wire: 1st or 2nd gear (both about 1.6 ohms)

sky blue: neutral

yellow/white: 3rd, 4th, 5th gear 

so will need to know gear group and then speed to calculate gear?

so can do 2.5 options

  1. 1 input with resistor selected output (voltage across resistors based on which gear is connected to ground and then the voltage read by the ecu would be different) - 1 input and some complexity in circuit
  2. 1 digital input for each of 3 wires - 3 inputs, potential difficulty in code

       2.5) 1 input for neutral switch and then a selector between 1/2 and 3/4/5 - uses 2 inputs and some complexity in code and some potential complexity in circuit

want to limit # of inputs used because limited amount on car, want to limit free circuitry in car as needs to be mounted and soldered properly, want to limit code complexity because... code(no real reason)
