Purdue Formula SAE : Wheel Nuts (PF20 and Prior)

Created by Unknown User (reyno105), last modified by Unknown User (xia107) on Dec 05, 2021

Stock Prep (Lathe to Drop Saw)

Start with disc of exact external dimensions of wheel nut. Recommended to prep stock for full planned quantity as CNC operation is quick so setup time is a significant portion of total time. Recommended to turn and face in same operation for each disc such that OD is perfectly perpendicular to one face. The other face may be out of parallel by approx .010" with minimum repercussion.


Hex Flats (CNC Mill)

Since all discs being machined will be at same dimensions, can just place into 3 jaw layed on CNC mill table, find zero on one part, and then run all parts. Place each disc into the jaws on the face that is known to be perfectly parallel to the OD axis so that wrench feature aligns. This will assure that the hex can be clamped in the lathe to have good alignment when thread turning.


Internal Threads (Manual Lathe)

Drill to as close to thread minor diameter as possible (for 2018 - 2020 + design, 1.75" drill bit is used). This drill bit is larger than 1/2" so will need to pilot drill. Choose a pilot drill that is just larger than the web of 1.75" drill bit. (The web of a drill bit is the flat part at the front of a drill bit that does not come to sharp point, which is typical for large drill bits). Break all the way through with both drill bits (duh). Use boring bar to finish to pre-thread diameter (just smaller than minor diameter, I think we get the value from MasterCam with some tool used for determining diameters prior to thread turning). Then thread turn.
