Purdue Formula SAE : Dyno | Troubleshooting

Created by Unknown User (jkrawitz), last modified by Paradise, Hunter on Jul 16, 2023

Step 1: Be able to bring power to PDM

  • Measure battery input to see if power goes to PDM
  • Measure output voltage from PDM to ECU

Step 2: All Power Systems

  • Solenoid: is it connected to battery and PDM properly
  • Starter switch: 
    • does the switch work
    • if battery on is voltage drop right
  • Starter: this is long wire from engine. does it have proper connection (needs to be powered)

Step 3: Use PDM to power engine

  • Confirm starter if you havent already
  • Turn on pdm and turn starter switch: if engine starts up all good
  • If doesnt turn on check route of wires from pdm/ecu to engine and check continuity

Step 4: ECU

  • Does power go from PDM to this module to turn it on?
  • Are all sensors on there? Do they have connectors?
  • Are there any shorts between power and ground wires? (continuity test)
  • Is it connected to CAN bus?
  • Is it connected to ethernet to read? check these wires

Step 5: Dash 

  • Is the CAN line connected?
  • Is it receiving power?
  • Is ethernet port set correctly?


If nothing starts/happens when you plug in battery and turn power switch start at Step 1.

If any module doesn't work, go to the associated step.

If a sensor doesn't work check its continuity, then the module(s) it is connected to. Check ECU/Dash depending on which it is connected to and then PDM/ power wires.
