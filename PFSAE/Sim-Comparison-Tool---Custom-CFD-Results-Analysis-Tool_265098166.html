Purdue Formula SAE : Sim Comparison Tool - Custom CFD Results Analysis Tool

Created by Zhang, Chenkai on Jul 11, 2023

Made a tool last summer since I got tired of manually checking for tiny differences. This tool basically allows you compare two simulations easily by allowing you to do 2 things:

  1. Obtain the delta between 2 2D scenes
  2. Blend the 2 2D scene or switch quickly back and forth between 2 pictures

For installation and and user guide click this link: https://github.com/Purdue-FSAE/SimComparisonTool/wiki

For download link click this link: https://github.com/Purdue-FSAE/SimComparisonTool/releases


You can easily make it better if you wish, here are a few things you can do:

  1. Add keyboard hot key for quickly switching alpha channel between 0 and 100% when blending
  2. Add color map to the delta image
  3. Find a custom library for delta image and implement it to make it faster

Item 1 and 2 are very easy to do, I've just been lazy. Item 3 can be a little difficult, but nothing undoable. If you want to create a new release, look carefully at the current zip file structure, and follow it exactly.
