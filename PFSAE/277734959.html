Purdue Formula SAE : ADAMs Car | Setup Environment

Created by Kurisko, Keith Gregory, last modified on Dec 05, 2023

Prior to opening any vehicle models and running simulations you must setup you ADAMs Environment

  1. Add to session our database which needs to be downloaded and cloned from GitHub
  2. Make sure to name Database Alias identical to the folder name but you can leave off the ".cdb"
  3. Click Apply
  4. Bottom right should have this drop down where you can select the active database
  5. You can now go to file→ open→ assembly→ <right click white space>→ Search→ <database>
  6. Click Apply
  7. The vehicle model should now be loaded and environment ready
  8. If you want to change units go to Settings→ Units NOTE: You will need to change this alot as it randomly resets
