Adams Car File Naming Convention

_nano aero
_wawith aero
_rreal
_ssimulated

What is the difference between no aero and with aero?

No Aero indicates Cl and Cd values are set to those expected to no aero package

What is the difference between real and simulated?

  • Real simulation involves results of kinematics and tire dynamics pertaining to camber curves and changing inclination angle
  • Real simulation is not always desired and the limitations of your simulation model/test should be known prior to deciding which to use. If you are unsure then you most likely want _s model where camber effects are turned off in tire property file
