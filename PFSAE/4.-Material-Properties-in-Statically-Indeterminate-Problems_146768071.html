Purdue Formula SAE : 4. Material Properties in Statically Indeterminate Problems

Created by Unknown User (napolia), last modified on Apr 28, 2020

When a statics problem is non determinate, when there are more unknown values than there are equilibrium equations, then material properties are used to make up the difference in equations. The material properties that we can use include Poisson's Ratio, Young's Modulus, and Modulus of Rigidity (or Shear Modulus), which is analogous to the Young's Modulus, but for shear stress instead of axial..
