Purdue Formula SAE : Frame Design Process

Created by Dien Quang Nguyen, last modified on Dec 08, 2023


The frame design process is summarized by the flowchart shown above, Generally, the frame is designed around the suspension points, i.e. the points at which suspension components (control arms/wishbones, tierod, pushrod, etc. connect to the frame. This is especially true for a spaceframe. 


Suspoints

Suspoints (suspension points) is a collection of all points which define the way suspension components interact with each other and the frame. This is often organized using a simple spreadsheet that contains data exported from simulation software such as Adams Car or Matlab models.


To start designing the frame, we import these points into CAD. 


After that, 2D sketches are made to determine major parameters of the frame. These are called the skeleton model and human model

(Both sketches shown above)


Then the cross-section sketch of the front bulkhead is created. Theoretically, we could start from any tube and design the entire frame from there. However, starting from major components such as the front bulkhead or roll hoops make sense because they are the most critical and are the major connection points for all other frame members

From then, a full wireframe is developed.

But before this can be accomplished, we must determine the chassis node. These are points on the chassis to which suspension components will be connected. Thus, they can be defined with an off-set from suspension points in a certain direction (vector). This ensures that there is a consistent gap between the chassis node and suspension points.

Note that this might not necessarily apply to  

Major components with more complex outlines such as the front bulkhead, roll hoops, bent
