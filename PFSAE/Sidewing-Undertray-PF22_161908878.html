Purdue Formula SAE : Sidewing-Undertray PF22

Created by Unknown User (agarw162), last modified by Bermack, Christopher Richard on Dec 15, 2020

To SW… or not to SW?

  • Main design avenues are going to center around whether I want to actually do a wing because Raunaq says it’s hard
  • Trying to get some inwash to go under the car to the underbody
  • Also might try a sort of hybrid wing-tunnel where I use an airfoil as the top section of the tunnel
    • Might also sweep this; we’ll see how it goes
  • Ideally want to make some sort of decision by 12/5 if I'm going to do wings or not; at least know how wings perform vs. tunnels

Update 12/14

  • So it's looking like either a swept wing very similar to 2020 or a hybrid flap/tunnel design (see 8+ series design iterations)
    • I really want to do the hybrid because it would be cool, and because it wouldn't be dependent on slot gaps, which is dependent on radiator and sidepod
    • I need to get this decided by the preliminary design review, which I think is early January
    • Also would need to hammer out some other design choices, like how I slant the hybrid if at all, if I even do the hybrid
    • And also would need to implement some better undertray profile (I believe Raunaq has been trying out UT stuff)
  • I'm currently getting the top-performing former designs (3b_slantwing2, 8a_hybrid2) in the rules bound and running with current geometry
    • The 2020 FW seems to be having a pretty significant effect on current SW design performance, so I need to sorta re-calibrate everything
  • Also working on getting radiator sweeps up and running (hopefully will do tomorrow, definitely sometime this week), which means designing sidepod
    • Expect drag to increase, downforce to be affected little; looking for mass flow targets
    • Ideally want sidepods for both leading designs (one slantwing, one hybrid)
    • Designing to 9x8" radiator, will likely need to design to smaller radiators in the coming couple of weeks
