Purdue Formula SAE : Stripping Tools

Created by Unknown User (jpeerbol), last modified by Paradise, Hunter on May 31, 2023

MoTeC recommends the "Ideal Industries 45-2133 stripping tool with LB1195 wire stop"

For some reason this is a crazy expensive tool. https://www.newark.com/ideal/45-2133/tools-wire-strippers/dp/96B6030


I personally used a similar looking tool, the 'Klein Tools Katapult 11063W Wire Cutter / Wire Stripper' and I have liked this tool. The stop for this set of strippers is 'Adjustable Wire Stop and Gauge for Katapult Wire Stripper Klein Tools 11081'.

The Klein Catapult 11063W will not strip 22 awg Tefzel because the insulation is too thin. You will need set of blades for 16 to 26 AWG wire (Klein part number 11074). https://www.mouser.com/ProductDetail/Klein-Tools/11074?qs=rkhjVJ6%2F3EKWRNdw%252BYeBKw%3D%3D


The Klein setup currently costs around $40 compared to the ideal $400 as of 1/12/2022.
