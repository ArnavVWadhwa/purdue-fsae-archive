Projects

My initial thoughts are to perform lap sim analysis or start with referring to previous years analysis and determine initial architecture targets while confirming results with Milliken Moment Method studies confirming the performance and drivability envelope. This can either be completed before or after suspension points designed along with additional analysis tools. *Next set of analysis should be done for future design along with PF22 and PF23. Handling of the car along with other design criteria and metrics will be determined using Steady State Handling analysis with both a bicycle model and 4 wheel model (Compare the two models along with historic data if available). Handling of the car along with other design criteria and metric will be determined using Transient Handling bicycle model and 4 wheel model. Step steer maneuver will allow for response analysis and frequency analysis of vehicle envelope. This should allow the study of roll center heights, roll gradient, ...etc. Longitudinal dynamics need to be considered aswell. Braking and acceleration simulations should be conducted to study performance and potential effects on stability or handling. We need to create a traction chart for acceleration.
