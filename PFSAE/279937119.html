Purdue Formula SAE : Build Revision | PF24

Created by Paradise, Hunter, last modified on Nov 20, 2023

This is a list of all the useless (to us) stuff in the base GPRP Build project that needs to be removed.


In attempting to delete Alternative Fuel I deleted all secondary injector attributes. This broke the build. Lesson learned: version control, version control, version control. Sadly M1 Build does not have this feature built in so you will need to create a revision on your own every so often.
