Purdue Formula SAE : FW-2-07 Y15

Created by Matthew James Lamear, last modified on Aug 14, 2023

Simulation ML-049

Changes/Justifications

Description of the ChangePictureJustification
Changed footplate to include a section of height increase.

This should start the vortex in a more outboard position, and higher up allowing the vortex to survive longer.
Cut out section of endplate, similar to last year's design.

The idea is to place the inwashing wake higher up such that it doesn't get inwashed. Specifically, this will work best in higher yaw cases, but should still prove beneficial in yaw 0.
In conjunction with the previous idea, the outboard most strake is lengthened.

This should house the wake for a longer period of time.

CFD Analysis

This:

Previous:

Due to the gurney design, the flow is accelerated, thus resulting in additional inwash; this powered up the four vortices more. This likely reduced overall inwash downstream which delayed enough to allow the tire squirt suppressor to somewhat function properly. 


I think the real winner is the endplate cut. I'm not entirely sure if this floor plate geometry is actually worth it, but the cut seemingly is.  

The cut releases the vortex into the first channel instead of affecting all the channels. Without the additional inwash generated by the floor plate, this should result in better performance overall.
